# Shannon Lite与Pro对比

## 简介

Shannon有两个版本。让我们比较它们，帮助您做出选择。

## 对比表

| 功能 | Shannon Lite | Shannon Pro |
|------|:-----------:|:-----------:|
| 源-汇分析 | 基础 | LLM驱动的数据流分析 |
| CVSS评分 | ❌ | ✅ |
| 修复指导 | 基础 | 代码级修复 |
| CI/CD管道 | ❌ | ✅ |
| API访问 | ❌ | ✅ |
| Jira/Linear/Slack | ❌ | ✅ |
| 托管方式 | 自托管 | 云或自托管 |
| 多用户和RBAC | ❌ | ✅ |
| SSO/SAML | ❌ | ✅ |
| 合规（OWASP、PCI-DSS、SOC2） | ❌ | ✅ |
| 支持 | 社区 | 专属 + SLA |
| 费用 | 免费 + API成本 | 联系销售 |

## 关键技术差异

**Shannon Lite**在LLM上下文窗口内分析代码——适用于单个文件和简单数据流。

**Shannon Pro**使用**LLM驱动的数据流分析**：
- 基于图的全代码库分析
- 跨文件和模块跟踪数据流
- 检测复杂的多组件漏洞

## 谁应该使用哪个

**Shannon Lite：** 独立开发者、小团队、个人项目、学习和研究。

**Shannon Pro：** 有合规要求的组织、大型代码库、CI/CD需求、企业支持需求。

## 许可证

- **Lite** — AGPL-3.0（免费，开源）
- **Pro** — 商业许可证

## 总结

- Lite是免费的，非常适合小团队和个人项目
- Pro适合需要深度分析和集成的组织
- 关键区别是Pro的高级基于图的数据流分析
