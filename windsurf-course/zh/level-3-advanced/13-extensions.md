# 扩展生态

## VS Code 兼容性

Windsurf 支持大多数 VS Code 扩展——数千种工具可用。

## 安装扩展

1. 点击活动栏中的**扩展**图标（或 `Ctrl+Shift+X`）
2. 搜索扩展
3. 点击 **Install**

## 推荐扩展

### 开发工具
- **ESLint** — JavaScript 代码检查
- **Prettier** — 代码格式化
- **GitLens** — 高级 Git 功能
- **Docker** — 容器管理

### 语言支持
- **Python** — IntelliSense 和调试
- **Go** — 官方 Go 支持
- **Rust Analyzer** — Rust 支持
- **Tailwind CSS IntelliSense** — CSS 工具类

### 生产力
- **Error Lens** — 内联错误显示
- **Todo Tree** — 查找代码中的 TODO
- **Path Intellisense** — 文件路径补全

## 扩展 + AI

Windsurf 的 AI 与扩展协同工作：
- ESLint 发现问题 → AI 建议修复
- 测试失败 → AI 解释原因
- Git 显示更改 → AI 帮写提交信息

## 管理扩展

- **禁用**不用的扩展以提升性能
- **工作区扩展** — 按项目安装
- 命令面板中用 `@ext` 管理

## 创建自己的扩展

如果你会 TypeScript：
1. 使用 Yeoman 生成器：`yo code`
2. 在 Windsurf 中开发和测试
3. 让 Cascade 帮你写扩展代码！

## 练习

从推荐列表安装 3 个扩展，看看它们如何与 AI 配合。
