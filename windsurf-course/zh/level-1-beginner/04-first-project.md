# 你的第一个项目

## 目标

在本课中，你将用 AI 创建一个简单的 Python 项目。

## 第一步：创建文件夹

1. 打开 Windsurf
2. 点击 **File → Open Folder**
3. 创建新文件夹 `my-first-project`
4. 打开它

## 第二步：创建文件

1. 在资源管理器中点击**新建文件**图标
2. 命名为 `app.py`

## 第三步：用 AI 编写代码

输入以下注释并按回车：

```python
# 创建一个简单的加法计算器
```

等一下。Windsurf 会建议代码。按 **Tab** 接受。

你会得到类似这样的代码：

```python
def add(a, b):
    return a + b

num1 = float(input("请输入第一个数: "))
num2 = float(input("请输入第二个数: "))
print(f"结果: {add(num1, num2)}")
```

## 第四步：运行代码

1. 打开终端：`` Ctrl+` ``
2. 输入：`python app.py`
3. 输入两个数字
4. 查看结果！

## 第五步：用 AI 扩展

尝试新的注释：

```python
# 添加减法、乘法和除法功能
```

看 Windsurf 如何生成所有四个函数。

## 你学到了什么

- 如何在 Windsurf 中创建项目
- 如何用注释引导 AI 生成代码
- 如何从内置终端运行代码

## 练习

创建一个将摄氏温度转换为华氏温度的程序。从注释开始。
