# 界面概览

## 主要区域

Windsurf 的界面类似 VS Code。主要区域如下：

### 1. 活动栏（左侧）
左侧的垂直图标栏：
- **资源管理器** — 浏览文件
- **搜索** — 在项目中查找文本
- **源代码管理** — Git 集成
- **扩展** — 安装插件
- **Cascade** — AI 助手面板

### 2. 编辑区（中间）
编写代码的地方：
- 多个文件标签页
- 水平或垂直分屏
- 输入时显示 AI 内联建议

### 3. 侧边栏
显示所选活动的详细信息。例如，资源管理器激活时显示文件树。

### 4. 底部面板
- **终端** — 运行命令
- **问题** — 查看错误和警告
- **输出** — 扩展日志

### 5. Cascade 面板
Windsurf 独有功能。打开方式：
- 点击活动栏中的 Cascade 图标
- 按 `Ctrl+Shift+L`（Windows/Linux）或 `Cmd+Shift+L`（Mac）

## 命令面板

按 `Ctrl+Shift+P`（Mac 上 `Cmd+Shift+P`）打开命令面板，输入名称即可访问任何功能。

## 关键设置

进入 **File → Preferences → Settings** 自定义：
- 字体大小
- Tab 大小
- AI 建议行为
- 主题

## 小贴士

- `Ctrl+B` 切换侧边栏
- `` Ctrl+` `` 切换终端
- 右键菜单包含 AI 选项
