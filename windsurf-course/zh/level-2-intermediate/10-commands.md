# 命令与快捷键

## 命令面板

按 `Ctrl+Shift+P`（Mac 上 `Cmd+Shift+P`）访问所有命令。输入关键词过滤。

## 常用快捷键

### 通用
| 操作 | Windows/Linux | Mac |
|------|--------------|-----|
| 命令面板 | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| 快速打开文件 | `Ctrl+P` | `Cmd+P` |
| 侧边栏 | `Ctrl+B` | `Cmd+B` |
| 终端 | `` Ctrl+` `` | `` Cmd+` `` |
| 设置 | `Ctrl+,` | `Cmd+,` |

### AI 功能
| 操作 | Windows/Linux | Mac |
|------|--------------|-----|
| 打开 Cascade | `Ctrl+Shift+L` | `Cmd+Shift+L` |
| 接受建议 | `Tab` | `Tab` |
| 下一个建议 | `Alt+]` | `Option+]` |
| 取消建议 | `Escape` | `Escape` |
| 内联 AI 编辑 | `Ctrl+K` | `Cmd+K` |

### 编辑
| 操作 | Windows/Linux | Mac |
|------|--------------|-----|
| 多光标 | `Alt+Click` | `Option+Click` |
| 复制行 | `Shift+Alt+↓` | `Shift+Option+↓` |
| 移动行 | `Alt+↑/↓` | `Option+↑/↓` |
| 注释切换 | `Ctrl+/` | `Cmd+/` |
| 格式化 | `Shift+Alt+F` | `Shift+Option+F` |

## 内联 AI 编辑（Ctrl+K）

强大的快捷方式：
1. 选中代码
2. 按 `Ctrl+K`
3. 输入要更改的内容
4. 按回车

示例：选中函数，按 `Ctrl+K`，输入"添加 TypeScript 类型"。

## 自定义快捷键

**File → Preferences → Keyboard Shortcuts** 自定义任何快捷键。
