# Extensions

## VS Code Compatibility

Windsurf supports most VS Code extensions. This gives you access to thousands of tools.

## Installing Extensions

1. Click the **Extensions** icon in the Activity Bar (or `Ctrl+Shift+X`)
2. Search for the extension
3. Click **Install**

## Recommended Extensions

### Development
- **ESLint** — JavaScript linting
- **Prettier** — code formatting
- **GitLens** — advanced Git features
- **Docker** — container management

### Languages
- **Python** — IntelliSense and debugging
- **Go** — official Go support
- **Rust Analyzer** — Rust support
- **Tailwind CSS IntelliSense** — CSS utilities

### Productivity
- **Error Lens** — inline error display
- **Todo Tree** — find TODOs in your code
- **Path Intellisense** — file path completion

## Extensions + AI

Windsurf's AI works alongside extensions:
- ESLint finds issues → AI suggests fixes
- Test runner shows failures → AI explains why
- Git shows changes → AI helps write commit messages

## Managing Extensions

- **Disable** extensions you don't use to improve performance
- **Workspace extensions** — install per project
- Use `@ext` in Command Palette to manage extensions

## Creating Your Own Extensions

If you know TypeScript, you can build extensions:
1. Use the Yeoman generator: `yo code`
2. Develop and test in Windsurf
3. Ask Cascade to help write extension code!

## Practice

Install 3 extensions from the recommended list and see how they work with Windsurf's AI.
