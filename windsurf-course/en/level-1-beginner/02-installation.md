# Installation & Setup

## System Requirements

- **OS**: Windows 10+, macOS 11+, or Linux (Ubuntu 20.04+)
- **RAM**: 4 GB minimum, 8 GB recommended
- **Disk**: 500 MB free space
- **Internet**: Required for AI features

## Step 1: Download Windsurf

1. Go to [codeium.com/windsurf](https://codeium.com/windsurf)
2. Click **Download** for your operating system
3. Run the installer

## Step 2: Create a Codeium Account

1. Open Windsurf after installation
2. You will see a sign-in prompt
3. Create a free account or sign in with Google/GitHub
4. Verify your email if needed

## Step 3: First Launch

When you open Windsurf for the first time:

1. Choose your **color theme** (dark or light)
2. Select your **preferred language** for coding
3. The AI model will initialize — this takes a few seconds

## Step 4: Verify AI is Working

1. Create a new file called `test.py`
2. Type `def hello`
3. You should see an AI suggestion appear in gray text
4. Press **Tab** to accept it

If you see the suggestion — everything is working!

## Troubleshooting

| Problem | Solution |
|---------|----------|
| No AI suggestions | Check your internet connection |
| Sign-in fails | Try a different browser for OAuth |
| Slow performance | Close other heavy applications |
