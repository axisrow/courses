# Commands & Shortcuts

## Command Palette

Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) to access every command. Type to filter.

## Essential Shortcuts

### General
| Action | Windows/Linux | Mac |
|--------|--------------|-----|
| Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Quick Open file | `Ctrl+P` | `Cmd+P` |
| Toggle Sidebar | `Ctrl+B` | `Cmd+B` |
| Toggle Terminal | `` Ctrl+` `` | `` Cmd+` `` |
| Settings | `Ctrl+,` | `Cmd+,` |

### AI Features
| Action | Windows/Linux | Mac |
|--------|--------------|-----|
| Open Cascade | `Ctrl+Shift+L` | `Cmd+Shift+L` |
| Accept suggestion | `Tab` | `Tab` |
| Next suggestion | `Alt+]` | `Option+]` |
| Dismiss suggestion | `Escape` | `Escape` |
| Inline AI edit | `Ctrl+K` | `Cmd+K` |

### Editing
| Action | Windows/Linux | Mac |
|--------|--------------|-----|
| Multi-cursor | `Alt+Click` | `Option+Click` |
| Duplicate line | `Shift+Alt+↓` | `Shift+Option+↓` |
| Move line | `Alt+↑/↓` | `Option+↑/↓` |
| Comment toggle | `Ctrl+/` | `Cmd+/` |
| Format document | `Shift+Alt+F` | `Shift+Option+F` |

## Inline AI Edit (Ctrl+K)

This is a powerful shortcut:
1. Select code
2. Press `Ctrl+K`
3. Type what you want to change
4. Press Enter

Example: Select a function, press `Ctrl+K`, type "add TypeScript types", press Enter.

## Custom Shortcuts

Go to **File → Preferences → Keyboard Shortcuts** to customize any binding.

## Practice

Try to work for 10 minutes using only keyboard shortcuts. No mouse!
