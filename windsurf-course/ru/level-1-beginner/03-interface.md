# Обзор интерфейса

## Основные области

Windsurf похож на VS Code. Вот ключевые зоны:

### 1. Панель активности (слева)
Вертикальная панель с иконками:
- **Проводник** — обзор файлов
- **Поиск** — поиск по проекту
- **Git** — контроль версий
- **Расширения** — установка плагинов
- **Cascade** — панель AI-ассистента

### 2. Область редактора (центр)
Здесь вы пишете код:
- Несколько файлов во вкладках
- Разделение экрана по горизонтали или вертикали
- AI-подсказки прямо в тексте

### 3. Боковая панель
Показывает детали выбранной активности. Например, дерево файлов.

### 4. Нижняя панель
- **Терминал** — выполнение команд
- **Проблемы** — ошибки и предупреждения
- **Вывод** — логи расширений

### 5. Панель Cascade
Уникальна для Windsurf. Открывается:
- Иконкой Cascade в панели активности
- Сочетанием `Ctrl+Shift+L` (Windows/Linux) или `Cmd+Shift+L` (Mac)

## Палитра команд

Нажмите `Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac) для доступа к любой функции по имени.

## Настройки

**File → Preferences → Settings**:
- Размер шрифта
- Размер табуляции
- Поведение AI-подсказок
- Тема оформления

## Советы

- `Ctrl+B` — показать/скрыть боковую панель
- `` Ctrl+` `` — показать/скрыть терминал
- Правый клик — контекстное меню с AI-опциями
