# Первый проект

## Цель

Создадим простой проект на Python с помощью AI.

## Шаг 1: Создайте папку

1. Откройте Windsurf
2. Нажмите **File → Open Folder**
3. Создайте папку `my-first-project`
4. Откройте её

## Шаг 2: Создайте файл

1. В проводнике нажмите иконку **Новый файл**
2. Назовите его `app.py`

## Шаг 3: Пишем код с AI

Введите комментарий и нажмите Enter:

```python
# Создать простой калькулятор, который складывает два числа
```

Подождите. Windsurf предложит код. Нажмите **Tab**.

Вы получите примерно такое:

```python
def add(a, b):
    return a + b

num1 = float(input("Введите первое число: "))
num2 = float(input("Введите второе число: "))
print(f"Результат: {add(num1, num2)}")
```

## Шаг 4: Запустите код

1. Откройте терминал: `` Ctrl+` ``
2. Введите: `python app.py`
3. Введите два числа
4. Увидите результат!

## Шаг 5: Расширяем с AI

Попробуйте новый комментарий:

```python
# Добавить функции вычитания, умножения и деления
```

Windsurf сгенерирует все четыре функции.

## Что вы узнали

- Как создать проект в Windsurf
- Как использовать комментарии для генерации кода
- Как запускать код из терминала
