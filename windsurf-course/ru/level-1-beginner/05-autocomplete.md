# Базовое AI-автодополнение

## Как это работает

Windsurf предсказывает, что вы хотите написать. Подсказки появляются серым текстом — это **ghost text**.

## Принятие подсказок

- **Tab** — принять всю подсказку
- **Ctrl+→** — принять слово за словом
- **Escape** — отклонить подсказку

## Типы дополнений

### 1. Однострочные
Windsurf дополняет текущую строку:

```javascript
const greeting = "Привет, " // AI предложит: + name + "!"
```

### 2. Многострочные
После сигнатуры функции AI пишет тело:

```python
def fibonacci(n):
    # AI напишет всю функцию
```

### 3. По комментарию
Напишите комментарий — AI превратит его в код:

```javascript
// получить данные пользователя из API и вернуть имя
```

## Советы для лучших подсказок

1. **Понятные имена переменных** — `userEmail` лучше, чем `x`
2. **Добавляйте комментарии** — они направляют AI
3. **Давайте контекст** — импорты помогают AI понять проект
4. **Будьте конкретны** — «отсортировать массив объектов по дате убывания» лучше, чем «отсортировать»

## Поддерживаемые языки

70+ языков: Python, JavaScript, TypeScript, Java, C++, Go, Rust, PHP, Ruby, Swift и другие.

## Настройки

В Settings можно:
- Включить/выключить автодополнение
- Изменить задержку подсказок
- Фильтровать по типу файлов

## Практика

1. Откройте новый файл на любом языке
2. Напишите 3 комментария с описанием функций
3. Дайте Windsurf сгенерировать каждую
4. Проверьте и отредактируйте результат
