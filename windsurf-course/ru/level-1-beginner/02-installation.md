# Установка и настройка

## Системные требования

- **ОС**: Windows 10+, macOS 11+, Linux (Ubuntu 20.04+)
- **RAM**: минимум 4 ГБ, рекомендуется 8 ГБ
- **Диск**: 500 МБ свободного места
- **Интернет**: обязателен для AI-функций

## Шаг 1: Скачайте Windsurf

1. Перейдите на [codeium.com/windsurf](https://codeium.com/windsurf)
2. Нажмите **Download** для вашей ОС
3. Запустите установщик

## Шаг 2: Создайте аккаунт Codeium

1. Откройте Windsurf после установки
2. Появится окно входа
3. Создайте бесплатный аккаунт или войдите через Google/GitHub
4. Подтвердите email при необходимости

## Шаг 3: Первый запуск

При первом открытии Windsurf:

1. Выберите **цветовую тему** (тёмная или светлая)
2. Укажите **предпочитаемый язык** программирования
3. AI-модель инициализируется — это займёт несколько секунд

## Шаг 4: Проверьте работу AI

1. Создайте файл `test.py`
2. Введите `def hello`
3. Должна появиться подсказка серым текстом
4. Нажмите **Tab**, чтобы принять

Если подсказка появилась — всё работает!

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Нет подсказок AI | Проверьте интернет |
| Не удаётся войти | Попробуйте другой браузер для OAuth |
| Медленная работа | Закройте другие тяжёлые приложения |
