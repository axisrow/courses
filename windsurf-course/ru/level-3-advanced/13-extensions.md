# Расширения

## Совместимость с VS Code

Windsurf поддерживает большинство расширений VS Code — тысячи инструментов.

## Установка расширений

1. Иконка **Расширения** в панели активности (или `Ctrl+Shift+X`)
2. Найдите расширение
3. Нажмите **Install**

## Рекомендуемые расширения

### Разработка
- **ESLint** — линтинг JavaScript
- **Prettier** — форматирование кода
- **GitLens** — продвинутый Git
- **Docker** — управление контейнерами

### Языки
- **Python** — IntelliSense и отладка
- **Go** — официальная поддержка
- **Rust Analyzer** — поддержка Rust
- **Tailwind CSS IntelliSense** — утилиты CSS

### Продуктивность
- **Error Lens** — ошибки прямо в строке
- **Todo Tree** — поиск TODO в коде
- **Path Intellisense** — автодополнение путей

## Расширения + AI

AI Windsurf работает вместе с расширениями:
- ESLint находит проблемы → AI предлагает исправления
- Тесты падают → AI объясняет почему
- Git показывает изменения → AI пишет commit-сообщения

## Управление

- **Отключайте** неиспользуемые расширения для скорости
- **Расширения для workspace** — устанавливайте по проектам
- `@ext` в палитре команд для управления

## Создание своих расширений

Если знаете TypeScript:
1. Генератор Yeoman: `yo code`
2. Разрабатывайте и тестируйте в Windsurf
3. Попросите Cascade помочь с кодом расширения!

## Практика

Установите 3 расширения из списка и посмотрите, как они работают с AI.
