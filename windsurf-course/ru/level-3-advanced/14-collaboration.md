# Командная работа

## Работа в команде

Windsurf поддерживает командные рабочие процессы.

## Интеграция с Git

### Базовый процесс
1. **Добавьте изменения** — нажмите `+` у файлов в Source Control
2. **Коммит** — напишите сообщение и нажмите ✓
3. **Push/Pull** — синхронизация с удалённым репозиторием

### AI-сообщения коммитов
1. Добавьте изменения в stage
2. Нажмите **AI-иконку** в поле сообщения коммита
3. Проверьте и отредактируйте
4. Закоммитьте

## Код-ревью с AI

Перед pull request:
1. Попросите Cascade: «Проверь мои изменения и найди проблемы»
2. Исправьте найденное
3. Попросите: «Напиши описание PR для этих изменений»

## Общие настройки

Храните единые настройки команды:

```json
// .vscode/settings.json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.tabSize": 2
}
```

Закоммитьте этот файл в репозиторий.

## Парное программирование с AI

Windsurf как «третий член команды»:
- Один пишет код, другой ревьюит с Cascade
- Cascade объясняет незнакомый код коллег
- Cascade помогает онбордить новых членов команды
