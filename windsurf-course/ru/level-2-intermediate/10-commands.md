# Команды и горячие клавиши

## Палитра команд

`Ctrl+Shift+P` (или `Cmd+Shift+P` на Mac) — доступ к любой команде. Начните вводить для фильтрации.

## Основные сочетания

### Общие
| Действие | Windows/Linux | Mac |
|----------|--------------|-----|
| Палитра команд | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Быстрое открытие файла | `Ctrl+P` | `Cmd+P` |
| Боковая панель | `Ctrl+B` | `Cmd+B` |
| Терминал | `` Ctrl+` `` | `` Cmd+` `` |
| Настройки | `Ctrl+,` | `Cmd+,` |

### AI-функции
| Действие | Windows/Linux | Mac |
|----------|--------------|-----|
| Открыть Cascade | `Ctrl+Shift+L` | `Cmd+Shift+L` |
| Принять подсказку | `Tab` | `Tab` |
| Следующая подсказка | `Alt+]` | `Option+]` |
| Отклонить подсказку | `Escape` | `Escape` |
| Inline AI-правка | `Ctrl+K` | `Cmd+K` |

### Редактирование
| Действие | Windows/Linux | Mac |
|----------|--------------|-----|
| Мультикурсор | `Alt+Click` | `Option+Click` |
| Дублировать строку | `Shift+Alt+↓` | `Shift+Option+↓` |
| Переместить строку | `Alt+↑/↓` | `Option+↑/↓` |
| Комментирование | `Ctrl+/` | `Cmd+/` |
| Форматирование | `Shift+Alt+F` | `Shift+Option+F` |

## Inline AI-правка (Ctrl+K)

1. Выделите код
2. Нажмите `Ctrl+K`
3. Опишите, что изменить
4. Нажмите Enter

Пример: выделите функцию, нажмите `Ctrl+K`, введите «добавь TypeScript-типы».

## Свои сочетания

**File → Preferences → Keyboard Shortcuts** — настройте любое сочетание.

## Практика

Попробуйте работать 10 минут только с клавиатурой. Без мыши!
