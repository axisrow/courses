# Отладка с AI

## AI-помощь в отладке

Поиск ошибок — одна из сложнейших задач. Windsurf помогает несколькими способами.

## Способ 1: Спросите Cascade

Скопируйте ошибку и вставьте в Cascade:

```
Получаю ошибку: "TypeError: Cannot read property 'map' of undefined"
в UserList.jsx строка 15
```

Cascade:
1. Посмотрит файл
2. Объяснит причину ошибки
3. Предложит исправление

## Способ 2: Подсказки при ошибках

Когда видите красное подчёркивание:
1. Наведите курсор для просмотра ошибки
2. Нажмите **Quick Fix** (или `Ctrl+.`)
3. Windsurf предложит AI-исправления

## Способ 3: Отладка через терминал

Попросите Cascade:
```
Запусти тесты и объясни, почему они падают
```

Cascade может выполнить команды и проанализировать вывод.

## Типичные сценарии

### Ошибки null/undefined
«Почему `user.name` может быть undefined здесь?»

### Логические ошибки
Выделите функцию: «Функция возвращает неправильный результат для отрицательных чисел. Найди баг.»

### Проблемы производительности
«Почему эта функция медленная? Как оптимизировать?»

### Ошибки API
«Получаю 403 от API. Вот мой код...»

## Советы

1. **Делитесь полной ошибкой** — включая stack trace
2. **Описывайте ожидаемое и реальное поведение**
3. **Упоминайте, что уже пробовали**
4. **Используйте breakpoints** — стандартная отладка VS Code работает

## Практика

Найдите баг в коде (или создайте специально). Попросите Cascade помочь найти и исправить.
