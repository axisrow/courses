# 了解界面

## 主要区域

Cursor 的界面有五个主要区域：

### 1. 侧边栏（左侧）
- **Explorer** — 项目文件和文件夹
- **Search** — 跨文件搜索
- **Source Control** — Git 集成
- **Extensions** — 扩展和插件

### 2. 编辑器（中间）
这是你写代码的地方：
- 在**标签页**中打开多个文件
- 分屏查看两个文件
- 输入时看到 AI 建议（灰色文字）

### 3. 聊天面板（右侧）
按 `Ctrl+L`（Mac 上是 `Cmd+L`）打开 AI 聊天面板。

### 4. 终端（底部）
按 `` Ctrl+` `` 切换终端。

### 5. 状态栏（最底部）
显示：语言、行号、Git 分支、账户状态。

## 常用快捷键

| 操作 | Windows/Linux | macOS |
|------|--------------|-------|
| 打开聊天 | `Ctrl+L` | `Cmd+L` |
| AI 内联编辑 | `Ctrl+K` | `Cmd+K` |
| 终端 | `` Ctrl+` `` | `` Cmd+` `` |
| 命令面板 | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| 快速打开文件 | `Ctrl+P` | `Cmd+P` |
| 保存 | `Ctrl+S` | `Cmd+S` |

## 命令面板

按 `Ctrl+Shift+P`，输入你想做的事：
- "theme" → 更换主题
- "font" → 更改字体大小
- "cursor settings" → Cursor 特定设置

## 小结

最重要的快捷键：
- `Ctrl+L` / `Cmd+L` → 与 AI 聊天
- `Ctrl+K` / `Cmd+K` → AI 编辑代码
- `Tab` → 接受 AI 建议
