# 你的第一个项目

## 打开文件夹

Cursor 以**文件夹**（项目）为单位工作。开始方式：

1. 点击 **File → Open Folder**
2. 选择已有文件夹或创建新的（如 `my-first-project`）
3. 点击 **Open**

文件夹会出现在左侧**侧边栏**中。

## 创建你的第一个文件

1. 在侧边栏右键 → **New File**
2. 命名为 `index.html`
3. 开始输入——AI 会立即开始帮助你

### 试一试

输入以下内容，当建议出现时按 `Tab`：

```
<!DOCTYPE html>
```

Cursor 会建议 HTML 的其余结构。

## 保存文件

- `Ctrl+S`（Windows/Linux）或 `Cmd+S`（macOS）
- 或启用 **Auto Save**：File → Auto Save

## 文件浏览器

左侧边栏显示项目中的所有文件：
- 📁 **文件夹** — 点击展开
- 📄 **文件** — 点击打开

## 创建多个文件

尝试创建：
- `style.css` — 样式
- `script.js` — 代码逻辑
- `README.md` — 项目说明

AI 理解文件之间的关系。当你编辑 `script.js` 时，它也知道 `index.html` 的内容。

## 内置终端

需要运行命令？打开内置终端：
- 按 `` Ctrl+` ``（反引号键）
- 或 **Terminal → New Terminal**

## 小结

现在你会了：
- ✅ 打开项目文件夹
- ✅ 创建和保存文件
- ✅ 使用文件浏览器
- ✅ 使用内置终端
