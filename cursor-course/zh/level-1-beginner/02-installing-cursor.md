# 安装 Cursor

## 第一步：下载

1. 访问 [cursor.com](https://cursor.com)
2. 点击 **Download** 按钮
3. Cursor 支持 **Windows**、**macOS** 和 **Linux**

## 第二步：安装

### macOS
- 打开下载的 `.dmg` 文件
- 将 Cursor 拖到"应用程序"文件夹
- 从"应用程序"中打开 Cursor

### Windows
- 运行下载的 `.exe` 安装程序
- 按照安装向导操作
- Cursor 将出现在开始菜单中

### Linux
- 下载 `.AppImage` 文件
- 设为可执行：右键 → 属性 → 权限
- 双击运行

## 第三步：首次启动

首次打开 Cursor 时：

1. **登录或创建账户** — 需要一个免费的 Cursor 账户
2. **选择主题** — 浅色或深色（之后可以更改）
3. **导入 VS Code 设置** — 如果之前使用过 VS Code，Cursor 可以导入你的扩展和设置

## 第四步：选择计划

- **Free** — 每月有限的 AI 请求（适合试用）
- **Pro** — 无限 AI 功能
- **Business / Enterprise** — 适合团队和公司

你可以先用免费计划，之后再升级。

## 第五步：验证

1. 打开 Cursor
2. 创建新文件（File → New File）
3. 开始输入代码——你会看到 AI 建议出现
4. 按 `Tab` 接受建议

🎉 恭喜！Cursor 已安装完毕，可以开始使用了。
