# 选择 AI 模型

## 什么是 AI 模型？

Cursor 提供多个来自不同公司的 **AI 模型**。每个模型有不同的优势。

## 可用模型

- **Anthropic**（Claude）— 细致、详细的代码
- **OpenAI**（GPT）— 通用且广泛使用
- **Google**（Gemini）— 推理能力强
- **xAI**（Grok）— 快速且强大
- **Cursor** — 专为编程优化的自有模型

## 如何切换模型

1. 打开聊天（`Ctrl+L` / `Cmd+L`）
2. 查看聊天顶部的**模型选择器**
3. 点击选择不同的模型

## 该用哪个模型？

| 任务 | 推荐 |
|------|------|
| 快速编辑和自动补全 | Cursor 内置模型（最快） |
| 复杂代码生成 | Claude 或 GPT |
| 理解大文件 | Gemini（大上下文窗口） |
| 一般编码任务 | 都可以——试试比较 |

## 技巧

- **多尝试** — 用不同模型试同一个提示
- **速度 vs. 质量** — 小模型更快，大模型更准确
- **关注更新** — 新模型定期添加

## 小结

- Cursor 提供来自顶级提供商的多种 AI 模型
- 在聊天面板中轻松切换模型
- 不同模型擅长不同任务
- 多实验，找到最适合你的工作流程
