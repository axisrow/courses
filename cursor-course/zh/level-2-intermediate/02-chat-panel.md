# 聊天面板 — 与 AI 对话

## 打开聊天

按 `Ctrl+L`（Mac 上是 `Cmd+L`）打开右侧的**聊天面板**。在这里你可以与 AI 进行完整对话。

## 聊天能做什么？

- **提问**代码相关问题："这个函数是做什么的？"
- **请求修改**："用接口重构 User 类"
- **调试问题**："这个测试为什么失败？"
- **规划功能**："如何构建用户管理的 REST API？"
- **学习概念**："解释什么是 async/await"

## 聊天 vs. Cmd+K

| | Cmd+K | 聊天 |
|--|-------|------|
| 适合 | 快速定向编辑 | 复杂任务和问题 |
| 界面 | 编辑器内联 | 侧面板 |
| 对话 | 单次提示 | 来回对话 |

## 应用聊天中的代码

当 AI 在聊天中建议代码时，你会看到 **Apply** 按钮。点击即可将代码插入正确的文件。你可以审查后接受或拒绝。

## 技巧

1. **要具体** — "修复第42行的登录 bug" 比 "修复 bug" 好
2. **给出上下文** — "我在用 React + TypeScript 开发"
3. **逐步改进** — 如果第一次回答不完美，说明需要什么调整
4. **分步请求** — 一次只要求一件事

## 小结

- `Ctrl+L` / `Cmd+L` 打开聊天面板
- 用聊天处理复杂任务和调试
- 一键应用建议的代码
- 通过来回对话改进结果
