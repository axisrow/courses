# Cmd+K：AI 内联编辑

## 什么是 Cmd+K？

`Cmd+K`（Windows/Linux 上是 `Ctrl+K`）在代码中打开一个**内联提示框**。你描述想要什么，AI 就地编辑代码。

## 如何使用

### 编辑现有代码
1. **选中**你想更改的代码
2. 按 `Cmd+K` / `Ctrl+K`
3. **输入指令**（如"添加错误处理"）
4. 按 `Enter`
5. **审查**更改——接受或拒绝

### 生成新代码
1. 将光标放在目标位置
2. 按 `Cmd+K` / `Ctrl+K`（不选择任何内容）
3. 描述你需要的（如"创建一个登录表单组件"）

## 好的指令

要具体：

| ❌ 模糊 | ✅ 具体 |
|--------|---------|
| "修复它" | "处理列表为空的情况" |
| "改好一点" | "添加输入验证和错误消息" |
| "改样式" | "把按钮改成蓝色圆角" |

## 审查更改

AI 修改后，你会看到 **diff 视图**：
- 🟢 绿色行 = 新增代码
- 🔴 红色行 = 删除代码

你可以：**接受**、**拒绝**，或**修改提示重试**。

## 实际示例

- 选中函数 → Cmd+K → "添加 try/catch 和清晰的错误消息"
- 选中代码 → Cmd+K → "从 JavaScript 转换为 TypeScript"
- 选中代码 → Cmd+K → "为每个步骤添加注释"

## 小结

- `Cmd+K` / `Ctrl+K` = AI 内联编辑
- 选中代码进行编辑，或放置光标生成新代码
- 指令要具体
- 接受前务必审查 diff
