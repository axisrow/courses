# Tab-автодополнение — ваш AI-помощник

## Что это такое?

Когда вы печатаете код, Cursor предсказывает, что вы хотите написать дальше. Подсказки появляются **серым текстом**. Нажмите `Tab`, чтобы принять.

Это самый простой и частый способ использования AI в Cursor.

## Как это работает

1. **Вы печатаете** несколько символов или строку кода
2. **Cursor предлагает** следующую строку, блок или целую функцию
3. **Нажмите Tab** для принятия, или продолжайте печатать, чтобы проигнорировать

AI учитывает:
- Что вы уже написали
- Другие файлы в проекте
- Язык программирования
- Общие паттерны кода

## Примеры

### Пример 1: Написание функции
Вы вводите:
```python
def calculate_total(items):
```
Cursor предлагает:
```python
    total = 0
    for item in items:
        total += item.price
    return total
```

### Пример 2: Сначала комментарий
Вы пишете комментарий:
```javascript
// функция проверки валидности email
```
Cursor предлагает готовую функцию ниже.

### Пример 3: Повторяющиеся паттерны
Если вы написали:
```python
user.first_name = data["first_name"]
user.last_name = data["last_name"]
```
Cursor автоматически предскажет следующее поле.

## Советы для лучших подсказок

1. **Пишите понятные комментарии** — AI читает их
2. **Используйте описательные имена** — `calculateTotalPrice` лучше, чем `calc`
3. **Держите связанные файлы открытыми** — Cursor использует их как контекст

## Принятие частичных подсказок

- **Tab** — принять всю подсказку
- **Ctrl+→** / **Cmd+→** — принимать по слову
- **Escape** — отклонить подсказку

## Итоги

- AI-подсказки появляются серым текстом
- Нажмите `Tab` для принятия
- Пишите понятные комментарии для лучших результатов
- Это только начало — впереди более мощные инструменты!
