# Знакомство с интерфейсом

## Основные области

Интерфейс Cursor состоит из пяти зон:

### 1. Боковая панель (слева)
- **Explorer** — файлы и папки проекта
- **Search** — поиск по всем файлам
- **Source Control** — интеграция с Git
- **Extensions** — расширения и плагины

### 2. Редактор (по центру)
Здесь вы пишете код. Можно:
- Открывать файлы во **вкладках**
- Разделить редактор для просмотра двух файлов одновременно
- Видеть AI-подсказки (серый текст)

### 3. Панель чата (справа)
Нажмите `Ctrl+L` (`Cmd+L` на Mac) — откроется AI-чат для общения с искусственным интеллектом.

### 4. Терминал (снизу)
Нажмите `` Ctrl+` `` для переключения терминала.

### 5. Строка состояния (внизу)
Показывает: язык, номер строки, ветку Git, статус аккаунта.

## Основные горячие клавиши

| Действие | Windows/Linux | macOS |
|----------|--------------|-------|
| Открыть чат | `Ctrl+L` | `Cmd+L` |
| Редактирование через AI | `Ctrl+K` | `Cmd+K` |
| Терминал | `` Ctrl+` `` | `` Cmd+` `` |
| Палитра команд | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Быстрое открытие файла | `Ctrl+P` | `Cmd+P` |
| Сохранить | `Ctrl+S` | `Cmd+S` |

## Палитра команд

Нажмите `Ctrl+Shift+P` и введите, что хотите сделать:
- "theme" → сменить тему
- "font" → изменить размер шрифта
- "cursor settings" → настройки Cursor

## Итоги

Самые важные сочетания клавиш:
- `Ctrl+L` / `Cmd+L` → Чат с AI
- `Ctrl+K` / `Cmd+K` → Редактирование кода через AI
- `Tab` → Принять подсказку AI
