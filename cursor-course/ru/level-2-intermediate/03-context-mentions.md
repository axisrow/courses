# Контекст: @files, @docs, @web

## Почему контекст важен

AI работает лучше, когда знает, с чем вы работаете. **Упоминания** через символ `@` направляют AI к нужной информации.

## Типы упоминаний

### @files — Ссылка на файлы
```
@utils.py Как улучшить обработку ошибок в этом файле?
```

### @folder — Целые папки
```
@src/components Создай новый компонент в стиле существующих
```

### @docs — Документация
```
@docs Найди в документации Next.js как использовать серверные компоненты
```

### @web — Поиск в интернете
```
@web Какой актуальный способ аутентификации в React в 2025?
```

### @codebase — Поиск по всему проекту
```
@codebase Где настроено подключение к базе данных?
```

## Комбинирование упоминаний

Можно использовать несколько упоминаний в одном запросе:
```
@api/routes.py @models/user.py Добавь эндпоинт для удаления пользователей по существующему шаблону
```

## Когда что использовать

| Ситуация | Используйте |
|----------|-------------|
| Редактировать конкретный файл | `@filename` |
| Следовать паттернам проекта | `@folder` |
| Использовать документацию фреймворка | `@docs` |
| Нужна свежая информация | `@web` |
| Найти что-то в проекте | `@codebase` |

## Итоги

- Используйте `@` для точного контекста
- `@files` для файлов, `@web` для интернета
- `@codebase` ищет по всему проекту
- Лучший контекст = лучшие ответы AI
