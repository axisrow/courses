# Панель чата — разговор с AI

## Открытие чата

Нажмите `Ctrl+L` (`Cmd+L` на Mac), чтобы открыть **панель чата** справа. Здесь вы ведёте полноценный диалог с AI.

## Что можно делать в чате?

- **Задавать вопросы** о коде: «Что делает эта функция?»
- **Просить изменения**: «Перепиши класс User с интерфейсами»
- **Отлаживать**: «Почему этот тест падает?»
- **Планировать**: «Как структурировать REST API для управления пользователями?»
- **Учиться**: «Объясни, что такое async/await»

## Чат vs. Cmd+K

| | Cmd+K | Чат |
|--|-------|-----|
| Лучше для | Быстрых точечных правок | Сложных задач и вопросов |
| Интерфейс | Встроен в редактор | Боковая панель |
| Разговор | Один запрос | Диалог туда-сюда |

## Применение кода из чата

Когда AI предлагает код, появляется кнопка **Apply**. Нажмите — код вставится в нужный файл. Вы можете проверить и принять/отклонить.

## Советы

1. **Будьте конкретны** — «Исправь баг входа на строке 42» лучше, чем «исправь баг»
2. **Давайте контекст** — «Я делаю React-приложение на TypeScript»
3. **Уточняйте** — Если первый ответ неидеален, скажите, что поправить
4. **Дробите** — Просите одно за раз

## Итоги

- `Ctrl+L` / `Cmd+L` открывает панель чата
- Используйте чат для сложных задач и отладки
- Применяйте предложенный код одним нажатием
- Ведите диалог для уточнения результатов
