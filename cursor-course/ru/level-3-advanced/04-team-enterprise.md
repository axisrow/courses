# Командная работа и Enterprise

## Cursor для команд

Cursor используют не только одиночки. Более половины компаний из Fortune 500 работают с ним, включая Salesforce (20 000+ разработчиков) и NVIDIA (40 000+ инженеров).

## Планы для бизнеса

- **Business** — Управление командой, единый биллинг, аналитика
- **Enterprise** — SSO, SAML, админ-контроль, соответствие требованиям, выделенная поддержка

## Общие правила для команды

### Общий .cursorrules
Добавьте `.cursorrules` в Git-репозиторий — все члены команды получат одинаковое поведение AI:
```
# Стандарты кодирования команды
- Используй нашу библиотеку компонентов @company/ui
- Следуй шаблонам API из docs/api-guide.md
- Все PR должны включать тесты
```

## Безопасность и приватность

Функции Enterprise:
- **Режим приватности** — ваш код не хранится на серверах Cursor
- **SOC 2** — безопасность корпоративного уровня
- **Self-hosted** — всё на вашей инфраструктуре
- **Админ-контроль** — управление моделями и функциями

## Адаптация новых сотрудников

Cursor помогает новым разработчикам понять код:
1. Откройте проект → Чат → «Объясни архитектуру этого проекта»
2. `@codebase` «Где находится главная точка входа?»
3. `@docs/` «Какие ключевые паттерны используются?»

## Итоги

- Cursor масштабируется от одного человека до 40 000+ разработчиков
- Делитесь `.cursorrules` через Git
- Enterprise-функции покрывают безопасность и админ-контроль
- AI помогает новым сотрудникам быстрее освоиться
