# Режим агента — полная автономия

## Что такое режим агента?

Режим агента — самая мощная функция Cursor. Вместо того чтобы вы направляли каждый шаг, AI **берёт инициативу**:
- Читает и ищет по кодовой базе
- Создаёт и редактирует файлы
- Запускает команды в терминале
- Устанавливает пакеты
- Запускает тесты и исправляет ошибки

## Как использовать

1. Откройте чат (`Ctrl+L` / `Cmd+L`)
2. Выберите режим **Agent** (переключатель вверху)
3. Опишите задачу на обычном языке
4. AI работает — вы наблюдаете за каждым шагом

## Ползунок автономии

Как описал Андрей Карпатый, Cursor имеет «ползунок автономии»:
- **Tab-дополнение** → Минимальная автономия (вы ведёте)
- **Cmd+K** → Средняя автономия (вы направляете, AI выполняет)
- **Режим агента** → Максимальная автономия (AI ведёт, вы проверяете)

## Что может режим агента

- «Настрой проект Node.js с Express, TypeScript и PostgreSQL»
- «Напиши тесты для всех API-эндпоинтов»
- «Найди и исправь баг, вызывающий ошибки при входе»
- «Перепиши систему аутентификации на JWT»

## Безопасность

Режим агента просит **ваше одобрение** перед:
- Запуском команд в терминале
- Серьёзными изменениями
- Установкой пакетов

Последнее слово всегда за вами.

## Лучшие практики

1. **Начните с чёткого плана** — опишите задачу целиком
2. **Проверяйте каждый шаг** — не принимайте всё слепо
3. **Используйте Git** — сделайте коммит перед началом
4. **Дробите большие задачи** — «Собери приложение целиком» слишком размыто
5. **Давайте контекст** — упоминайте файлы через `@`

## Итоги

- Режим агента = максимальная автономия AI
- AI планирует, пишет код, запускает команды, исправляет ошибки
- Вы проверяете и одобряете каждый шаг
- Лучше всего подходит для крупных, чётко описанных задач
