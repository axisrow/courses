# Installing Cursor

## Step 1: Download

1. Go to [cursor.com](https://cursor.com)
2. Click the **Download** button
3. Cursor is available for **Windows**, **macOS**, and **Linux**

The download starts automatically for your operating system.

## Step 2: Install

### On macOS
- Open the downloaded `.dmg` file
- Drag Cursor to your Applications folder
- Open Cursor from Applications

### On Windows
- Run the downloaded `.exe` installer
- Follow the installation wizard
- Cursor will appear in your Start menu
