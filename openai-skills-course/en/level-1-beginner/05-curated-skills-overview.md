# Curated Skills Overview

## Introduction

Curated skills are skills reviewed and approved by the OpenAI team. They live in the `.curated` folder of the openai/skills repository and are installed via skill-installer.

## Installing a Curated Skill

```
$skill-installer gh-fix-ci
```

After installation, **restart Codex** for the skill to become available.

## Summary

- Curated skills are verified by OpenAI and production-ready
- There are 30+ curated skills covering files, deployment, integrations, and more
- Installation takes one command via skill-installer
- Restart Codex after installing a skill
