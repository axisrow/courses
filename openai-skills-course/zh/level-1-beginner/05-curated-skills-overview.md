# 精选技能概览

## 简介

精选（curated）技能是经OpenAI团队审核和批准的技能。它们位于openai/skills仓库的`.curated`文件夹中。

## 精选技能列表

| 技能 | 描述 |
|------|------|
| **figma** | 通过MCP服务器使用Figma设计 |
| **screenshot** | 截屏 |
| **gh-fix-ci** | 诊断和修复GitHub Actions CI |
| **gh-address-comments** | 处理GitHub PR评论 |
| **pdf** | 处理PDF文档 |
| **imagegen** | 生成图像 |
| **sora** | 通过Sora生成视频 |
| **speech** | 文本转语音 |
| **transcribe** | 音频转录 |
| **jupyter-notebook** | Jupyter笔记本 |
| **playwright** | Playwright浏览器自动化 |
| **vercel-deploy** | 部署到Vercel |
| **netlify-deploy** | 部署到Netlify |
| **cloudflare-deploy** | 部署到Cloudflare |
| **linear** | Linear集成 |
| **sentry** | Sentry集成 |
| **notion-meeting-intelligence** | Notion会议笔记 |
| **security-best-practices** | 安全最佳实践 |
| **develop-web-game** | Web游戏开发 |

## 安装精选技能

```
$skill-installer gh-fix-ci
```

安装后**重启Codex**以使技能生效。

## 总结

- 精选技能经OpenAI验证，可直接使用
- 超过30个精选技能，涵盖文件处理、部署、集成等
- 通过skill-installer一条命令安装
- 安装后需要重启Codex
