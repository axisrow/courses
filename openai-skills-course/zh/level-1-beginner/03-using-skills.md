# 如何使用技能

## 简介

Codex中的技能自动工作。您不需要手动"启用"技能——Codex会自动确定哪个技能适合您的请求。

## Codex如何选择技能

当您给Codex一个任务时，它经历三个步骤：

1. **读取元数据** — 检查所有已安装技能的`name`和`description`
2. **选择合适的** — 找到描述与您的任务匹配的技能
3. **加载指令** — 读取SKILL.md并执行任务

## 使用示例

### 示例1：截屏

您写道：
> "截取我的屏幕"

Codex找到`screenshot`技能并执行任务。

### 示例2：GitHub

您写道：
> "检查我的PR为什么CI失败了"

Codex激活`gh-fix-ci`，检查状态并提出修复方案。

### 示例3：图像生成

您写道：
> "为我的项目生成一个logo"

Codex使用`imagegen`技能。

## 按名称调用技能

您可以使用`$`前缀明确指定技能：

```
$screenshot 截取全屏
```

## 查看已安装的技能

```
$skill-installer
```

## 总结

- Codex根据您的请求自动选择技能
- 技能通过元数据中的描述触发
- 可以通过`$技能名`显式调用技能
- 通过skill-installer查看技能列表
