# Как использовать навыки

## Введение

Навыки в Codex работают автоматически. Вам не нужно вручную «включать» навык — Codex сам определяет, какой навык подходит для вашего запроса. В этом уроке вы узнаете, как это происходит.

## Как Codex выбирает навык

Когда вы даёте задание Codex, он проходит три шага:

1. **Читает метаданные** — смотрит на `name` и `description` всех установленных навыков
2. **Выбирает подходящий** — находит навык, чья описание совпадает с вашей задачей
3. **Загружает инструкции** — читает SKILL.md и выполняет задачу

## Примеры использования

### Пример 1: Скриншот

Вы пишете Codex:
> «Сделай скриншот моего экрана»

Codex находит навык `screenshot` и выполняет задачу.

### Пример 2: Работа с GitHub

Вы пишете:
> «Посмотри, почему CI падает в моём PR»

Codex активирует навык `gh-fix-ci`, который проверит статус проверок и предложит исправления.

### Пример 3: Генерация изображений

Вы пишете:
> «Сгенерируй логотип для моего проекта»

Codex использует навык `imagegen`.

## Вызов навыка по имени

Вы можете явно указать навык, используя префикс `$`:

```
$screenshot сделай снимок всего экрана
```

## Просмотр установленных навыков

Чтобы увидеть список доступных навыков, используйте skill-installer:

```
$skill-installer
```

Codex покажет список навыков с пометками, какие уже установлены.

## Итоги

- Codex автоматически выбирает навык на основе вашего запроса
- Навык активируется по описанию (description) в его метаданных
- Можно вызвать навык явно через `$имя-навыка`
- Список навыков доступен через skill-installer
