# Установка Codex

## Введение

Codex — это AI-агент от OpenAI, который работает в командной строке (CLI). Он использует навыки (Skills) для выполнения задач. В этом уроке мы установим Codex на ваш компьютер.

## Требования

- Компьютер с macOS, Windows или Linux
- Установленный Node.js (версия 18 или новее)
- Аккаунт OpenAI с API-ключом

## Пошаговая установка

### Шаг 1: Проверьте Node.js

Откройте терминал и введите:

```bash
node --version
```

Если версия ниже 18 или Node.js не установлен, скачайте его с [nodejs.org](https://nodejs.org).

### Шаг 2: Установите Codex

```bash
npm install -g @openai/codex
```

### Шаг 3: Настройте API-ключ

```bash
export OPENAI_API_KEY="ваш-ключ"
```

Чтобы ключ сохранялся между сессиями, добавьте эту строку в файл `~/.bashrc` или `~/.zshrc`.

### Шаг 4: Проверьте установку

```bash
codex --help
```

Вы должны увидеть список доступных команд.

## Где хранятся навыки

После установки Codex создаёт домашнюю директорию:

```
~/.codex/
└── skills/     — сюда устанавливаются навыки
```

Системные навыки (`.system`) уже встроены в Codex и доступны сразу.

## Итоги

- Codex устанавливается через npm
- Для работы нужен API-ключ OpenAI
- Навыки хранятся в `~/.codex/skills/`
- Системные навыки доступны сразу после установки
