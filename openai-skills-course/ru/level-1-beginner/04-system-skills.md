# Системные навыки (.system)

## Введение

Системные навыки — это навыки, которые встроены в Codex и доступны сразу после установки. Они находятся в папке `.system` и не требуют отдельной установки.

## Какие системные навыки существуют

На данный момент в Codex встроены два системных навыка:

### 1. skill-installer

**Что делает:** Устанавливает новые навыки в Codex из репозитория openai/skills или из любого GitHub-репозитория.

**Когда используется:**
- Вы хотите узнать, какие навыки доступны
- Вы хотите установить курированный или экспериментальный навык
- Вы хотите установить навык из стороннего репозитория

**Пример использования:**

```
$skill-installer покажи доступные навыки
```

```
$skill-installer установи gh-fix-ci
```

### 2. skill-creator

**Что делает:** Помогает создавать новые навыки. Проводит через весь процесс: от идеи до готового навыка.

**Когда используется:**
- Вы хотите создать собственный навык
- Вы хотите обновить существующий навык

**Пример использования:**

```
$skill-creator создай навык для работы с базой данных
```

## Как устроен skill-installer

Skill-installer содержит:

```
skill-installer/
├── SKILL.md                    — инструкции для Codex
├── agents/openai.yaml          — метаданные для UI
├── scripts/
│   ├── list-skills.py          — показывает список навыков
│   ├── install-skill-from-github.py  — устанавливает навык
│   └── github_utils.py         — вспомогательные функции
└── LICENSE.txt
```

## Как устроен skill-creator

Skill-creator содержит:

```
skill-creator/
├── SKILL.md                         — подробное руководство
├── agents/openai.yaml               — метаданные
├── scripts/
│   ├── init_skill.py                — создаёт шаблон навыка
│   ├── generate_openai_yaml.py      — генерирует openai.yaml
│   └── quick_validate.py            — проверяет навык
└── references/
    └── openai_yaml.md               — справка по openai.yaml
```

## Итоги

- В Codex встроены два системных навыка: skill-installer и skill-creator
- skill-installer устанавливает навыки из GitHub
- skill-creator помогает создавать свои навыки
- Системные навыки не нужно устанавливать — они работают сразу
