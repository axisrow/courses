# Стандарт AgentSkills (agentskills.io)

## Введение

AgentSkills — это открытый стандарт для описания навыков AI-агентов. Он опубликован на [agentskills.io](https://agentskills.io) и описывает единый формат, который могут использовать разные AI-агенты, не только Codex.

## Зачем нужен стандарт

Без стандарта каждый AI-агент использует свой формат навыков. Это приводит к тому, что навык, написанный для одного агента, нельзя использовать в другом.

AgentSkills решает эту проблему: **написал один раз — используй везде**.

## Основные принципы стандарта

### 1. Папка как навык

Навык — это обычная папка с файлами. Не пакет, не контейнер, не образ — просто папка.

### 2. SKILL.md как точка входа

Каждый навык обязан иметь файл `SKILL.md` в корне папки. Это единственный обязательный файл.

### 3. YAML frontmatter

SKILL.md начинается с YAML-метаданных:

```yaml
---
name: имя-навыка
description: Описание навыка
---
```

### 4. Папка agents/ для метаданных агентов

Каждый агент может иметь свой файл конфигурации:

```
agents/
├── openai.yaml      — для Codex
├── anthropic.yaml   — для Claude (в будущем)
└── другой.yaml      — для другого агента
```

### 5. Ресурсы

Стандартные папки для ресурсов:

- `scripts/` — исполняемые скрипты
- `references/` — справочные материалы
- `assets/` — файлы для вывода (шаблоны, иконки)

## Совместимость

Навыки, созданные по стандарту AgentSkills, совместимы с:

- OpenAI Codex (сейчас)
- Потенциально другими AI-агентами в будущем
