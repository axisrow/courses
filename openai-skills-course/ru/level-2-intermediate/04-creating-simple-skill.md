# Создание простого навыка

## Введение

В этом уроке мы создадим свой первый навык с нуля. Мы будем использовать системный навык skill-creator, который проведёт нас через весь процесс.

## Шаг 1: Придумайте идею

Хороший навык решает конкретную повторяющуюся задачу. Примеры:

- Форматирование отчётов по шаблону компании
- Проверка кода по стандартам команды
- Генерация документации в определённом стиле

Для этого урока создадим навык **greeting** — приветствие пользователей на разных языках.

## Шаг 2: Инициализация навыка

Используйте скрипт `init_skill.py` из skill-creator:

```bash
python scripts/init_skill.py greeting --path ./skills
```

Это создаст структуру:

```
skills/greeting/
├── SKILL.md
└── agents/
    └── openai.yaml
```

## Шаг 3: Напишите SKILL.md

Откройте `SKILL.md` и заполните:

```markdown
---
name: greeting
description: Приветствие пользователей на разных языках. Используйте, когда пользователь просит поприветствовать кого-то, создать приветственное сообщение или перевести приветствие.
---

# Greeting

Создание приветствий на разных языках.

## Поддерживаемые языки

- Русский: «Привет, {имя}! Добро пожаловать!»
- English: "Hello, {name}! Welcome!"
- 中文: "{name}，你好！欢迎！"
- Español: "¡Hola, {name}! ¡Bienvenido!"

## Инструкции

1. Определить целевой язык из контекста запроса
2. Подставить имя пользователя в шаблон
3. Если язык не указан, использовать английский
```

## Шаг 4: Настройте openai.yaml

Файл `agents/openai.yaml` содержит метаданные для интерфейса:

```yaml
display_name: "Greeting"
short_description: "Приветствия на разных языках"
default_prompt: "Поприветствуй пользователя"
```

## Шаг 5: Проверьте навык

Запустите валидацию:

```bash
python scripts/quick_validate.py ./skills/greeting
```

Скрипт проверит:
- Правильность YAML frontmatter
- Наличие обязательных полей
- Соответствие правилам именования

## Шаг 6: Установите навык

Скопируйте папку навыка в домашнюю директорию Codex:

```bash
cp -r ./skills/greeting ~/.codex/skills/
```

Перезапустите Codex — навык готов к использованию!

## Итоги

- Создание навыка начинается с `init_skill.py`
- Главный файл — SKILL.md с frontmatter и инструкциями
- `openai.yaml` задаёт метаданные для интерфейса
- `quick_validate.py` проверяет навык на ошибки
- После копирования в `~/.codex/skills/` навык доступен
