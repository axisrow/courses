# Экспериментальные навыки

## Введение

Помимо курированных навыков, в репозитории openai/skills есть папка `.experimental`. Здесь находятся навыки, которые ещё не прошли полную проверку, но уже могут быть полезны.

## Чем отличаются от курированных

| | Курированные | Экспериментальные |
|---|---|---|
| Проверка | Проверены OpenAI | Не полностью проверены |
| Стабильность | Высокая | Может меняться |
| Поддержка | Официальная | Сообщество |
| Папка | `.curated` | `.experimental` |

## Как установить экспериментальный навык

Укажите папку `.experimental` явно:

```
$skill-installer install the create-plan skill from the .experimental folder
```

Или укажите полный URL на GitHub:

```
$skill-installer install https://github.com/openai/skills/tree/main/skills/.experimental/create-plan
```

## Как посмотреть список экспериментальных навыков

```
$skill-installer покажи экспериментальные навыки
```

Skill-installer вызовет `list-skills.py --path skills/.experimental` и покажет результат.

## Предупреждения

- Экспериментальные навыки могут работать нестабильно
- Они могут быть удалены или перемещены в курированные
- Используйте их на свой страх и риск
- Если навык вам понравился — помогите улучшить его через контрибьюцию

## Итоги

- Экспериментальные навыки — это навыки от сообщества в стадии разработки
- Устанавливаются через skill-installer с указанием папки `.experimental`
- Менее стабильны, чем курированные, но могут быть очень полезны
