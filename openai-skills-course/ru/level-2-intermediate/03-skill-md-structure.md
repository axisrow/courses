# Структура SKILL.md

## Введение

SKILL.md — это главный файл каждого навыка. Именно его Codex читает, чтобы понять, что делать и как. В этом уроке мы разберём его структуру.

## Общая структура

SKILL.md состоит из двух частей:

1. **YAML frontmatter** — метаданные (имя и описание)
2. **Тело (body)** — инструкции в формате Markdown

## YAML Frontmatter

Frontmatter находится в самом начале файла между `---`:

```yaml
---
name: my-skill
description: Описание навыка — когда и для чего он используется. Это самая важная часть, потому что именно по ней Codex решает, нужно ли активировать навык.
---
```

### Поле `name`

- Имя навыка
- Используйте строчные буквы, цифры и дефисы
- Примеры: `pdf`, `gh-fix-ci`, `notion-meeting-intelligence`

### Поле `description`

- Самая важная часть навыка!
- Описывает **что** делает навык и **когда** его использовать
- Codex читает это описание для каждого запроса, чтобы решить, какой навык активировать
- Будьте подробны: перечислите все сценарии использования

**Хороший пример:**
```yaml
description: "Создание, редактирование и анализ документов .docx с поддержкой отслеживания изменений, комментариев и форматирования. Используйте, когда нужно: (1) создать новый документ, (2) изменить содержимое, (3) работать с отслеживанием изменений."
```

## Тело (Body)

После frontmatter идут инструкции в Markdown:

```markdown
# Название навыка

## Обзор
Краткое описание того, что делает навык.

## Как использовать
Пошаговые инструкции.

## Скрипты
Описание доступных скриптов и когда их запускать.
```

### Важные правила для тела:

- **Краткость** — контекстное окно ограничено, каждое слово должно быть полезным
- **Не дублируйте** — не повторяйте «когда использовать» из description
- **Императивный стиль** — пишите «Сделай», «Проверь», а не «Навык делает»
- **До 500 строк** — если больше, выносите в `references/`

## Пример полного SKILL.md

```markdown
---
name: pdf-rotate
description: Поворот страниц PDF-файлов. Используйте, когда пользователь просит повернуть, перевернуть или изменить ориентацию страниц в PDF.
---

# PDF Rotate

Поворот страниц в PDF-документах.

## Быстрый старт

Запустить скрипт:
\`\`\`bash
python scripts/rotate_pdf.py input.pdf --angle 90 --output rotated.pdf
\`\`\`

## Параметры

- `--angle` — угол поворота (90, 180, 270)
- `--pages` — номера страниц (по умолчанию все)
- `--output` — путь к результату
```

## Итоги

- SKILL.md состоит из frontmatter (name + description) и тела (инструкции)
- Description — самая важная часть, определяет когда навык активируется
- Тело должно быть кратким и практичным
- Используйте императивный стиль
