## Расширение и вклад в RAPTOR

### RAPTOR создан для расширения

Модульная архитектура позволяет легко добавлять новые возможности.

### Добавление нового пакета

1. Создайте директорию в `packages/`:

```
packages/your_capability/
  ├── __init__.py
  ├── analyzer.py
  └── config.py
```

2. Зарегистрируйте в `raptor.py`
3. Создайте команду Claude в `.claude/commands/your-command.md`

### Создание навыков

Команда `/create-skill` позволяет сохранять подходы:

```
/create-skill
```

Навыки хранятся в `.claude/skills/` и доступны в будущих сессиях.

### Экспертные персоны

Вы можете создавать новые персоны, добавляя markdown-файлы в систему Tier 2. Персона определяет:

- **Личность** — кто этот эксперт
- **Подход** — как он анализирует проблемы
- **Области фокуса** — специализация
- **Стиль коммуникации** — как представляет находки

### Вклад в сообщество

RAPTOR с открытым исходным кодом (лицензия MIT):

- Новые правила Semgrep
- Запросы CodeQL для специфичных языков
- Фаззинг-обвязки для популярных библиотек
- Интеграции с новыми AI-провайдерами
- Улучшения документации и переводы
