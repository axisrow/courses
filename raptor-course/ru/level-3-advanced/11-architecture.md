## Архитектура RAPTOR: глубокое погружение

### Многоуровневый дизайн

RAPTOR использует архитектуру **прогрессивного раскрытия** — загружает только то, что нужно:

#### Уровень 1: Bootstrap (CLAUDE.md)
Загружается всегда. Содержит основную систему принятия решений и базовые команды.

#### Уровень 2: Навыки Tier 1
Автозагрузка при необходимости:
- Состязательное мышление
- Руководство по анализу
- Процедуры восстановления

#### Уровень 3: Экспертные персоны Tier 2
Загружаются по запросу. Девять специализированных экспертных персон:

1. **Марк Дауд** — глубокий аудит кода
2. **Чарли Миллер / Хальвар Флейк** — эксплуатация бинарных файлов
3. **Исследователь безопасности** — общие исследования
4. **Инженер патчей** — безопасное исправление кода
5. **Пентестер** — методология наступательного тестирования
6. **Стратег фаззинга** — оптимизация фаззинга
7. **Специалист по бинарной эксплуатации** — низкоуровневая эксплуатация
8. **Аналитик CodeQL Dataflow** — написание запросов
9. **Аналитик находок CodeQL** — интерпретация результатов

#### Уровень 4: Агенты
Автономные агенты, например **Специалист по наступательной безопасности**.

### Python-слой выполнения

```
raptor.py          → Единый лаунчер (точка входа CLI)
packages/          → 9 пакетов безопасности
  ├── autonomous/         → Оркестрация полного процесса
  ├── binary_analysis/    → Анализ бинарных файлов
  ├── codeql/             → Интеграция с CodeQL
  ├── exploit_feasibility/→ Оценка эксплуатируемости
  ├── exploitability_validation/ → Движок валидации
  ├── fuzzing/            → Интеграция с AFL++
  ├── llm_analysis/       → AI-анализ
  ├── recon/              → Разведка
  └── sca/                → Анализ состава ПО
core/              → Общие утилиты
engine/            → Правила и запросы
```

### Бюджет токенов

Прогрессивная загрузка экономит контекст:
- Bootstrap: ~360 токенов
- Tier 1: ~925 токенов
- Полная загрузка с персонами: до ~2500 токенов

### Главный вывод

Архитектура RAPTOR модульна и эффективна. Понимание уровней помогает использовать нужную глубину инструментов для каждой задачи.
