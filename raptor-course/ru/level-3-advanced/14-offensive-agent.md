## Агент наступательной безопасности

### Что это?

RAPTOR включает **автономного агента наступательной безопасности** на основе SecOpsAgentKit, который может самостоятельно планировать и выполнять операции тестирования.

### Возможности

- **Тестирование веб-приложений** — SQLi, XSS, CSRF, обход аутентификации
- **Сетевое тестирование на проникновение** — сканирование портов, перечисление сервисов
- **Бинарная эксплуатация** — переполнения буфера, форматные строки, ROP-цепочки
- **Фаззинг** — целевые кампании фаззинга
- **Разработка эксплойтов** — полная генерация PoC
- **Ревью кода** — аудит с позиции атакующего

### Как использовать

```
"Use the offensive security specialist agent to test this application"
```

### Модель безопасности

| Тип операции | Поведение |
|-------------|-----------|
| **Безопасные** (сканирование, перечисление) | Автовыполнение |
| **Потенциально опасные** (эксплуатация) | Требуют подтверждения |
| **Деструктивные** | Заблокированы по умолчанию |

### SecOpsAgentKit

Навыки агента хранятся в `.claude/skills/SecOpsAgentKit/`. Они модульны и расширяемы.

### Комбинация с другими командами

1. `/scan` — найти уязвимости
2. Агент — попытаться эксплуатировать
3. `/patch` — сгенерировать исправления
4. Агент — проверить патчи

### Главный вывод

Агент наступательной безопасности привносит автономный пентест в RAPTOR. Он мощный, но работает в рамках ограничений безопасности, всегда сохраняя контроль человека.
