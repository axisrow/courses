## 攻击性安全专家代理

### 它是什么？

RAPTOR 包含一个由 SecOpsAgentKit 驱动的**自主攻击性安全代理**，可以独立规划和执行安全测试操作。

### 能力

- **Web 应用测试** — SQLi、XSS、CSRF、认证绕过
- **网络渗透测试** — 端口扫描、服务枚举
- **二进制利用** — 缓冲区溢出、格式化字符串、ROP 链
- **模糊测试** — 有针对性的模糊测试活动
- **漏洞利用开发** — 完整的 PoC 生成
- **代码审查** — 以攻击者视角进行安全审查

### 如何使用

```
"Use the offensive security specialist agent to test this application"
```

### 安全模型

| 操作类型 | 行为 |
|----------|------|
| **安全**（扫描、枚举）| 自动执行 |
| **潜在危险**（利用）| 需要明确确认 |
| **破坏性** | 默认阻止 |

### SecOpsAgentKit

代理的技能存储在 `.claude/skills/SecOpsAgentKit/` 中，模块化且可扩展。

### 与其他命令组合

1. `/scan` — 发现漏洞
2. 代理 — 尝试利用
3. `/patch` — 生成修复
4. 代理 — 验证补丁

### 核心要点

攻击性安全代理为 RAPTOR 带来自主渗透测试能力。它很强大，但在安全护栏内运行，始终让人类控制危险操作。
