# Что такое gogcli?

## Обзор

**gogcli** (команда: `gog`) — это инструмент, который позволяет работать с сервисами Google — Gmail, Календарём, Диском, Контактами, Задачами, Таблицами и другими — прямо из терминала (командной строки), без открытия браузера.

Представьте себе пульт управления вашим аккаунтом Google, в который вы вводите команды вместо нажатия кнопок.

## Зачем это нужно?

- **Скорость**: проверить почту или создать событие одной командой вместо навигации по веб-страницам.
- **Автоматизация**: писать скрипты, которые автоматически обрабатывают письма, обновляют таблицы или управляют календарём.
- **Несколько аккаунтов**: легко переключаться между личным и рабочим аккаунтами Google.
- **Интеграция с другими инструментами**: передавать данные Google в другие программы командной строки.

## Что умеет gogcli?

| Сервис | Возможности |
|--------|------------|
| **Gmail** | Поиск, чтение, отправка писем, управление ярлыками и черновиками |
| **Календарь** | Просмотр, создание, обновление событий, проверка доступности |
| **Диск** | Просмотр, загрузка, скачивание, обмен файлами |
| **Контакты** | Поиск, создание, обновление контактов |
| **Задачи** | Управление списками задач и задачами |
| **Таблицы** | Чтение и запись данных в электронные таблицы |
| **Документы** | Экспорт и создание документов |
| **Презентации** | Экспорт и создание презентаций |
| **Формы** | Создание форм и просмотр ответов |
| **Чат** | Отправка сообщений в Google Chat (только Workspace) |

## Для кого этот курс?

Этот курс для всех, кто хочет научиться работать с gogcli с нуля. Вам не нужен опыт программирования, но вы должны уметь вводить команды в терминале. Мы начнём с основ и постепенно перейдём к продвинутым темам.
