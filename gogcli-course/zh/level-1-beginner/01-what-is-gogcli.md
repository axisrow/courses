# 什么是 gogcli？

## 概述

**gogcli**（命令名：`gog`）是一个命令行工具，让你可以直接在终端中使用 Google 服务——Gmail、日历、云端硬盘、通讯录、任务等，无需打开浏览器。

你可以把它想象成一个用键盘操作的 Google 遥控器。

## 为什么使用它？

- **速度快**：一条命令就能查邮件或创建日程，不用点来点去
- **自动化**：编写脚本自动处理邮件、更新表格、管理日历
- **多账户**：轻松切换个人和工作 Google 账户
- **与其他工具集成**：将 Google 数据传输给其他命令行工具

## 支持的 Google 服务

| 服务 | 命令 |
|------|------|
| Gmail | `gog mail` |
| 日历 | `gog calendar` |
| 云端硬盘 | `gog drive` |
| 通讯录 | `gog contacts` |
| 任务 | `gog tasks` |
| 表格 | `gog sheets` |
