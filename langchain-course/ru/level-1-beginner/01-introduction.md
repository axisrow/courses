# Введение в LangChain

## Что такое LangChain?

LangChain — это фреймворк с открытым кодом для создания приложений на основе больших языковых моделей (LLM). Он предоставляет инструменты для подключения LLM к внешним источникам данных, API и другим компонентам.

## Зачем использовать LangChain?

- **Простота**: Создавайте сложные LLM-приложения за несколько строк кода.
- **Модульность**: Комбинируйте компоненты — промпты, модели, память.
- **Экосистема**: Сотни интеграций с популярными сервисами.
- **Сообщество**: Активная разработка и растущее сообщество.

## Основные концепции

1. **Модели** — обёртки для LLM (OpenAI, Anthropic и др.)
2. **Промпты** — шаблоны для структурирования запросов
3. **Цепочки** — последовательности вызовов моделей и компонентов
4. **Память** — хранение истории диалога
5. **Агенты** — LLM, которые сами выбирают инструменты
6. **Извлечение** — получение данных из внешних источников

## Экосистема LangChain

| Компонент | Назначение |
|-----------|-----------|
| `langchain-core` | Базовые абстракции и LCEL |
| `langchain` | Цепочки, агенты, извлечение |
| `langchain-community` | Интеграции сторонних сервисов |
| `langserve` | Развёртывание цепочек как REST API |
| `langsmith` | Отладка и мониторинг |
