# Что такое Dify?

## Обзор

Dify — это open-source платформа для создания AI-приложений на основе больших языковых моделей (LLM). С её помощью можно создавать чат-боты, генераторы текста и AI-агентов — почти без кода.

## Почему Dify?

- **Визуальный конструктор** — перетаскивайте блоки для создания приложений
- **Много моделей** — подключайте OpenAI, Anthropic, локальные модели
- **Встроенный RAG** — загружайте свои документы как базу знаний
- **Open source** — разворачивайте у себя или используйте облако [dify.ai](https://dify.ai)

## Ключевые понятия

| Понятие | Описание |
|---------|----------|
| **Приложение** | AI-приложение, которое вы создаёте (чат-бот, агент, генератор) |
| **Провайдер моделей** | Сервис LLM (OpenAI, Ollama и др.) |
| **Промпт** | Инструкции, которые говорят AI как себя вести |
| **База знаний** | Ваши документы, по которым AI ищет ответы |
| **Воркфлоу** | Визуальная цепочка шагов для сложной логики |

## Что вы изучите

- **Уровень 1**: Установка Dify, первое приложение, публикация
- **Уровень 2**: Базы знаний, API, воркфлоу, плагины
- **Уровень 3**: Кастомные модели, корпоративное развёртывание, масштабирование
