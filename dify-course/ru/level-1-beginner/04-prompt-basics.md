# Основы промпт-инженерии

## Что такое промпт?

Промпт — это инструкция для AI. Хороший промпт = хороший ответ.

## Шаблон РОЛЬ-ЗАДАЧА-ФОРМАТ

```
РОЛЬ: Ты — [роль].
ЗАДАЧА: Помоги пользователю [сделать что-то].
ФОРМАТ: Отвечай в [формате].
```

### Пример

```
Ты — опытный диетолог.
Помоги пользователям составлять здоровые планы питания.
Отвечай структурированным дневным планом с маркированным списком.
```

## Переменные

Dify поддерживает переменные в промптах через `{{имя_переменной}}`:

```
Напиши {{тон}} письмо о {{тема}} для {{аудитория}}.
```

Пользователи заполняют переменные при использовании приложения.

## Советы

1. **Будьте конкретны** — «Резюмируй в 3 пунктах» лучше, чем «Резюмируй»
2. **Давайте примеры** — покажите AI, как выглядит хороший ответ
3. **Ставьте ограничения** — «НЕ давай медицинских советов»
4. **Используйте простой язык** — чёткие инструкции работают лучше

## Температура и параметры

| Параметр | Низкое значение | Высокое значение |
|----------|----------------|-----------------|
| **Температура** | Точный, предсказуемый | Креативный, разнообразный |
| **Top P** | Узкий выбор слов | Широкий выбор слов |
| **Макс. токенов** | Короткие ответы | Длинные ответы |

## Типичные ошибки

- ❌ Слишком размыто: «Помоги мне»
- ✅ Конкретно: «Напиши описание товара на 100 слов для синего кожаного кошелька»
- ❌ Слишком длинно: стена текста путает модель
- ✅ Структурно: нумерованные списки и разделы
