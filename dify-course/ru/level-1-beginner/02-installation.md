# Установка и настройка

## Вариант 1: Облако Dify

Самый быстрый способ. Зайдите на [cloud.dify.ai](https://cloud.dify.ai), создайте аккаунт — готово.

## Вариант 2: Docker (самостоятельная установка)

### Требования

- Docker и Docker Compose
- Минимум 4 ГБ оперативной памяти

### Шаги

```bash
# Клонируйте репозиторий
git clone https://github.com/langgenius/dify.git
cd dify/docker

# Скопируйте файл окружения
cp .env.example .env

# Запустите Dify
docker compose up -d
```

Dify будет доступен по адресу `http://localhost/install`.

### Первый запуск

1. Откройте `http://localhost/install` в браузере
2. Создайте учётную запись администратора
3. Вы попадёте на главную панель

## Добавление провайдера моделей

Перед созданием приложений подключите хотя бы одну LLM:

1. Перейдите в **Настройки → Провайдеры моделей**
2. Нажмите **Добавить** рядом с нужным провайдером (например, OpenAI)
3. Вставьте API-ключ
4. Нажмите **Сохранить**

## Проверка

Перейдите в **Студию** и нажмите **Создать приложение**. Если видите диалог создания — всё работает.
