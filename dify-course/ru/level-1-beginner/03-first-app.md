# Ваше первое AI-приложение

## Цель

Создать простого чат-бота, который отвечает на вопросы по выбранной теме.

## Пошаговая инструкция

### 1. Создайте приложение

1. Перейдите в **Студию → Создать приложение**
2. Выберите **Чат-бот**
3. Дайте ему имя (например, «Помощник путешественника»)
4. Нажмите **Создать**

### 2. Напишите системный промпт

В разделе **Промпт** напишите инструкции:

```
Ты — дружелюбный помощник путешественника. Помогай пользователям планировать поездки.
Давай практичные советы о направлениях, бюджете и сборах.
Отвечай коротко — максимум 2-3 абзаца.
```

### 3. Выберите модель

В селекторе модели (вверху справа) выберите LLM:
- `gpt-4o-mini` — быстрая и дешёвая
- `gpt-4o` — умнее, но дороже

### 4. Протестируйте

В панели **Предпросмотр** справа напишите сообщение:

> «Хочу поехать в Японию на 10 дней с ограниченным бюджетом. Есть советы?»

Проверьте, насколько ответ полезен и соответствует инструкциям.

### 5. Настройте

- Ответы слишком длинные → добавьте «Будь краток»
- Не хватает деталей → попросите «подробные пошаговые советы»
- **Температура**: ниже = точнее, выше = креативнее

## Типы приложений в Dify

| Тип | Для чего |
|-----|----------|
| **Чат-бот** | Многоходовые разговоры |
| **Генератор текста** | Один вход → один выход |
| **Агент** | AI с инструментами и действиями |
| **Воркфлоу** | Сложная логика из нескольких шагов |
