# Инструменты и плагины

## Что такое инструменты?

Инструменты позволяют AI-приложению взаимодействовать с внешним миром — искать в интернете, обращаться к базам данных, вызывать API.

## Встроенные инструменты

| Инструмент | Что делает |
|-----------|-----------|
| **Поиск в интернете** | Поиск в Google/Bing/DuckDuckGo |
| **Wikipedia** | Запросы к статьям Wikipedia |
| **Веб-скрапер** | Извлечение контента из URL |
| **Калькулятор** | Математические вычисления |
| **Текущее время** | Получение текущей даты/времени |

## Подключение инструментов

1. Перейдите в **Студия → ваше приложение**
2. Переключитесь в режим **Агент** (инструменты работают только с агентами)
3. В разделе **Инструменты** нажмите **Добавить**
4. Выберите из списка и настройте ключи доступа

## Кастомные инструменты (через API)

Вы можете добавить любой REST API как инструмент:

1. Перейдите в **Инструменты → Кастомные → Создать**
2. Вставьте **OpenAPI/Swagger** схему
3. Настройте аутентификацию (API-ключ, Bearer-токен)
4. Протестируйте инструмент
5. Сохраните — он теперь доступен во всех приложениях

## Режим агента

Когда вы добавляете инструменты в приложение-агент, AI сам решает, когда их использовать:

- Пользователь спрашивает «Какая погода в Москве?» → AI вызывает инструмент погоды
- Пользователь просит «Расскажи анекдот» → AI отвечает напрямую
