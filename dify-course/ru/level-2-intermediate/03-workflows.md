# Рабочие процессы (Workflows)

## Что такое воркфлоу?

Воркфлоу — это визуальная цепочка шагов. Вместо одного промпта вы соединяете несколько узлов для обработки сложной логики.

## Когда использовать

- Многоступенчатая обработка (классификация → маршрутизация → ответ)
- Преобразование данных до или после вызова LLM
- Условная логика (если/иначе)
- Комбинирование нескольких инструментов и API

## Создание воркфлоу

1. **Студия → Создать приложение → Воркфлоу**
2. Вы увидите визуальный холст с узлом **Старт**

## Типы узлов

| Узел | Назначение |
|------|-----------|
| **Старт** | Точка входа, определяет входные переменные |
| **LLM** | Вызов языковой модели |
| **Поиск по знаниям** | Поиск в базе знаний |
| **Код** | Запуск Python или JavaScript |
| **ЕСЛИ/ИНАЧЕ** | Условное ветвление |
| **Шаблон** | Форматирование текста через Jinja2 |
| **HTTP-запрос** | Вызов внешних API |
| **Присвоение переменных** | Установка или преобразование переменных |
| **Конец** | Узел вывода |

## Пример: умный классификатор

1. **Старт** → пользователь задаёт вопрос
2. **LLM** → классификация: «продажи», «поддержка» или «другое»
3. **ЕСЛИ/ИНАЧЕ** → ветвление по результату
4. **LLM** (для каждой ветки) → специализированный ответ
5. **Конец** → возврат ответа

## Советы

- **Тестируйте каждый узел** — нажмите на узел и используйте «Запуск»
- **Используйте переменные** — передавайте данные через `{{имя_узла.output}}`
- **Начинайте просто** — сначала малое, потом добавляйте сложность
