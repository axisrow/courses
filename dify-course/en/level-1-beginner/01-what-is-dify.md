# What is Dify?

## Overview

Dify is an open-source platform for building AI applications powered by large language models (LLMs). It lets you create chatbots, text generators, and AI agents — without writing much code.

## Why Dify?

- **Visual builder** — drag-and-drop interface for creating AI apps
- **Multiple LLM support** — connect OpenAI, Anthropic, local models, and more
- **Built-in RAG** — add your own documents as a knowledge base
- **Open source** — self-host or use the cloud version at [dify.ai](https://dify.ai)

## Key Concepts

| Concept | Description |
|---------|-------------|
| **App** | An AI application you build (chatbot, agent, text generator) |
| **Model Provider** | The LLM service (OpenAI, Ollama, etc.) |
| **Prompt** | Instructions that tell the AI how to behave |
| **Knowledge Base** | Your documents that the AI can search and reference |
| **Workflow** | A visual chain of steps for complex AI logic |
