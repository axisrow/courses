# Team Collaboration

## Workspaces

Dify organizes everything into workspaces. Each workspace has its own apps, knowledge bases, and members.

## Roles

| Role | Permissions |
|------|------------|
| **Owner** | Full access, manage billing and members |
| **Admin** | Manage apps, knowledge, members |
| **Editor** | Create and edit apps and knowledge |
| **Viewer** | View apps and logs only |

## Sharing Knowledge Bases

Knowledge bases can be shared across apps within the same workspace. This means:

- One set of company docs serves multiple chatbots
- Updates to docs automatically apply to all connected apps
