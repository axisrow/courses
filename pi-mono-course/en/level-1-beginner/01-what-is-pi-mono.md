# What is Pi Mono

## Introduction

Pi Mono is an open-source toolkit for building AI agents and managing large language models (LLMs). Created by Mario Zechner, it is hosted on GitHub: [badlogic/pi-mono](https://github.com/badlogic/pi-mono).

## What's Inside Pi Mono

Pi Mono is a monorepo containing several packages:

| Package | Description |
|---------|-------------|
| **pi-ai** | Unified API for multiple LLM providers (OpenAI, Anthropic, Google, etc.) |
| **pi-agent-core** | Agent runtime with tool calling support |
| **pi-coding-agent** | Interactive terminal-based coding assistant |
| **pi-mom** | Slack bot that delegates tasks to the coding agent |
| **pi-tui** | Terminal UI library |
| **pi-web-ui** | Web components for chat interfaces |
| **pi-pods** | CLI for deploying LLMs on GPU pods |
