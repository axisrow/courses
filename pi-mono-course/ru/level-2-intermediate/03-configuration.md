# Конфигурация

## Введение

Pi предлагает гибкую систему настройки через файлы конфигурации, переменные окружения и команды.

## Переменные окружения

Основные переменные для API-ключей:

```bash
export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_API_KEY=sk-...
export GOOGLE_API_KEY=...
export MISTRAL_API_KEY=...
export GROQ_API_KEY=...
```

## Настройки Pi

В интерактивном режиме используйте команду `/settings`:

```
/settings
```

Здесь можно настроить:
- Модель по умолчанию
- Уровень «размышления» (thinking)
- Тему оформления
- Поведение автозаполнения

## Выбор модели

```bash
pi --provider anthropic --model claude-sonnet-4-20250514
```

Или в интерактивном режиме:

```
/model anthropic claude-sonnet-4-20250514
```

## Контекстные файлы проекта

| Файл | Назначение |
|------|-----------|
| `AGENTS.md` | Инструкции для агента о проекте |
| `TOOLS.md` | Заметки об инструментах и окружении |
| `.pi/skills/` | Навыки — контекстно-зависимые инструкции |

## Темы

Pi поддерживает пользовательские темы для терминального интерфейса. Темы определяют цвета и стили компонентов.

## Итоги

- API-ключи задаются через переменные окружения
- Настройки доступны через /settings и CLI-флаги
- Контекстные файлы настраивают поведение агента
