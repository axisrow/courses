# Контрибьюция

## Введение

Pi Mono — проект с открытым исходным кодом под лицензией MIT. В этом уроке мы узнаем, как вносить вклад в проект.

## Начало работы

### Шаг 1: Форкните репозиторий

```bash
git clone https://github.com/badlogic/pi-mono.git
cd pi-mono
```

### Шаг 2: Установите зависимости

```bash
npm install
```

### Шаг 3: Соберите проект

```bash
npm run build
```

### Шаг 4: Запустите тесты

```bash
./test.sh
```

## Правила проекта

Изучите файлы:
- **CONTRIBUTING.md** — гайд по контрибьюции
- **AGENTS.md** — правила проекта (для людей и агентов)

## Рабочий процесс

1. Создайте feature-ветку
2. Внесите изменения
3. Запустите `npm run check`
4. Запустите тесты
5. Создайте Pull Request

## Запуск Pi из исходников

```bash
./pi-test.sh    # Запуск pi из исходного кода
```

## Структура тестов

Тесты, зависящие от LLM API-ключей, пропускаются, если ключи не заданы. Это позволяет запускать базовые тесты без настройки провайдеров.
