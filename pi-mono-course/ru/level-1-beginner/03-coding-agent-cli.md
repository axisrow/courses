# Coding Agent CLI

## Введение

Pi Coding Agent — это интерактивный AI-помощник, который работает прямо в терминале. Он может читать и редактировать файлы, выполнять команды и помогать вам писать код.

## Основные возможности

- **Чтение файлов** — агент видит содержимое файлов в проекте
- **Редактирование** — может создавать и изменять файлы
- **Bash-команды** — выполняет команды в терминале
- **Работа с контекстом** — понимает структуру проекта

## Запуск

```bash
pi
```

Это запустит интерактивный режим. Вы увидите редактор для ввода сообщений.

## Режимы работы

Pi работает в четырёх режимах:

1. **Интерактивный** — диалог в терминале (по умолчанию)
2. **Print/JSON** — вывод результата в stdout
3. **RPC** — для интеграции с другими программами
4. **SDK** — встраивание в ваши приложения

## Основные инструменты агента

По умолчанию у агента есть 4 инструмента:

- `read` — чтение файлов
- `write` — запись файлов
- `edit` — редактирование файлов
- `bash` — выполнение команд

## Пример использования

```
> Создай файл hello.py с программой Hello World

Агент создаст файл и покажет результат.
```

## Итоги

- Pi Coding Agent — терминальный AI-помощник
- Работает в 4 режимах
- Имеет 4 базовых инструмента для работы с файлами и командами
