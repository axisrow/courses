# Что такое Pi Mono

## Введение

Pi Mono — это набор инструментов (toolkit) с открытым исходным кодом для создания AI-агентов и управления большими языковыми моделями (LLM). Проект создан Марио Цехнером и размещён на GitHub: [badlogic/pi-mono](https://github.com/badlogic/pi-mono).

## Что входит в Pi Mono

Pi Mono — это монорепозиторий, содержащий несколько пакетов:

| Пакет | Описание |
|-------|----------|
| **pi-ai** | Единый API для работы с разными LLM-провайдерами (OpenAI, Anthropic, Google и др.) |
| **pi-agent-core** | Движок агентов с поддержкой вызова инструментов |
| **pi-coding-agent** | Интерактивный AI-помощник для программирования в терминале |
| **pi-mom** | Slack-бот, делегирующий задачи coding agent |
| **pi-tui** | Библиотека терминального интерфейса |
| **pi-web-ui** | Веб-компоненты для чат-интерфейсов |
| **pi-pods** | CLI для развёртывания LLM на GPU-серверах |

## Для кого этот курс

Этот курс подходит для всех, кто хочет научиться использовать AI-агентов — от простого запуска coding agent до создания собственных плагинов и развёртывания моделей на GPU.

## Итоги

- Pi Mono — открытый набор инструментов для AI-агентов
- Содержит 7 пакетов для разных задач
- Поддерживает множество LLM-провайдеров
