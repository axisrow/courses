# Первый запуск

## Введение

В этом уроке мы впервые запустим Pi и выполним несколько простых задач.

## Подготовка

Убедитесь, что Pi установлен и API-ключ настроен:

```bash
pi --help
```

## Пошаговые инструкции

### Шаг 1: Запустите Pi

```bash
mkdir my-project && cd my-project
pi
```

### Шаг 2: Поздоровайтесь

Просто напишите:

```
Привет! Что ты умеешь?
```

Агент расскажет о своих возможностях.

### Шаг 3: Попросите создать файл

```
Создай файл README.md с описанием проекта "Мой первый проект"
```

### Шаг 4: Попросите выполнить команду

```
Покажи список файлов в текущей папке
```

### Шаг 5: Выход

Нажмите `Ctrl+C` или введите `/exit`.

## Полезные команды

| Команда | Действие |
|---------|----------|
| `/help` | Список команд |
| `/login` | Авторизация у провайдера |
| `/model` | Выбор модели |
| `/exit` | Выход |

## Итоги

- Запустили Pi в интерактивном режиме
- Попросили агента создать файл и выполнить команду
- Познакомились с основными командами
