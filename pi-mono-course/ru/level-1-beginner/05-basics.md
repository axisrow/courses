# Основы работы

## Введение

В этом уроке мы изучим основные концепции работы с Pi: сессии, контекстные файлы и навыки.

## Сессии

Pi сохраняет историю диалога в сессиях. Вы можете:

- Продолжить предыдущий разговор
- Создать ветку (branch) от любого момента
- Сжать (compact) длинную историю для экономии токенов

```bash
pi --session my-session    # Продолжить или создать сессию
pi --sessions              # Список сессий
```

## Контекстные файлы

Файлы `AGENTS.md` и `TOOLS.md` в корне проекта автоматически подгружаются в контекст агента. Используйте их, чтобы дать агенту знания о вашем проекте:

```markdown
# AGENTS.md
Этот проект — веб-приложение на React.
Используй TypeScript и следуй нашему стилю кода.
```

## Навыки (Skills)

Навыки — это файлы с инструкциями, которые Pi подгружает при необходимости. Создайте папку `.pi/skills/`:

```bash
mkdir -p .pi/skills
echo "При работе с базой данных используй Prisma ORM" > .pi/skills/database.md
```

## Расширения (Extensions)

Расширения — TypeScript-модули, добавляющие новые инструменты агенту. Их можно установить через npm или написать самостоятельно.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+C` | Отмена/выход |
| `Shift+Ctrl+D` | Отладка |
| `Enter` | Отправить сообщение |

## Итоги

- Сессии хранят историю диалога
- AGENTS.md и TOOLS.md задают контекст проекта
- Навыки и расширения позволяют настроить агента
