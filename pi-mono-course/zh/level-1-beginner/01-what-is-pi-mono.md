# 什么是 Pi Mono

## 简介

Pi Mono 是一个开源工具包，用于构建 AI 代理和管理大型语言模型（LLM）。由 Mario Zechner 创建，托管在 GitHub：[badlogic/pi-mono](https://github.com/badlogic/pi-mono)。

## Pi Mono 包含什么

Pi Mono 是一个包含多个软件包的 monorepo：

| 软件包 | 描述 |
|--------|------|
| **pi-ai** | 统一的多提供商 LLM API（OpenAI、Anthropic、Google 等） |
| **pi-agent-core** | 支持工具调用的代理运行时 |
| **pi-coding-agent** | 交互式终端编程助手 |
| **pi-mom** | 将任务委托给编程代理的 Slack 机器人 |
| **pi-tui** | 终端 UI 库 |
| **pi-web-ui** | 聊天界面的 Web 组件 |
| **pi-pods** | 在 GPU pod 上部署 LLM 的 CLI |

## 本课程适合谁

本课程适合所有想学习使用 AI 代理的人——从运行简单的编程代理到创建插件和在 GPU 上部署模型。

## 总结

- Pi Mono 是开源 AI 代理工具包
- 包含 7 个不同用途的软件包
- 支持多种 LLM 提供商
