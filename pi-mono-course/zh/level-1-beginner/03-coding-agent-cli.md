# 编程代理 CLI

## 简介

Pi 编程代理是一个交互式 AI 助手，直接在终端中工作。它可以读取和编辑文件、运行命令并帮助你编写代码。

## 主要功能

- **读取文件** — 代理可以查看项目中的文件内容
- **编辑文件** — 可以创建和修改文件
- **Bash 命令** — 执行终端命令
- **上下文感知** — 理解项目结构

## 启动

```bash
pi
```

这将启动交互模式。你会看到一个用于输入消息的编辑器。

## 工作模式

Pi 有四种工作模式：

1. **交互式** — 终端对话（默认）
2. **Print/JSON** — 输出到 stdout
3. **RPC** — 用于与其他程序集成
4. **SDK** — 嵌入到你自己的应用中

## 默认工具

代理自带 4 个内置工具：

- `read` — 读取文件
- `write` — 写入文件
- `edit` — 编辑文件
- `bash` — 执行命令

## 使用示例

```
> 创建一个包含 Hello World 程序的 hello.py 文件

代理将创建文件并显示结果。
```

## 总结

- Pi 编程代理是终端 AI 助手
- 有 4 种工作模式
- 有 4 个基本的文件和命令工具
