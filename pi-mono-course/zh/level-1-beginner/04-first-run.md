# 首次运行

## 简介

在本课中，我们将首次运行 Pi 并完成几个简单任务。

## 准备

确保 Pi 已安装并配置了 API 密钥：

```bash
pi --help
```

## 分步说明

### 第1步：启动 Pi

```bash
mkdir my-project && cd my-project
pi
```

### 第2步：打个招呼

输入：

```
你好！你能做什么？
```

代理会描述它的功能。

### 第3步：让它创建文件

```
创建一个 README.md，描述"我的第一个项目"
```

### 第4步：让它执行命令

```
显示当前目录中的文件
```

### 第5步：退出

按 `Ctrl+C` 或输入 `/exit`。

## 有用的命令

| 命令 | 操作 |
|------|------|
| `/help` | 命令列表 |
| `/login` | 认证提供商 |
| `/model` | 选择模型 |
| `/exit` | 退出 |

## 总结

- 以交互模式启动了 Pi
- 让代理创建文件和执行命令
- 学习了基本命令
