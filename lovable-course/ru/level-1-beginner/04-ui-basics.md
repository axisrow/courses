# Основы интерфейса

Научитесь управлять внешним видом приложения.

## Макет

Lovable создаёт адаптивные макеты. Можно попросить:

- **Одна колонка** — простой вертикальный поток.
- **Сайдбар + контент** — стиль дашборда.
- **Сетка** — карточки в строках и колонках.

Пример:

> Сделай сайдбар слева и основной контент справа.

## Цвета

> Используй сине-белую цветовую схему. Шапка — тёмно-синяя.

## Шрифты

> Увеличь заголовки. Сделай основной текст удобнее для чтения.

## Кнопки и формы

> Добавь контактную форму: имя, email, сообщение. Кнопка «Отправить».

## Изображения и иконки

> Добавь изображение-героя вверху. Иконки рядом с пунктами меню.

Lovable использует библиотеку иконок Lucide.

## Адаптивный дизайн

Приложения адаптивны по умолчанию. Можно уточнить:

> На мобильных навигация должна сворачиваться в меню-гамбургер.

## Популярные UI-паттерны

| Паттерн | Пример промпта |
|---------|---------------|
| Навигация | «Добавь верхнее меню: Главная, О нас, Контакты» |
| Сетка карточек | «Покажи элементы карточками в 3 колонки» |
| Модальное окно | «Открывай попап при нажатии Редактировать» |
| Подвал | «Добавь футер с копирайтом и соцсетями» |

## Итог

Через промпты вы управляете макетом, цветами, шрифтами и компонентами. Дальше — публикация приложения.
