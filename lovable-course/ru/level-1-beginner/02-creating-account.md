# Создание аккаунта

Давайте настроим ваш аккаунт в Lovable.

## Шаг 1: Откройте Lovable

Перейдите на [lovable.dev](https://lovable.dev) в браузере.

## Шаг 2: Регистрация

Нажмите **Sign Up**. Можно зарегистрироваться через:

- Аккаунт Google
- Аккаунт GitHub
- Email и пароль

## Шаг 3: Выберите план

Есть бесплатный план для старта. Обновить можно позже.

| План | Что входит |
|------|-----------|
| Free | Ограниченные промпты, 1 проект |
| Starter | Больше промптов, несколько проектов |
| Pro | Безлимитные промпты, приоритетная поддержка |

## Шаг 4: Изучите дашборд

После входа вы увидите дашборд:

- **Новый проект** — начать создание.
- **Ваши проекты** — список созданного.
- **Настройки** — управление аккаунтом.

## Редактор

Когда вы открываете проект:

- **Панель чата** (слева) — общение с AI.
- **Предпросмотр** (справа) — живое отображение приложения.
- **Код** — переключение на просмотр кода.

## Итог

У вас есть аккаунт, и вы знаете интерфейс. Дальше — создаём первое приложение.
