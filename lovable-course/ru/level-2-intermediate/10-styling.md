# Стили и темы

Сделайте приложение красивым и профессиональным.

## Tailwind CSS

Lovable использует Tailwind CSS. Знать Tailwind не обязательно:

> Добавь больше отступов у карточек. Заголовок — жирный и крупнее.

## Тёмная тема

> Добавь переключатель тёмной темы. Запоминай выбор пользователя.

Lovable создаст переключатель с сохранением в localStorage.

## Своя палитра

> Используй палитру: основной #6366f1, вторичный #ec4899, фон #f8fafc.

## Анимации

> Добавь анимацию появления карточек. Кнопки немного увеличиваются при наведении.

## Шрифты

> Основной шрифт — «Inter», для кода — «JetBrains Mono».

## Единый стиль

> Все кнопки — скруглённые, одинаковые отступы. Карточки — с лёгкой тенью.

## Адаптивность

| Брейкпоинт | Экран |
|-----------|-------|
| sm | Мобильный (640px+) |
| md | Планшет (768px+) |
| lg | Десктоп (1024px+) |
| xl | Большой (1280px+) |

> На мобильных сайдбар располагается под контентом. На десктопе — рядом.

## Итог

Tailwind CSS управляет стилями. Через промпты вы контролируете цвета, темы, анимации, шрифты и адаптивность.
