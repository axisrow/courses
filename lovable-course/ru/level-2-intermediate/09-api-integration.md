# Интеграция API

Подключайте внешние сервисы к приложению.

## Что такое API?

API позволяет приложению общаться с другими сервисами: погода, платежи, AI-модели и т.д.

## Получение данных

> Получи последние новости из публичного API и покажи карточками.

Lovable создаст логику запросов, индикаторы загрузки и обработку ошибок.

## Edge Functions

Для серверной логики используйте Supabase Edge Functions:

> Создай edge function, которая вызывает OpenAI API и возвращает краткое содержание текста.

Edge Functions работают на сервере — API-ключи в безопасности.

## Переменные окружения

> Храни API-ключ в переменной окружения. Не прописывай его в коде.

Lovable использует секреты Supabase.

## Популярные интеграции

| Сервис | Назначение |
|--------|-----------|
| Stripe | Платежи и подписки |
| OpenAI | AI-функции |
| SendGrid | Email-уведомления |
| Cloudinary | Загрузка изображений |
| Google Maps | Карты и геолокация |

## Вебхуки

> При создании нового заказа отправь вебхук в Slack.

## Обработка ошибок

> Покажи дружелюбное сообщение об ошибке, если API не отвечает. Добавь кнопку «Повторить».

## Итог

API расширяют возможности приложения. Используйте Edge Functions для безопасных серверных вызовов.
