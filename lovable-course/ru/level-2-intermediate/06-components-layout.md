# Компоненты и макет

Создавайте сложные интерфейсы из переиспользуемых компонентов.

## Что такое компоненты?

Компоненты — это строительные блоки: кнопка, карточка, навигация. Lovable автоматически генерирует React-компоненты.

## Создание через промпты

> Создай переиспользуемый компонент карточки с изображением, заголовком, описанием и кнопкой «Подробнее».

AI создаст отдельный файл компонента.

## Многостраничные приложения

> Добавь страницу «О нас». Сделай навигацию между Главной и О нас.

Lovable настроит маршрутизацию (React Router).

## Паттерны макетов

### Дашборд

> Создай дашборд с сайдбаром, шапкой с аватаром пользователя и основной областью контента.

### Лендинг

> Создай лендинг: секция-герой, сетка фич, отзывы, футер.

## Компоненты shadcn/ui

Lovable использует [shadcn/ui](https://ui.shadcn.com/):

- Dialog, Sheet, Drawer
- Table, DataTable
- Tabs, Accordion
- Toast-уведомления
- Выпадающие меню

Пример:

> Добавь таблицу данных с сортировкой и пагинацией для списка пользователей.

## Организация кода

```
src/
  components/
    ui/          ← примитивы shadcn/ui
    Header.tsx
    Sidebar.tsx
    TaskCard.tsx
  pages/
    Home.tsx
    About.tsx
```

## Итог

Компоненты делают приложение организованным. Lovable создаёт компоненты, маршрутизацию и использует shadcn/ui.
