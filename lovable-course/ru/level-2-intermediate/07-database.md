# База данных Supabase

Храните и получайте данные для приложения.

## Что такое Supabase?

Supabase — это open-source бэкенд: база данных PostgreSQL, аутентификация и API. Lovable интегрируется с Supabase нативно.

## Подключение

1. В проекте нажмите **Supabase** в настройках.
2. Lovable может создать проект Supabase автоматически.
3. Или подключите существующий через URL и anon key.

## Создание таблиц

> Создай таблицу «posts» с колонками: id, title, content, created_at. Покажи все посты на главной.

Lovable:
- Создаст таблицу в Supabase.
- Сгенерирует код для получения и отображения данных.
- Добавит TypeScript-типы.

## CRUD-операции

> Пользователи могут создавать посты через форму. Добавь кнопки редактирования и удаления.

Полный CRUD готов.

## Реальное время

> Список постов должен обновляться в реальном времени при добавлении нового поста.

## Безопасность (RLS)

> Пользователи могут редактировать только свои посты.

AI настроит политики Row Level Security.

## Типичные паттерны

| Паттерн | Пример |
|---------|--------|
| Профили | Имя, аватар, био для каждого пользователя |
| Контент | Посты, статьи, комментарии |
| E-commerce | Товары, заказы, корзина |
| Аналитика | События, просмотры, метрики |

## Итог

Supabase даёт мощную базу данных. Через промпты вы создаёте таблицы, CRUD, реальное время и безопасность.
