# Кастомный код

Выходите за рамки промптов — редактируйте код напрямую.

## Просмотр кода

Переключите на вид кода в редакторе. Lovable создаёт чистый React + TypeScript.

## Когда редактировать вручную

- Точная настройка логики.
- Сложные бизнес-правила.
- Интеграция редких библиотек.

## Редактирование файлов

Можно редактировать любой файл. Изменения сразу отражаются в предпросмотре.

## npm-пакеты

> Добавь библиотеку «date-fns» для форматирования дат.

Lovable добавит в `package.json` и импортирует.

## Кастомные хуки

> Создай хук useLocalStorage для сохранения состояния в localStorage.

## TypeScript-типы

> Добавь интерфейсы TypeScript для типов Product и Order.

## Структура файлов

```
src/
  components/     ← UI-компоненты
  hooks/          ← Кастомные хуки
  lib/            ← Утилиты
  pages/          ← Страницы
  integrations/   ← Клиент Supabase
  types/          ← TypeScript-типы
```

## Промпты + код

Лучший подход:
1. Промпты для крупных изменений.
2. Ручное редактирование для мелких правок.
3. AI учитывает ваши ручные изменения.

## Итог

У вас полный доступ к коду. Редактируйте, добавляйте пакеты, создавайте кастомную логику. Lovable уважает ваши ручные правки.
