# 发布应用

应用已经准备好了，让全世界看到它。

## 一键发布

Lovable 有内置托管。点击右上角的 **Publish** 按钮。

你的应用会获得一个 URL：`https://your-app.lovable.app`

## 自定义域名

想用自己的域名？

1. 进入 **Settings → Domain**。
2. 添加你的域名（如 `myapp.com`）。
3. 按照说明更新 DNS 记录。

## 分享

把链接发给任何人。他们可以直接在浏览器中使用——不需要安装。

## 导出代码

想把代码带走？

1. 点击 **Export to GitHub**。
2. Lovable 会创建一个包含所有代码的仓库。
3. 你可以部署到任何地方：Vercel、Netlify、自己的服务器。

## 导出内容

- React + TypeScript 源代码
- Tailwind CSS 样式
- Supabase 配置（如果使用了数据库）
- 包含所有依赖的 Package.json

## 发布后更新

在 Lovable 中修改，再次点击 **Publish**。更新立即生效。

## 总结

一键发布、自定义域名或导出到 GitHub。恭喜——你已完成初级课程！🎉
