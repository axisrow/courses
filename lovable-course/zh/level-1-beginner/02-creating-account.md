# 创建账户

让我们注册 Lovable。

## 第一步：打开 Lovable

在浏览器中访问 [lovable.dev](https://lovable.dev)。

## 第二步：注册

点击 **Sign Up**，可以通过以下方式注册：

- Google 账户
- GitHub 账户
- 邮箱和密码

## 第三步：选择计划

Lovable 提供免费计划。之后可以升级。

| 计划 | 内容 |
|------|------|
| Free | 有限提示词，1 个项目 |
| Starter | 更多提示词，多个项目 |
| Pro | 无限提示词，优先支持 |

## 第四步：探索仪表盘

登录后你会看到仪表盘：

- **新建项目** — 开始创建。
- **你的项目** — 查看已创建的内容。
- **设置** — 管理账户。

## 编辑器

打开项目后，你会看到：

- **聊天面板**（左侧）— 与 AI 对话。
- **预览面板**（右侧）— 应用的实时渲染。
- **代码视图** — 切换查看生成的代码。

## 总结

你已经有了账户，也了解了界面。下一课我们将创建第一个应用。
