# Редактирование в режиме дизайна

## Что такое режим дизайна?

Режим дизайна (Design Mode) — это визуальный редактор в v0. Вместо написания промптов вы кликаете на элементы и меняете их напрямую — как в простом графическом редакторе.

## Как войти в режим дизайна

1. Сгенерируйте интерфейс промптом или откройте проект
2. Найдите кнопку «Design Mode» или «Edit»
3. Нажмите — предпросмотр станет редактируемым

## Что можно делать

### Менять текст
Кликните на текст и отредактируйте заголовок, описание или надпись на кнопке.

### Менять цвета
Выберите элемент и измените фон, цвет текста или границы.

### Изменять размеры
Тяните за края элементов, чтобы увеличить или уменьшить их.

### Перемещать элементы
Перетаскивайте элементы для изменения расположения.

### Менять шрифты
Выберите текст и настройте шрифт, размер, жирность и отступы.

## Когда использовать режим дизайна, а когда промпты

| Задача | Режим дизайна | Промпты |
|--------|--------------|---------|
| Исправить опечатку | ✅ | |
| Поменять один цвет | ✅ | |
| Добавить новый раздел | | ✅ |
| Перестроить всю страницу | | ✅ |
| Подправить отступы | ✅ | |

## Советы

- Режим дизайна идеален для мелких правок после генерации
- Для крупных изменений лучше использовать чат
- Изменения сохраняются автоматически

## Итог

Режим дизайна позволяет визуально редактировать интерфейс. Используйте его для тонкой настройки. Дальше — шаблоны.
