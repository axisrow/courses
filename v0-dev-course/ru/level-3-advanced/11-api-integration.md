# Интеграция с API

## Что такое API?

API (Application Programming Interface) — это способ, которым ваше приложение общается с другими сервисами. Например, получает данные о погоде, принимает платежи или отправляет письма.

Представьте API как официанта: вы говорите, что хотите (запрос), он приносит из кухни (ответ).

## API в v0

v0 умеет автоматически интегрироваться с популярными сервисами:

- Для многих интеграций не нужна отдельная регистрация
- AI понимает стандартные паттерны API
- Описывайте нужные данные обычным языком

## Как добавить API

Просто опишите в промпте:

- "Create a weather widget that shows the current weather for Moscow"
- "Build a contact form that sends emails via Resend"
- "Make a page that displays the latest news from an RSS feed"

v0 сгенерирует код с нужными API-вызовами.

## Популярные интеграции

| Сервис | Назначение |
|--------|-----------|
| Stripe | Приём платежей |
| Resend | Отправка email |
| OpenAI | Генерация текста AI |
| Supabase | База данных и авторизация |
| Cloudinary | Управление изображениями |

## API-ключи

Некоторые API требуют ключ (как пароль):

1. Зарегистрируйтесь в сервисе API
2. Получите ключ в панели управления
3. Добавьте его в переменные окружения в Vercel
4. Никогда не вставляйте ключи прямо в код

## Итог

API позволяют приложению связываться с внешними сервисами. v0 упрощает интеграцию — просто опишите, что нужно. Дальше — базы данных.
