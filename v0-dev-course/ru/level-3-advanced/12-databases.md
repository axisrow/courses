# Работа с базами данных

## Что такое база данных?

База данных хранит информацию вашего приложения: аккаунты пользователей, статьи, товары, заказы.

Представьте умную таблицу, которую ваше приложение может читать и заполнять автоматически.

## Базы данных в v0

v0 умеет работать с базами данных:

- Создавать схему данных (структуру)
- Генерировать код для чтения и записи
- Настраивать подключения автоматически

## Как добавить базу данных

1. Опишите в промпте, какие данные нужны:
   - "Create a to-do app that saves tasks to a database"
   - "Build a blog with posts stored in a database"
2. v0 предложит базу данных (часто Supabase или Vercel Postgres)
3. Следуйте инструкциям для создания аккаунта

## Основные операции

| Операция | Что делает | Пример |
|----------|-----------|--------|
| Create | Добавляет данные | Новая статья в блог |
| Read | Читает данные | Список всех товаров |
| Update | Обновляет данные | Редактирование профиля |
| Delete | Удаляет данные | Удаление комментария |

Это называют **CRUD**-операции.

## Провайдеры баз данных

- **Vercel Postgres** — встроен в экосистему Vercel
- **Supabase** — популярный, есть бесплатный план
- **PlanetScale** — на основе MySQL
- **Neon** — серверлесс Postgres

## Советы

- Начинайте с простых структур данных
- Продумайте, какие данные нужны, до начала разработки
- Используйте AI для проектирования схемы

## Итог

Базы данных хранят данные приложения. v0 настроит подключение и сгенерирует код. Дальше — агентные возможности.
