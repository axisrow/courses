# Публикация на Vercel

## Что значит «опубликовать»?

Публикация (деплой) — это размещение проекта в интернете по реальному адресу, который может посетить любой. До публикации ваш проект видите только вы.

## Публикация в один клик

v0 создан Vercel, поэтому публикация максимально простая:

1. Откройте проект в v0
2. Нажмите «Deploy» или «Publish»
3. v0 размещает проект на серверах Vercel
4. Вы получаете адрес (например, `your-project.vercel.app`)

Всё. Ваш сайт в интернете.

## Свой домен

Хотите свой адрес (например, `www.moybiznes.ru`)?

1. Купите домен у регистратора
2. Зайдите в панель управления Vercel
3. Добавьте домен к проекту
4. Обновите DNS-настройки домена по инструкции
5. Сайт доступен по вашему адресу

## Что происходит после публикации?

- Сайт доступен по всему миру
- Работает быстро — Vercel использует глобальную CDN
- Безопасно — HTTPS включён автоматически
- Каждое обновление создаёт новую версию

## Бесплатно vs. платно

| Возможность | Бесплатно (Hobby) | Платно (Pro) |
|-------------|-------------------|-------------|
| Публикации | Безлимит | Безлимит |
| Трафик | 100 ГБ/мес | 1 ТБ/мес |
| Свой домен | Да | Да |
| Командная работа | Ограничено | Полная |
