# 部署到 Vercel

## "部署"是什么意思？

部署就是把你的项目放到互联网上，给它一个真实的网址，任何人都可以访问。部署前，只有你能看到你的作品。

## 一键部署

v0 由 Vercel 开发，所以部署非常顺畅：

1. 在 v0 中打开你的项目
2. 点击"Deploy"或"Publish"按钮
3. v0 把项目部署到 Vercel 的服务器
4. 你得到一个实时网址（例如 `your-project.vercel.app`）

就这样，你的网站上线了。

## 自定义域名

想要自己的域名（例如 `www.mybusiness.com`）？

1. 从域名注册商购买域名
2. 进入 Vercel 控制面板
3. 把域名添加到项目
4. 按指示更新域名的 DNS 设置
5. 你的网站现在可以通过自定义地址访问了

## 部署后会发生什么？

- 网站全球可访问
- 速度快——Vercel 使用全球 CDN
- 安全——自动启用 HTTPS
- 每次更新都创建新版本
