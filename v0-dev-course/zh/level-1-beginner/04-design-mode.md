# 使用设计模式编辑

## 什么是设计模式？

设计模式（Design Mode）是 v0 内置的可视化编辑器。你不用写提示词或代码，直接点击元素进行修改——就像使用简单的设计工具。

## 如何进入设计模式

1. 用提示词生成界面（或打开已有项目）
2. 找到工具栏上的"Design Mode"或"Edit"按钮
3. 点击它——预览变成可编辑状态

## 你能做什么

### 修改文字
点击任何文字来编辑标题、描述或按钮文字。

### 调整颜色
选择元素，用颜色选择器更改背景色、文字色或边框色。

### 调整大小
拖动元素边缘来放大或缩小。

### 移动元素
拖放元素来重新排列布局。

### 更改字体
选择文字，调整字体、大小、粗细和间距。

## 何时用设计模式，何时用提示词

|------|---------|--------|
| 改错别字 | ✅ | |
| 改一个颜色 | ✅ | |
| 添加新区块 | | ✅ |
| 重新布局整个页面 | | ✅ |
| 微调间距 | ✅ | |

## 技巧

- 设计模式适合 AI 生成基础布局后的小调整
- 大的结构变化用聊天提示词
- 更改自动保存

## 总结

设计模式让你无需写提示词或代码就能可视化编辑界面。用它来做微调。下一课——模板。
