# Работа с инструментами

## Что такое инструменты?

Инструменты (Tools) дают агентам возможность взаимодействовать с внешним миром — искать в интернете, читать файлы, вызывать API.

Без инструментов агенты используют только свои обучающие данные. С инструментами они становятся по-настоящему полезными.

## Встроенные инструменты

```bash
pip install 'crewai[tools]'
```

| Инструмент | Что делает |
|------------|-----------|
| `SerperDevTool` | Поиск в Google |
| `ScrapeWebsiteTool` | Извлечение контента с сайтов |
| `FileReadTool` | Чтение локальных файлов |
| `DirectoryReadTool` | Список содержимого папки |
| `PDFSearchTool` | Поиск в PDF-файлах |
| `CSVSearchTool` | Запросы к CSV-данным |

## Использование

```python
from crewai import Agent
from crewai_tools import SerperDevTool

search = SerperDevTool()

researcher = Agent(
    role="Веб-исследователь",
    goal="Находить актуальную информацию в интернете",
    backstory="Вы проверяете факты через веб-поиск.",
    tools=[search]
)
```

Агент сам решит, когда вызвать инструмент.

## Инструменты на уровне задачи

```python
task = Task(
    description="Найдите новости о регулировании AI.",
    expected_output="Резюме 3 свежих статей.",
    agent=researcher,
    tools=[search]  # доступен только для этой задачи
)
```

## Переменные окружения

Большинство инструментов требуют API-ключи:

```bash
export SERPER_API_KEY="your-serper-key"
export OPENAI_API_KEY="your-openai-key"
```

## Советы

1. Давайте агентам только нужные инструменты — меньше инструментов = лучший фокус
2. Тестируйте инструменты отдельно перед добавлением к агентам
3. Проверяйте требования к API-ключам
4. Используйте `verbose=True` чтобы видеть вызовы инструментов

## Итого

Инструменты расширяют возможности агентов. Используйте встроенные для типовых задач. Назначайте на уровне агента или задачи.
