# Обработка ошибок и отладка

## Типичные проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| Пустой результат | Размытое описание | Конкретизируйте description и expected_output |
| Ошибки API | Нет ключа | Проверьте переменные окружения |
| Бесконечный цикл | Агент не может завершить | Установите `max_iter` |
| Неверный инструмент | Слишком много инструментов | Сократите список tools |

## Режим verbose

```python
crew = Crew(
    agents=[agent1, agent2],
    tasks=[task1, task2],
    verbose=True
)
```

Показывает: какой агент работает, какие инструменты вызываются, что агент «думает», итоговый результат.

## Ограничение итераций

```python
agent = Agent(
    role="Аналитик",
    goal="Анализировать данные",
    backstory="Вы — аналитик данных.",
    max_iter=10
)
```

## Чек-лист отладки

1. ✅ Включите `verbose=True`
2. ✅ Проверьте API-ключи
3. ✅ Убедитесь, что роли и цели осмысленны
4. ✅ Проверьте конкретность описаний задач
5. ✅ Убедитесь, что expected_output чёткий
6. ✅ Проверьте настройку инструментов
7. ✅ Установите `max_iter`
8. ✅ Начните с простых задач

## Логирование

```python
import logging
logging.basicConfig(level=logging.DEBUG)
```

## Итого

Отлаживайте через `verbose=True`, ставьте `max_iter`, пишите конкретные описания задач. Большинство проблем — от размытых инструкций или отсутствия API-ключей.
