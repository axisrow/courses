# Что такое CrewAI?

## Введение

CrewAI — это open-source фреймворк на Python для создания **мультиагентных AI-систем**. Вместо одного AI, который делает всё, вы создаёте команду (crew) специализированных агентов, которые работают вместе.

Представьте компанию: есть исследователь, автор, редактор — каждый со своей ролью. CrewAI позволяет построить такую команду в коде.

## Зачем нужны мультиагенты?

Один промпт может сделать ограниченный объём работы. Когда вы распределяете задачи между агентами:

- Каждый агент фокусируется на своей специализации
- Агенты могут использовать разные инструменты
- Сложные задачи разбиваются на управляемые части
- Результаты получаются надёжнее и детальнее

## Ключевые концепции

| Концепция | Что это значит |
|-----------|---------------|
| **Agent (Агент)** | AI-работник с ролью, целью и предысторией |
| **Task (Задача)** | Конкретная работа, назначенная агенту |
| **Crew (Команда)** | Группа агентов, работающих вместе |
| **Tool (Инструмент)** | Внешняя возможность агента (поиск, чтение файлов, API) |
| **Process (Процесс)** | Как агенты координируются — последовательно или иерархически |

## Простой пример

```python
from crewai import Agent, Task, Crew

researcher = Agent(
    role="Исследователь",
    goal="Найти точную информацию о трендах AI",
    backstory="Вы — опытный технический исследователь."
)

task = Task(
    description="Исследуйте топ-3 тренда AI в 2025 году.",
    expected_output="Краткое описание 3 трендов.",
    agent=researcher
)

crew = Crew(agents=[researcher], tasks=[task])
result = crew.kickoff()
print(result)
```

## Когда использовать CrewAI

- Конвейеры генерации контента
- Исследования и аналитика
- Обработка данных в несколько этапов
- Любая задача, где полезно разделение труда

## Итого

CrewAI позволяет легко создавать команды AI-агентов на Python. Каждый агент имеет роль, задачи и инструменты. Вместе они решают сложные задачи лучше, чем один промпт.
