# Понимание задач

## Что такое задача?

Задача (Task) — это конкретная работа, которую вы поручаете агенту. Она описывает **что** нужно сделать и **какой результат** ожидается.

## Создание задачи

```python
from crewai import Task

research_task = Task(
    description="Исследуйте последние тренды возобновляемой энергетики на 2025 год.",
    expected_output="Список из 5 трендов с кратким описанием.",
    agent=researcher
)
```

## Основные параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `description` | str | Описание задачи |
| `expected_output` | str | Ожидаемый формат результата |
| `agent` | Agent | Кто выполняет задачу |
| `tools` | list | Инструменты для задачи |
| `output_file` | str | Сохранить результат в файл |
| `context` | list | Задачи, результат которых передаётся в эту |
| `async_execution` | bool | Выполнять параллельно |

## Зависимости через контекст

```python
research_task = Task(
    description="Исследуйте тренды AI.",
    expected_output="Список из 5 трендов.",
    agent=researcher
)

writing_task = Task(
    description="Напишите статью на основе исследования.",
    expected_output="Статья на 500 слов.",
    agent=writer,
    context=[research_task]  # получает результат исследования
)
```

## Сохранение в файл

```python
report_task = Task(
    description="Напишите финальный отчёт.",
    expected_output="Отчёт в формате markdown.",
    agent=writer,
    output_file="report.md"
)
```

## YAML-конфигурация

```yaml
research_task:
  description: "Исследуйте тренды в {topic}."
  expected_output: "Список из 5 ключевых трендов."
  agent: researcher

writing_task:
  description: "Напишите статью о {topic} на основе исследования."
  expected_output: "Статья на 500 слов в markdown."
  agent: writer
  context:
    - research_task
```

## Советы

1. **Будьте точны** в описаниях — размытые задачи дают размытые результаты
2. **Чётко определяйте ожидаемый результат** — формат, длину, структуру
3. **Используйте context** для цепочки задач
4. **Сохраняйте важные результаты** в файлы

## Итого

Задачи определяют работу. У каждой есть описание, ожидаемый результат и назначенный агент. Используйте context для передачи данных между задачами.
