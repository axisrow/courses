# 通过 SSH 连接到你的 VPS

## 什么是 SSH？

**SSH**（安全外壳协议）就像你的笔记本和 VPS 之间的安全电话线。让你像坐在 VPS 前面一样输入命令。

### 准备工作

你需要：
- VPS 的 **IP 地址**（类似 `123.45.67.89`）
- VPS 的 **密码**（提供商给的）
- 笔记本上的 **终端** 程序

### 获取终端

- **Mac**：打开内置的「终端」应用
- **Windows**：从 Microsoft Store 安装 [Windows Terminal](https://aka.ms/terminal)
- **Linux**：你已经有了！

### 连接

在终端中输入（替换成你的真实 IP）：

```bash
ssh root@123.45.67.89
```

会要求输入密码。输入密码（看不到字符是正常的）然后按回车。

### 连上了！

如果看到类似 `root@server:~#` 的内容，恭喜 — 你现在正在控制云端电脑了。

### SSH 密钥（更安全）

密码可以被猜到。SSH 密钥更安全 — 就像数字锁和钥匙。

ACFS 会在安装过程中为你设置 SSH 密钥。之后就不需要密码了。

### 关键要点

SSH 让你从笔记本控制 VPS。输入 `ssh root@你的IP` 连接。ACFS 稍后会设置密钥认证来提高安全性。
