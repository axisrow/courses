# Первый осмотр

## Добро пожаловать в вашу новую мастерскую

После завершения установки ACFS вы увидите красивый терминал. Давайте осмотримся.

### Терминал теперь выглядит иначе

Теперь используется **Powerlevel10k** — красивая тема, показывающая полезную информацию:
- В какой папке вы находитесь
- В Git-репозитории ли вы
- Сколько заняла последняя команда

### Попробуйте эти команды

```bash
# Посмотрите, что установлено
acfs doctor
```

Проверяет, что всё установлено правильно. Зелёный = хорошо!

```bash
# Запустите обучающий курс
onboard
```

Откроется встроенный курс, проводящий через основы.

### Знакомьтесь с AI-агентами

У вас три AI-помощника. Попробуйте поздороваться:

```bash
# Claude Code (от Anthropic)
cc "Привет, что ты умеешь?"

# Codex CLI (от OpenAI)
cod "Привет, что ты умеешь?"

# Gemini CLI (от Google)
gmi "Привет, что ты умеешь?"
```

> **Примечание**: для каждого агента нужны API-ключи. Установщик поможет их настроить.

### Tmux — ваша страховка

VPS использует **tmux** — инструмент, который сохраняет вашу работу, даже если интернет отключится. Как автосохранение в игре.

```bash
# Посмотрите активные tmux-сессии
tmux list-sessions
```

### Главный вывод

После установки используйте `acfs doctor` для проверки, `onboard` для обучения, и `cc`, `cod` или `gmi` для общения с AI-агентами. Tmux сохраняет работу даже при отключении.
