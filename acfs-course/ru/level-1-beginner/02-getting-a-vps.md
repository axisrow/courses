# Получаем облачный компьютер (VPS)

## Что такое VPS?

**VPS** (Virtual Private Server) — это компьютер, который находится в дата-центре где-то в мире. Вы арендуете его помесячно, как квартиру. Подключаетесь к нему через интернет.

### Почему VPS, а не ноутбук?

| Ваш ноутбук | VPS |
|-------------|-------|
| Выключается, когда закрываете крышку | Работает 24/7 |
| Ограниченная мощность | Вы выбираете мощность |
| Сломается — работа потеряна | Одноразовый — просто создайте новый |
| AI-агенты тормозят систему | Выделен под AI-работу |

### Где арендовать VPS

Популярные провайдеры:
- **OVH** — доступный европейский провайдер
- **Contabo** — бюджетный, хорошие характеристики
- **Hetzner** — отличное соотношение цена/качество
- **DigitalOcean** — удобный интерфейс для новичков

### Что выбрать

- **Операционная система**: Ubuntu 25.10 (ACFS создан для неё)
- **RAM**: минимум 4 ГБ (лучше 8 ГБ)
- **Диск**: минимум 40 ГБ SSD
- **Стоимость**: обычно $5–20/месяц

### Пошагово

1. Зайдите на сайт провайдера VPS
2. Создайте аккаунт
3. Выберите Ubuntu 25.10 как ОС
4. Выберите тариф с минимум 4 ГБ RAM
5. Провайдер выдаст вам **IP-адрес** и **пароль**

### Главный вывод

VPS — это арендованный компьютер в облаке. Вам нужен VPS с Ubuntu 25.10 для использования ACFS. Стоит примерно $5–20 в месяц.
