# Запускаем установщик ACFS

## Волшебная команда

Подключившись к VPS через SSH, выполните одну команду:

```bash
curl -fsSL "https://raw.githubusercontent.com/Dicklesworthstone/agentic_coding_flywheel_setup/main/install.sh" | bash -s -- --yes --mode vibe
```

### Что это делает?

Разберём простыми словами:

- `curl` — скачивает файл из интернета
- `|` — отправляет его в...
- `bash` — программа, которая выполняет команды
- `--yes` — автоматически отвечает «да» на все вопросы
- `--mode vibe` — устанавливает всё с максимальной скоростью

### Что устанавливается (30+ инструментов)

Установщик настраивает:

1. **Красивый терминал** — цветной, с удобными ярлыками
2. **Языки программирования** — Python, JavaScript, Rust, Go
3. **Три AI-агента** — Claude Code, Codex CLI, Gemini CLI
4. **10 инструментов координации** — для управления несколькими AI-агентами
5. **Облачные инструменты** — для развёртывания проектов
6. **Инструменты разработчика** — для поиска, редактирования и управления кодом

### Сколько времени занимает?

Около **30 минут**. Можете наблюдать или сходить за кофе.

### А если что-то пойдёт не так?

Установщик **идемпотентный** — это значит, что его можно безопасно запустить повторно. Если интернет отключится или что-то пойдёт не так, просто запустите ту же команду снова. Он продолжит с того места, где остановился.

### Главный вывод

Одна команда устанавливает всё. Занимает около 30 минут. Если не получилось — запустите снова, установщик к этому готов.
