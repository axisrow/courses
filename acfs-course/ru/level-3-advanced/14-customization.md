# Настройка окружения ACFS

## Делаем по-своему

ACFS поставляется с разумными настройками по умолчанию, но всё можно настроить. Ваши настройки сохраняются при обновлениях.

### Ключевые файлы конфигурации

| Файл | Что настраивает |
|------|----------------|
| `~/.acfs/` | Основная директория конфигурации |
| `~/.zshrc` | Поведение оболочки и алиасы |
| `~/.tmux.conf` | Раскладка tmux и горячие клавиши |

### Переменные окружения

```bash
export ACFS_HOME=~/.acfs
export ACFS_LOG_DIR=/var/log/acfs
export TARGET_USER=ubuntu
```

### Режимы установки

- **`--mode vibe`** — Максимальная скорость. Опасные флаги включены. Для одноразовых VPS.
- Обычный режим — более осторожный, с подтверждениями.

### Настройки агентов

```bash
# Настройки Claude Code:
~/.acfs/claude/settings.json
```

### Сайт-мастер

Для визуальной настройки посетите [agent-flywheel.com](https://agent-flywheel.com) — 13-шаговый мастер с визуальным интерфейсом.

### Безопасность

**Режим vibe** включает флаги:
- `--dangerously-skip-permissions` (Claude)
- `--dangerously-bypass-approvals-and-sandbox` (Codex)
- `--yolo` (Gemini)

Используйте только на:
- ✅ Одноразовых VPS
- ✅ Личных экспериментах
- ❌ Никогда на продакшн-серверах
- ❌ Никогда на общей инфраструктуре

### Главный вывод

ACFS настраивается через переменные окружения, конфиг-файлы и режимы установки. Vibe mode — для скорости на одноразовых серверах, никогда на продакшне.
