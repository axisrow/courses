# Осваиваем Tmux для постоянных сессий

## Зачем нужен Tmux

Когда вы закрываете ноутбук или пропадает WiFi, обычная SSH-сессия умирает — и всё, что работало, останавливается. **Tmux** полностью решает эту проблему.

### Основы

Представьте tmux как несколько рабочих столов на вашем VPS:

- **Сессии** — отдельные рабочие пространства
- **Окна** — как вкладки в браузере
- **Панели** — как разделение экрана

### Основные команды

```bash
# Создать именованную сессию
tmux new -s mywork

# Отсоединиться (сессия продолжает работать)
# Нажмите: Ctrl+B, затем D

# Список всех сессий
tmux ls

# Подключиться к сессии
tmux attach -t mywork
```

### Настройки Tmux от ACFS

ACFS заранее настраивает tmux:
- Поддержка мыши (можно кликать по панелям)
- Удобные горячие клавиши
- Статусная строка с полезной информацией

### Разделение экрана

```bash
# Разделить горизонтально (верх/низ)
Ctrl+B, затем "

# Разделить вертикально (лево/право)
Ctrl+B, затем %

# Переключаться между панелями
Ctrl+B, затем стрелки
```

### Рабочий процесс

```
Сессия "coding"           Сессия "monitoring"
├── Окно 1: Агент 1       ├── Окно 1: Логи
├── Окно 2: Агент 2       └── Окно 2: Статистика
└── Окно 3: Тесты
```

### Главный вывод

Tmux сохраняет работу при отключении. Запомните: `tmux new -s ИМЯ`, `Ctrl+B D` для отсоединения, `tmux attach -t ИМЯ` для подключения. Это ваша страховка.
