# NTM: ваш оркестратор агентов

## Что такое NTM?

**NTM** (Neuro-Tmux Manager) — инструмент, управляющий AI-агентами как дирижёр оркестром. Вместо ручного открытия терминалов и ввода команд для каждого агента, NTM делает всё из одного места.

### Зачем он нужен

Без NTM:
- Открыть терминал → запустить Claude → дать задачу → ждать → переключить терминал → запустить Codex...

С NTM:
- Открыть NTM → назначить задачи 3 агентам сразу → они работают параллельно

### Запуск NTM

```bash
ntm
```

Откроется **TUI** (текстовый интерфейс) — визуальная программа в терминале.

### Ключевые понятия

- **Слоты** — пронумерованные позиции для агентов (как парковочные места)
- **Промпты** — инструкции, которые вы отправляете агентам
- **Рассылка** — отправка одной инструкции всем агентам сразу

### Базовый рабочий процесс

1. Откройте NTM: `ntm`
2. Запустите агентов в слоты
3. Отправьте промпты одному или всем агентам
4. Наблюдайте за параллельной работой
5. Изучите результаты

### Палитра команд

Нажмите горячую клавишу для вызова палитры команд — быстрого меню всех доступных действий.

### Когда использовать NTM

- Проект выигрывает от нескольких точек зрения
- Нужно сравнить решения разных AI-агентов
- Хотите распараллелить работу (фронтенд + бэкенд + тесты)

### Главный вывод

NTM позволяет управлять несколькими AI-агентами из одного интерфейса. Запускайте агентов, рассылайте промпты и наблюдайте за параллельной работой. Это ваш центр управления.
