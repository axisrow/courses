# Обновление системы

## Зачем обновляться

ACFS активно развивается. Обновления приносят новые функции, исправления безопасности и улучшения.

### Обновление ACFS

```bash
acfs update
```

Эта команда:
- Скачивает последний код ACFS
- Обновляет конфигурации
- Обновляет версии инструментов
- Сохраняет ваши настройки

### Проверка здоровья системы

```bash
acfs doctor
```

Запускайте периодически. Проверяет:
- Все инструменты установлены и работают
- Версии актуальны
- Конфигурация корректна

### Обновление отдельных инструментов

```bash
bun upgrade       # JavaScript
rustup update     # Rust
uv self update    # Python
```

### Фиксация версий

Для важных проектов можно зафиксировать версию ACFS:

```bash
ACFS_REF=v0.5.0 curl -fsSL "..." | bash -s -- --yes --mode vibe
```

### Главный вывод

`acfs update` — для обновлений. `acfs doctor` — для проверки здоровья. Для критичных проектов фиксируйте версию через `ACFS_REF`.
