# Быстрый старт

## Введение

DeerFlow запущен и работает. Пора попробовать его в деле! В этом уроке вы сделаете свои первые запросы и увидите, на что способен суперагент.

## Открываем веб-интерфейс

1. Убедитесь, что DeerFlow запущен (`make docker-start` или `make dev`)
2. Откройте браузер и перейдите на **http://localhost:2026**
3. Вы увидите чат-интерфейс, похожий на ChatGPT

## Ваш первый запрос

Введите в поле чата:

```
Найди информацию о последних новостях в области искусственного интеллекта
и напиши краткий обзор на 5 пунктов.
```

### Что происходит «за кулисами»

1. **Главный агент** получает ваш запрос
2. Он решает, что нужно использовать **поиск в интернете**
3. Запускает инструмент `web_search` для поиска новостей
4. Использует `web_fetch` для загрузки содержимого страниц
5. Анализирует найденное и формирует ответ

Вы увидите процесс в реальном времени — DeerFlow показывает, какие инструменты использует.

## Пробуем создание кода

Введите:

```
Напиши Python-скрипт, который скачивает прогноз погоды
и выводит температуру на ближайшие 3 дня.
Запусти его и покажи результат.
```

DeerFlow не только напишет код, но и **выполнит** его в песочнице. Вы увидите реальный результат.

## Создаём документ

Попробуйте:

```
Создай отчёт о языке программирования Rust:
история, преимущества, примеры использования.
Сохрани как файл в формате Markdown.
```

DeerFlow создаст файл в своей файловой системе, и вы сможете его скачать.

## Режим плана (Plan Mode)

Для сложных задач включите **режим плана** — DeerFlow сначала составит список шагов, а потом будет их выполнять:

```
[Включите Plan Mode в интерфейсе]

Создай веб-страницу — портфолио для фотографа.
Должны быть: главная, галерея, контакты.
Используй современный дизайн.
```

Вы увидите чек-лист задач и прогресс выполнения каждой.

## Загрузка файлов

DeerFlow поддерживает загрузку файлов для анализа:

1. Нажмите кнопку загрузки в интерфейсе
2. Выберите файл (PDF, Excel, Word, PowerPoint)
3. Попросите агента проанализировать его:

```
Проанализируй загруженный файл и создай краткое резюме.
```

Поддерживаемые форматы: PDF, PPTX, XLSX, DOCX — они автоматически конвертируются в текст.

## Полезные советы

- **Будьте конкретны**: чем точнее задача, тем лучше результат
- **Разбивайте на шаги**: для сложных задач используйте Plan Mode
- **Экспериментируйте**: DeerFlow может удивить вас своими возможностями
- **Проверяйте результат**: ИИ может ошибаться, всегда проверяйте важные данные

## Итоги

- Веб-интерфейс DeerFlow доступен по адресу http://localhost:2026
- Агент может искать информацию, писать и запускать код, создавать файлы
- Режим плана помогает со сложными многоэтапными задачами
- Можно загружать файлы для анализа
