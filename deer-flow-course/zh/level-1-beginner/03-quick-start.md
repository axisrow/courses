# 快速入门

## 简介

DeerFlow已经启动运行。现在让我们来试试！在本课中，您将发出第一个请求，看看超级智能体能做什么。

## 打开Web界面

1. 确保DeerFlow正在运行（`make docker-start`或`make dev`）
2. 在浏览器中访问 **http://localhost:2026**
3. 您会看到类似ChatGPT的聊天界面

## 您的第一个请求

在聊天中输入：

```
搜索关于人工智能最新新闻的信息，并写一个包含5个要点的简要概述。
```

### 幕后发生了什么

1. **主智能体**接收您的请求
2. 决定使用**网络搜索**
3. 运行`web_search`工具查找新闻
4. 使用`web_fetch`加载页面内容
5. 分析发现的内容并组织回复

您可以实时看到过程——DeerFlow会显示它正在使用哪些工具。

## 尝试代码生成

```
编写一个Python脚本，下载天气预报并显示未来3天的温度。运行它并显示结果。
```

DeerFlow不仅会编写代码，还会在沙箱中**执行**它。

## 创建文档

```
创建一份关于Rust编程语言的报告：历史、优势和实际应用案例。保存为Markdown文件。
```

## 计划模式

对于复杂任务，启用**计划模式**——DeerFlow会先创建步骤列表，然后执行：

```
[在界面中启用计划模式]

创建一个摄影师的作品集网站。
包括：首页、画廊和联系方式。
使用现代设计。
```

## 文件上传

DeerFlow支持上传文件进行分析。点击上传按钮，选择文件（PDF、Excel、Word、PowerPoint），然后让智能体分析它。

## 提示

- **具体明确**：任务描述越精确，结果越好
- **分解任务**：复杂任务使用计划模式
- **大胆实验**：DeerFlow的能力可能会让您惊喜
- **验证结果**：AI可能出错——始终检查重要数据

## 总结

- DeerFlow的Web界面在 http://localhost:2026
- 智能体可以搜索、编写和运行代码、创建文件
- 计划模式适用于复杂的多步骤任务
- 您可以上传文件进行分析
