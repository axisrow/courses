# 什么是DeerFlow

## 简介

想象一下，有一个智能助手，它不仅能回答问题，还能真正**做事**：搜索互联网、编写代码、创建报告、演示文稿，甚至网页。这就是**DeerFlow**。

**DeerFlow**（Deep Exploration and Efficient Research Flow）是字节跳动（TikTok的创造者）开发的免费开源平台。它是一个"超级智能体"——一个由AI驱动的程序，能够：

- 🔍 在互联网上**搜索信息**
- 💻 在安全环境中**编写和运行代码**
- 📝 **创建文档**、报告和演示文稿
- 🧠 跨会话**记住**您的偏好
- 👥 将任务**委派**给助手智能体

## 核心概念

### 什么是"智能体"？

**智能体**是一个能够独立做出决定并采取行动的AI程序。与普通聊天机器人不同，智能体不仅仅是回复文字——它可以调用工具、运行代码并与外部服务交互。

### 什么是"超级智能体"？

**超级智能体**是管理其他智能体的智能体。它接收您的任务，将其分解为多个部分，并将每个部分分配给单独的助手（子智能体）。就像一个分配工作的团队负责人。

### 什么是"沙箱"？

**沙箱**是运行代码的隔离环境。代码在单独的容器中运行，不会损害您的计算机。就像一个可以随意实验而没有后果的房间。

## 工作原理（简化版）

```
您给出任务
       ↓
主智能体分析任务
       ↓
分解为子任务 → 启动子智能体
       ↓
子智能体使用工具：
  • 网络搜索
  • 代码编写
  • 文件创建
       ↓
收集结果 → 您收到最终答案
```

## DeerFlow与ChatGPT的区别

| 功能 | ChatGPT | DeerFlow |
|------|---------|----------|
| 回答问题 | ✅ | ✅ |
| 独立文件系统 | ❌ | ✅ |
| 沙箱中执行代码 | 有限 | ✅ 完整 |
| 子智能体 | ❌ | ✅ |
| 长期记忆 | 有限 | ✅ |
| 开源 | ❌ | ✅ |
| 选择任意AI模型 | ❌ | ✅ |

## 您可以用DeerFlow做什么？

- **研究**：给出主题——获得带来源的详细报告
- **编程**：描述任务——获得可运行的代码
- **演示文稿**：给出主题——获得幻灯片
- **网页**：描述设计——获得HTML/CSS
- **数据分析**：上传文件——获得可视化和洞察
- **自动化**：任何多步骤任务

## 总结

- DeerFlow是一个开源的"超级智能体"平台
- 它不仅能回答问题，还能执行真正的任务
- 使用子智能体、沙箱、记忆和可扩展技能
- 支持任意AI模型（GPT-4、Claude、DeepSeek等）
- 由字节跳动创建，由社区开发
