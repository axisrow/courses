# 第一个任务

## 简介

是时候练习了！在本课中，我们将用DeerFlow完成几个典型任务。

## 任务1：主题研究

```
研究：什么是WebAssembly？
包括：
1. 简单定义
2. 历史
3. 主要优势
4. 实际应用案例
5. 未来展望
将结果保存为Markdown文件。
```

DeerFlow将搜索网络，阅读多个来源，综合信息，并创建文件。

## 任务2：编写和运行代码

```
编写一个Python程序：
1. 生成100个1到1000之间的随机数
2. 计算平均值、中位数和众数
3. 创建直方图并保存为PNG
4. 在控制台输出统计数据

运行程序并显示结果。
```

## 任务3：处理上传文件

1. 上传任何PDF或文本文件
2. 请求：

```
分析上传的文件：
- 写一个简要摘要（5-7句话）
- 突出关键主题
- 提出3个讨论问题
```

## 任务4：创建网页

```
为一家名为"晨间咖啡"的咖啡店创建一个漂亮的HTML登陆页面。
要求：
- 现代简约设计
- 部分包括：横幅、菜单（5种饮品）、关于我们、联系方式
- 响应式布局（适配手机）
- 所有内容在一个HTML文件中（内联样式）
```

## 任务5：使用计划模式的自动化

启用计划模式并输入：

```
为一个虚构的库"SuperCalc"创建一个迷你文档网站：
1. 带描述的首页
2. "安装"页面及说明
3. "API参考"页面包含5个函数
4. "示例"页面包含3个使用示例
所有页面应通过导航链接。
```

## 高效工作的提示

1. **上下文保持**：在对话中DeerFlow记住之前讨论的一切
2. **迭代改进**：如果结果不完美——要求修改
3. **使用计划模式**处理需要10分钟以上的任务
4. **上传示例**：如果需要特定风格——上传样本
5. **检查文件**：创建的文件可以通过界面下载

## 总结

- DeerFlow处理研究、代码、文档和网页
- 复杂任务使用计划模式
- 在单个对话中可以迭代改进结果
- 所有创建的文件都可以下载

🎉 恭喜！您已完成第1级。继续第2级，学习DeerFlow的高级功能。
