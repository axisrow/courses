# Урок 1. Что такое Claude Code и зачем он нужен

## Зачем это нужно

Представьте, что у вас есть умный помощник, который понимает программный код, может писать его за вас, находить ошибки и объяснять, как всё работает. Именно это делает **Claude Code** — инструмент от компании Anthropic.

Даже если вы никогда не программировали, Claude Code поможет вам автоматизировать рутинные задачи, создавать сайты, скрипты и приложения, просто описывая словами, что вам нужно.

---

## Что такое Claude Code

**Claude Code** — это AI-помощник для работы с кодом (программами). Вы пишете ему задачу обычным человеческим языком, а он:

- Анализирует ваш проект (все файлы и папки)
- Пишет или редактирует код
- Запускает команды
- Ищет и исправляет ошибки
- Создаёт коммиты в Git (система для хранения истории изменений кода)

> **Простыми словами:** вы говорите «добавь кнопку на сайт», а Claude Code сам находит нужный файл, пишет код и показывает вам, что изменилось.

---

## Чем Claude Code отличается от ChatGPT

| | ChatGPT | Claude Code |
|---|---------|-------------|
| **Где работает** | В браузере (окно чата) | В терминале, VS Code, JetBrains, браузере, десктоп-приложении |
| **Видит ваш проект** | Нет — вы вручную копируете код | Да — читает все файлы проекта целиком |
| **Может менять файлы** | Нет — только показывает код | Да — редактирует файлы напрямую (с вашего разрешения) |
| **Запускает команды** | Нет | Да — может запускать тесты, устанавливать пакеты, делать коммиты |
| **Работает с Git** | Нет | Да — создаёт ветки, коммиты, помогает с конфликтами |

> **Git** — это система контроля версий. Думайте о ней как о «машине времени» для вашего кода: можно посмотреть, что менялось, и откатить изменения.

## Где работает Claude Code

Claude Code доступен в **пяти местах** — выбирайте то, что вам удобнее:

### 1. Терминал (командная строка)
Это текстовое окно, куда вы вводите команды. Основной и самый мощный способ работы с Claude Code. Вы просто набираете `claude` и начинаете диалог.

> **Терминал** — программа для ввода текстовых команд. На Mac — приложение «Terminal», на Windows — «PowerShell» или «Командная строка».

### 2. VS Code
**VS Code** — популярный бесплатный редактор кода от Microsoft. Claude Code встраивается прямо в него как расширение (плагин). Вы видите код и ответы Claude рядом.

### 3. JetBrains (IntelliJ IDEA, PyCharm и др.)
Семейство профессиональных редакторов кода. Claude Code работает как плагин — так же, как в VS Code.

### 4. Десктоп-приложение
Отдельная программа для macOS и Windows. Не нужен ни редактор кода, ни терминал — просто скачайте и запустите.

### 5. Браузер (claude.ai/code)
Работает прямо в вашем браузере. Не нужно ничего устанавливать — откройте сайт и начните. Удобно для задач, которые выполняются долго: запустили и вернулись позже.

## Что умеет Claude Code — примеры

Вот реальные задачи, которые можно решить одной фразой:

| Ваш запрос | Что сделает Claude Code |
|---|---|
| `что делает этот проект?` | Проанализирует все файлы и объяснит |
| `добавь валидацию в форму регистрации` | Найдёт нужный файл и добавит проверку полей |
| `исправь баг — форма отправляется пустой` | Найдёт причину и исправит |
| `напиши тесты для калькулятора` | Создаст файл с тестами |
| `сделай коммит с описанием изменений` | Создаст коммит в Git с понятным сообщением |
