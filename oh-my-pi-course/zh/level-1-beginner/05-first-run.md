# 首次运行

## 介绍

你已经安装了 OMP 并配置了 API 密钥。让我们启动代理并尝试基本功能。

## 启动

```bash
omp
```

你会看到带有 logo 和提示的欢迎界面。

## 配置模型角色

使用 `/model` 分配模型：

- **default** — 常规工作
- **smol** — 快速/低成本任务
- **slow** — 深度分析
- **plan** — 规划模式
- **commit** — 提交生成

## 第一步操作

### 与代理对话

```
> 当前目录中有哪些文件？
```

### 使用 @ 引用文件

```
> 解释 @src/main.ts
```

### 运行命令

```
> !git status
```

（`!` 前缀执行命令并将输出包含在上下文中。）

### 尝试规划模式

```
/plan
> 规划认证模块的重构方案
```

## 会话管理

```bash
omp -c              # 继续上次会话
omp -r              # 选择会话
omp --no-session    # 不保存
```

## 常用斜杠命令

| 命令 | 描述 |
|------|------|
| `/model` | 模型选择器 |
| `/plan` | 规划模式 |
| `/settings` | 设置 |
| `/extensions` | 扩展和上下文 |
| `/compact` | 压缩上下文 |
| `/export` | 导出为 HTML |
| `/exit` | 退出 |

## 总结

你已经启动了 OMP、配置了模型并尝试了基本功能。下一级别将介绍高级工具。
