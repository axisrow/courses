# Что такое Oh My Pi

## Введение

Oh My Pi (OMP) — это AI-агент для написания кода, который работает прямо в терминале. Это форк проекта [pi-mono](https://github.com/badlogic/pi-mono) от Mario Zechner, расширенный Can Bölük.

## Что это даёт

OMP — это полноценная среда для работы с AI прямо в командной строке:

- **Интерактивный терминальный интерфейс** — работайте с AI как в чате
- **Встроенные инструменты** — редактирование файлов, поиск, grep, bash, Python, браузер
- **Сессии с ветвлением** — сохраняйте и возвращайтесь к любой точке диалога
- **Расширяемость** — свои команды, хуки, навыки, инструменты
- **Множество провайдеров** — Anthropic, OpenAI, Google, xAI, Groq и десятки других

## Чем отличается от pi-mono

OMP добавляет поверх pi-mono:

- **Hashline-редактирование** — уникальные хеш-якоря для каждой строки, нет ошибок «строка не найдена»
- **Нативный движок на Rust** — 7500 строк Rust для grep, shell, подсветки синтаксиса
- **LSP-интеграция** — поддержка 40+ языков с форматированием и диагностикой
- **Commit Tool** — AI-генерация коммитов с разбивкой по хункам
- **Python Tool** — IPython-ядро с хелперами
- **TTSR** — правила, которые срабатывают только когда нужны (ноль контекста до активации)
- **Субагенты** — параллельное выполнение задач
- **65+ тем оформления**

## Пример запуска

```bash
omp
> Покажи все .ts файлы в src/
```
