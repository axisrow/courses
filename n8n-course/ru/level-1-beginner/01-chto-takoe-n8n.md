# Что такое n8n?

## Введение

n8n (произносится «ноудмейшн») — это бесплатная платформа автоматизации с открытым исходным кодом. Она позволяет соединять разные приложения и сервисы, чтобы они работали автоматически — без написания кода.

## Зачем нужен n8n?

- **Экономия времени** — автоматизация рутинных задач
- **Без кода** — визуальный редактор с перетаскиванием
- **Свой сервер** — данные остаются у вас
- **Открытый код** — бесплатно, можно модифицировать
- **200+ интеграций** — Slack, Google Sheets, GitHub и многое другое

## Как это работает?

В n8n вы создаёте **рабочие процессы** (workflows). Рабочий процесс — это цепочка шагов (называемых **нодами**), соединённых между собой. Каждый нод выполняет действие — читает данные, отправляет сообщение или преобразует информацию.

Представьте конвейер на заводе: данные входят с одной стороны, обрабатываются на каждой станции и выходят преобразованными с другой.

## Ключевые понятия

| Понятие | Описание |
|---------|----------|
| **Workflow** | Последовательность нодов для автоматизации задачи |
| **Нод (Node)** | Один шаг, выполняющий действие |
| **Триггер** | Специальный нод, запускающий workflow |
| **Соединение** | Связь между нодами для передачи данных |
| **Выполнение** | Один запуск workflow от начала до конца |

## Примеры использования

1. **Уведомления** — получить сообщение в Slack при важном письме
2. **Синхронизация** — копировать строки из Google Sheets в базу данных
3. **Соцсети** — автопост в Twitter при публикации статьи
4. **Бэкапы** — сохранять вложения из почты в Google Drive
