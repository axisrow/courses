# Установка и настройка

## Требования

- Компьютер с Windows, macOS или Linux
- **Node.js** версии 18 или новее
- Базовые навыки работы с терминалом

## Способ 1: Установка через npm

```bash
npm install n8n -g
```

Запуск:

```bash
n8n start
```

Откройте в браузере `http://localhost:5678`.

## Способ 2: Docker

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v n8n_data:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```

## Способ 3: n8n Cloud

Не хотите устанавливать? Используйте [n8n Cloud](https://n8n.io/cloud/) — облачную версию, готовую за секунды.

## Первый запуск

При первом открытии n8n вы:

1. Создадите **учётную запись** (email и пароль)
2. Увидите **редактор workflow** — чистый холст для автоматизаций
3. Заметите **панель нодов** слева — здесь все доступные ноды

## Интерфейс

- **Холст** — основная область для нодов и соединений
- **Панель нодов** — поиск и добавление нодов (кнопка `+`)
- **Список выполнений** — история запусков workflow
- **Настройки** — конфигурация n8n

## Проверка установки

1. Нажмите `+` на холсте
2. Найдите «Manual Trigger» и добавьте
3. Добавьте нод «Set» и соедините
4. Нажмите **Execute Workflow**

Зелёные галочки — всё работает!

## Итог

n8n установлен и работает. В следующем уроке вы создадите свой первый workflow.
