# Ваш первый Workflow

## Цель

Создадим workflow, который получает случайную шутку из интернета.

## Шаг 1: Создание workflow

1. Откройте n8n в браузере
2. Нажмите **Add Workflow**
3. Назовите: «My First Workflow»

## Шаг 2: Добавление триггера

1. Нажмите `+` на холсте
2. Найдите **Manual Trigger**
3. Добавьте на холст

## Шаг 3: HTTP-запрос

1. Нажмите `+` справа от триггера
2. Найдите **HTTP Request**
3. URL: `https://official-joke-api.appspot.com/random_joke`
4. Метод: **GET**
5. Нажмите **Execute Step** для проверки

Вы увидите шутку с полями `setup` и `punchline`.

## Шаг 4: Форматирование

1. Добавьте нод **Set** после HTTP Request
2. **Add Value** → **String**
3. Имя: `joke`
4. Значение: `{{ $json.setup }} — {{ $json.punchline }}`

## Шаг 5: Запуск

1. Нажмите **Execute Workflow**
2. Проверьте — все ноды должны показать зелёные галочки
3. Кликните на последний нод, чтобы увидеть результат

## Поток данных

Данные идут слева направо:

```
Manual Trigger → HTTP Request → Set
                 (получает)      (форматирует)
```

## Сохранение

Нажмите **Save** (Ctrl+S).

## Практика

- Поменяйте URL на другой API
- Добавьте больше полей в Set
- Добавьте третий нод

## Итог

Вы создали свой первый workflow! Далее — знакомство с основными нодами.
