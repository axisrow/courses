# IDE Integration

## Introduction

Working with Claude Code, Cursor, Codex, and OpenCode.

## Claude Code

Primary platform with best support:

```bash
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
```

## Cursor

Integration via Agent chat:
```
/plugin-add superpowers
```

## Codex

Manual setup via fetch instructions.

## OpenCode

Similar to Codex with manual setup.

## Platform Comparison

| Feature | Claude Code | Cursor | Codex | OpenCode |
|---------|-------------|--------|-------|----------|
| Marketplace | ✓ | ✓ | ✗ | ✗ |
| Auto-update | ✓ | ✓ | ✗ | ✗ |
| Subagents | ✓ | ✓ | Partial | Partial |
| Ease | ★★★ | ★★★ | ★★ | ★★ |

## Summary

- Claude Code — best integration
- Cursor — good option with GUI
- Codex and OpenCode — require manual setup
- All platforms support the core workflow
