# Contributing to Superpowers

## Introduction

How to contribute to the Superpowers project.

## How to Contribute

1. **Fork** the repository: https://github.com/obra/superpowers
2. Create a branch for your skill
3. Follow the `writing-skills` skill
4. Submit a PR

## What to Contribute

### New Skills
- Technology-specific skills (React, Python, Go)
- Process skills (CI/CD, deployment)
- Improvements to existing skills

### Documentation
- README improvements
- Usage examples
- Translations

### Bug Reports
- Problem description
- Reproduction steps
- Expected vs. actual behavior

## Quality Standards

- Skills must be **specific** and **verifiable**
- Include **anti-patterns**
- Specify **activation triggers**
- Test on **real** tasks

## Community

- **Issues**: https://github.com/obra/superpowers/issues
- **Sponsorship**: https://github.com/sponsors/obra
- **License**: MIT

## Summary

- Contribute via standard GitHub workflow
- Skills follow the writing-skills template
- Quality over quantity
- Community is open to new ideas
