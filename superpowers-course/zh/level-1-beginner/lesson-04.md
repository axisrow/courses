# 第一个 Superpowers 项目

## 简介

从想法到可运行代码，构建你的第一个项目。

## 准备工作

1. 确保已安装 Superpowers（第 2 课）
2. 创建空项目文件夹
3. 初始化 git 仓库

```bash
mkdir my-first-project
cd my-first-project
git init
```

## 步骤 1：描述你的想法

打开编码代理，描述任务：
```
我想创建一个 CLI 工具来统计文本文件中的单词数。
它应该接受文件路径，输出单词数、行数和字符数。
```

## 步骤 2：回答问题

代理会开始头脑风暴，如实回答问题。

## 步骤 3：批准规格说明

代理会展示设计文档。阅读并确认或请求修改。

## 步骤 4：批准计划

代理创建任务计划。审查并说「开始」。

## 步骤 5：观察执行

代理启动子代理，每个子代理：
- 编写测试
- 看到测试失败（RED）
- 编写代码
- 看到测试通过（GREEN）
- 重构（REFACTOR）

## 结果

你将得到：
- 可运行的 CLI 工具
- 完整的测试套件
- 整洁的代码
- 有意义的 git 历史

## 总结

- 用简单语言描述你的想法
- 让代理提问
- 批准规格说明和计划
- 观察子代理自主工作
