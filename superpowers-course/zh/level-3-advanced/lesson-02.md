# 创建自定义技能

## 简介

如何为 Superpowers 创建自定义技能。

## 技能结构

```
skills/my-skill/
└── SKILL.md
```

### SKILL.md 格式

```markdown
# 技能名称

## 何时使用
激活触发器描述。

## 流程
代理的分步指令。

## 示例
具体使用示例。

## 反模式
应该避免什么。
```

## `writing-skills` 元技能

告诉代理：
```
我想创建一个自动生成文档的技能
```

## 好技能的原则

- **具体性** — 精确指令，非笼统建议
- **触发器** — 明确激活条件
- **可验证性** — 如何检查技能有效
- **极简主义** — 只包含必要内容

## 总结

- 技能 = SKILL.md 文件加指令
- 使用 writing-skills 元技能
- 技能应具体且可验证
- 在真实任务上测试
