# 为 Superpowers 做贡献

## 简介

如何为 Superpowers 项目做贡献。

## 如何贡献

1. **Fork** 仓库：https://github.com/obra/superpowers
2. 为你的技能创建分支
3. 遵循 `writing-skills` 技能
4. 提交 PR

## 可以贡献什么

### 新技能
- 特定技术的技能（React、Python、Go）
- 流程技能（CI/CD、部署）
- 改进现有技能

### 文档
- README 改进
- 使用示例
- 翻译

### Bug 报告
- 问题描述
- 重现步骤
- 预期与实际行为

## 质量标准

- 技能必须**具体**且**可验证**
- 包含**反模式**
- 指定**激活触发器**
- 在**真实**任务上测试

## 社区

- **Issues**: https://github.com/obra/superpowers/issues
- **赞助**: https://github.com/sponsors/obra
- **许可证**: MIT

## 总结

- 通过标准 GitHub 流程贡献
- 技能遵循 writing-skills 模板
- 质量重于数量
- 社区欢迎新想法
