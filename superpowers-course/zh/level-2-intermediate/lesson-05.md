# IDE 集成

## 简介

与 Claude Code、Cursor、Codex 和 OpenCode 配合使用。

## Claude Code

主要平台，支持最好：
```bash
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
```

## Cursor

通过 Agent 聊天集成：
```
/plugin-add superpowers
```

## Codex 和 OpenCode

需要手动设置。

## 平台对比

| 功能 | Claude Code | Cursor | Codex | OpenCode |
|------|-------------|--------|-------|----------|
| 市场 | ✓ | ✓ | ✗ | ✗ |
| 自动更新 | ✓ | ✓ | ✗ | ✗ |
| 子代理 | ✓ | ✓ | 部分 | 部分 |
| 易用性 | ★★★ | ★★★ | ★★ | ★★ |

## 总结

- Claude Code — 最佳集成
- Cursor — 带 GUI 的好选择
- Codex 和 OpenCode — 需要手动设置
- 所有平台支持核心工作流
