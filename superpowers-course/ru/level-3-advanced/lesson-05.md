# Контрибьюция в Superpowers

## Введение

Как внести свой вклад в проект Superpowers.

## Как внести вклад

1. **Форкните** репозиторий: https://github.com/obra/superpowers
2. Создайте ветку для вашего навыка
3. Следуйте навыку `writing-skills` для создания
4. Отправьте PR

## Что можно контрибьютить

### Новые навыки
- Навыки для конкретных технологий (React, Python, Go)
- Навыки для конкретных процессов (CI/CD, деплой)
- Улучшения существующих навыков

### Документация
- Улучшения README
- Примеры использования
- Переводы

### Баг-репорты
- Описание проблемы
- Шаги воспроизведения
- Ожидаемое vs. фактическое поведение

## Процесс создания навыка

```bash
# 1. Форк и клон
git clone https://github.com/YOUR_USERNAME/superpowers
cd superpowers

# 2. Новая ветка
git checkout -b skill/my-new-skill

# 3. Создайте навык
mkdir skills/my-new-skill
# Напишите SKILL.md следуя writing-skills

# 4. Протестируйте
# Используйте навык в реальном проекте

# 5. PR
git push origin skill/my-new-skill
```

## Стандарты качества

- Навык должен быть **конкретным** и **проверяемым**
- Включите **антипаттерны** — чего НЕ делать
- Укажите **триггеры** активации
- Протестируйте на **реальных** задачах
