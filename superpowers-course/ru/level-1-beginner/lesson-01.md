# Что такое Superpowers

## Введение

Знакомство с фреймворком Superpowers — набором навыков (skills) для coding-агентов.

Superpowers — это фреймворк от Jesse Vincent (obra), который превращает обычного coding-агента в систематического разработчика. Вместо хаотичной генерации кода агент следует чёткому процессу: сначала понимает задачу, потом проектирует решение, потом пишет код.

## Что такое Superpowers

Superpowers — это набор **навыков (skills)**, которые автоматически активируются в вашем coding-агенте. Навыки — это инструкции, которые направляют агента на каждом этапе разработки.

### Ключевые принципы

- **Test-Driven Development** — сначала тесты, потом код
- **Систематичность** — процесс вместо угадывания
- **Снижение сложности** — простота как главная цель
- **Доказательства вместо заявлений** — проверяй перед тем, как объявить успех

### Основной рабочий процесс

1. **Brainstorming** — агент уточняет задачу, задаёт вопросы
2. **Спецификация** — формализует требования в документ
3. **План** — разбивает работу на задачи по 2-5 минут
4. **Субагенты** — запускает отдельных агентов на каждую задачу
5. **Code Review** — проверяет результат
6. **Завершение** — мержит или создаёт PR

### Какие навыки есть

| Категория | Навыки |
|-----------|--------|
| Тестирование | test-driven-development |
| Отладка | systematic-debugging, verification-before-completion |
| Совместная работа | brainstorming, writing-plans, executing-plans, subagent-driven-development |
| Git | using-git-worktrees, finishing-a-development-branch |
| Мета | writing-skills, using-superpowers |

## Пример

Вы говорите агенту: «Сделай REST API для заметок». Без Superpowers агент сразу начнёт писать код. С Superpowers:

1. Агент спросит: «Какие поля у заметки? Нужна ли авторизация? Какая БД?»
2. Покажет спецификацию по частям
3. Составит план из 10-15 задач
4. Выполнит каждую задачу через субагента с TDD

## Итоги

- Superpowers — это фреймворк навыков для coding-агентов
- Навыки активируются автоматически
- Основа: спецификация → план → субагенты → TDD
- Работает с Claude Code, Cursor, Codex, OpenCode
