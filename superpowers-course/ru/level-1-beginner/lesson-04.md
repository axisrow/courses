# Первый проект с Superpowers

## Введение

Создаём первый проект от идеи до работающего кода.

## Подготовка

1. Убедитесь, что Superpowers установлен (урок 2)
2. Создайте пустую папку для проекта
3. Инициализируйте git-репозиторий

```bash
mkdir my-first-project
cd my-first-project
git init
```

## Шаг 1: Опишите идею агенту

Откройте coding-агент и опишите задачу:

```
Я хочу создать CLI-утилиту для подсчёта слов в текстовых файлах.
Она должна принимать путь к файлу и выводить количество слов, строк и символов.
```

## Шаг 2: Отвечайте на вопросы

Агент начнёт brainstorming. Отвечайте на вопросы честно:

- «Какой язык?» → «TypeScript»
- «Нужна ли поддержка нескольких файлов?» → «Да»
- «Формат вывода?» → «Таблица в консоли»

## Шаг 3: Утвердите спецификацию

Агент покажет дизайн-документ. Прочитайте и подтвердите или попросите изменить.

## Шаг 4: Утвердите план

Агент составит план из задач. Просмотрите и скажите «go».

## Шаг 5: Наблюдайте за выполнением

Агент запустит субагентов. Каждый:
- Напишет тест
- Увидит, что тест падает (RED)
- Напишет код
- Увидит, что тест проходит (GREEN)
- Сделает рефакторинг (REFACTOR)

## Что получится

В итоге у вас будет:
- Рабочая CLI-утилита
- Полный набор тестов
- Чистый код
- Git-история с осмысленными коммитами

## Итоги

- Начните с описания идеи простым языком
- Позвольте агенту задать вопросы
- Утвердите спецификацию и план
- Наблюдайте за автономной работой субагентов
- Результат: работающий проект с тестами
