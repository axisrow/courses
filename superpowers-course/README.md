# Superpowers Course

Курс по [Superpowers](https://github.com/obra/superpowers) — agentic skills framework для coding agents.

## Структура

```
ru/   — Русский
en/   — English
zh/   — 中文

Каждый язык:
  level-1-beginner/      — 5 уроков (основы)
  level-2-intermediate/  — 5 уроков (продвинутый)
  level-3-advanced/      — 5 уроков (экспертный)
```

## Уровни

### Level 1 — Beginner
1. Что такое Superpowers
2. Установка (Claude Code / Cursor / Codex / OpenCode)
3. Как работает: спецификация → план → субагенты
4. Первый проект
5. TDD-подход

### Level 2 — Intermediate
1. Subagent-driven development
2. Спецификации и планы
3. Навыки (Skills)
4. Настройка под проект
5. Интеграция с разными IDE

### Level 3 — Advanced
1. Архитектура системы
2. Создание кастомных навыков
3. Advanced workflows
4. Отладка и оптимизация
5. Контрибьюция в проект
