# 用户认证

## 什么是认证？

认证让用户可以创建账户并登录你的应用。这样每个用户都能看到自己的数据和设置。

## 添加认证

Bolt.new 内置用户管理功能。只需请求：

> "为应用添加用户注册和登录功能。用户通过邮箱和密码注册。登录后看到个人仪表盘。"

## Bolt 会创建什么

- **注册页面**——新用户创建账号
- **登录页面**——现有用户登录
- **密码安全**——密码自动加密
- **会话管理**——用户保持登录状态
- **受保护页面**——只有登录用户才能看到

## 社交登录

你也可以要求社交登录：

> "在注册页面添加 Google 和 GitHub 登录选项"

用户可以用现有账号登录，无需新密码。

## 用户角色

对于更复杂的应用，可以添加角色：

> "创建两个用户角色：'管理员'和'普通用户'。管理员可以查看所有数据并管理用户。普通用户只能看到自己的数据。"

## 常见认证功能

- 邮箱验证
- 密码重置（"忘记密码？"流程）
- 个人资料编辑
- 账户删除
- 记住我/保持登录

## 安全提示

- 始终使用 Bolt 内置认证——不要自己造轮子
- 要求邮箱验证以防止假账号
- 添加密码强度要求

## 练习
