# SEO и оптимизация производительности

## Зачем нужен SEO

SEO (поисковая оптимизация) помогает людям находить ваш сайт через Google и другие поисковики. Без него ваше приложение может остаться невидимым.

## Встроенный SEO в Bolt

Bolt.new включает функции SEO-оптимизации. Усильте их:

> «Оптимизируй сайт для SEO. Добавь meta-заголовки и описания для каждой страницы, используй семантические заголовки HTML, добавь alt-текст ко всем изображениям и создай карту сайта.»

## Ключевые элементы SEO

### Заголовки и описания страниц
> «Установи заголовок главной: "Лучший итальянский ресторан в Москве — Белла Виста", описание: "Аутентичная итальянская кухня в центре Москвы. Забронируйте столик сегодня."»

### Оптимизация изображений
> «Сожми все изображения для быстрой загрузки и добавь описательный alt-текст»

### Структура URL
> «Используй понятные URL: /menu, /about, /contact вместо /page1, /page2»

### Мобильная версия
Google отдаёт приоритет мобильным сайтам. Всегда проверяйте адаптивный дизайн.

## Оптимизация производительности

Скорость важна и для пользователей, и для поисковых рейтингов.

> «Оптимизируй для быстрой загрузки: ленивая загрузка изображений, минификация кода, индикаторы загрузки»

### Ключевые области:
- **Размер изображений** — сжатие и ресайз
- **Разделение кода** — загружать только нужное для страницы
- **Кэширование** — хранить часто используемые данные локально
- **Лёгкие анимации** — плавные, но не тяжёлые

## Чек-лист SEO

- ✅ Уникальный заголовок для каждой страницы
- ✅ Meta-описания на всех страницах
- ✅ Alt-текст у всех изображений
- ✅ Адаптивный дизайн
- ✅ Быстрая загрузка
- ✅ Чистые URL
- ✅ Правильная иерархия заголовков (H1, H2, H3)

## Практическое задание

Возьмите проект и попросите Bolt провести полную SEO-оптимизацию. Проверьте каждый пункт чек-листа.
