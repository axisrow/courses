# Свой домен и хостинг

## Профессиональное присутствие в сети

Свой домен (например, `www.вашбизнес.com`) делает приложение профессиональным и вызывает доверие. Bolt.new поддерживает свои домены на платных планах.

## Настройка своего домена

### Шаг 1: Купите домен
Приобретите домен у регистратора:
- Namecheap
- Google Domains
- reg.ru
- Cloudflare

Выберите короткое, запоминающееся и релевантное бизнесу имя.

### Шаг 2: Подключите к Bolt
В настройках проекта Bolt добавьте свой домен. Bolt выдаст DNS-записи для настройки.

### Шаг 3: Настройте DNS
У регистратора домена добавьте DNS-записи от Bolt (обычно CNAME или A-запись). Активация может занять до 24 часов.

### Шаг 4: HTTPS
Bolt автоматически предоставляет SSL-сертификат (замочек в браузере), так что сайт защищён с первого дня.

## Хостинг Bolt Cloud

Bolt Cloud обеспечивает хостинг корпоративного уровня:

- **99.9% аптайм** — сайт почти всегда доступен
- **Глобальный CDN** — быстрая загрузка по всему миру
- **Автомасштабирование** — справляется с пиками трафика
- **Встроенная аналитика** — видите количество посетителей

## Аналитика

Со встроенной аналитикой Bolt можно отслеживать:
- Количество посетителей
- Популярные страницы
- Откуда приходят посетители
- Типы устройств (мобильные vs. десктоп)

## Когда стоит перейти на платный план

- Нужен свой домен
- Приложение обслуживает реальных клиентов
- Нужна лучшая производительность
- Хотите аналитику и поддержку

## Практическое задание

Если у вас есть домен — подключите его к проекту Bolt. Если нет — изучите регистраторов и выберите имя домена для будущего проекта.
