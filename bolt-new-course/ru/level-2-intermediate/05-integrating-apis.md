# Интеграция внешних API

## Что такое API?

API (программный интерфейс) — это способ связи вашего приложения с другими сервисами: показ погоды, приём платежей, отправка писем.

## Как добавить интеграцию

Просто скажите Bolt, что хотите подключить:

> «Добавь виджет погоды, показывающий текущую погоду для любого города»

> «Подключи оплату через Stripe для приёма банковских карт»

> «Добавь форму обратной связи, которая отправляет email при заполнении»

## Популярные интеграции

| Сервис | Что делает |
|--------|-----------|
| Stripe | Приём платежей |
| Google Maps | Карты и местоположения |
| Email-сервисы | Автоматическая отправка писем |
| API погоды | Отображение погоды |
| Соцсети | Кнопки «поделиться», ленты |
| Аналитика | Отслеживание посетителей |

## API-ключи

Некоторые сервисы требуют API-ключ (пароль для вашего приложения). Bolt подскажет, когда он нужен, и поможет его получить.

> «У меня есть ключ Google Maps API: ABC123. Добавь карту с адресом моего бизнеса: ул. Пушкина, 10, Москва.»

## Советы

- **Начинайте с бесплатных API** — у многих сервисов есть бесплатные тарифы
- **Храните ключи в секрете** — никогда не публикуйте их
- **Обрабатывайте ошибки** — попросите Bolt показывать дружелюбное сообщение, если сервис недоступен
- **Тестируйте тщательно** — внешние сервисы могут вести себя неожиданно

## Практическое задание

Добавьте виджет погоды или карту в один из ваших проектов. Чётко опишите, какой сервис хотите и где он должен появиться.
