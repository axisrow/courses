# Работа с базами данных

## Что такое база данных?

База данных — это место, где приложение хранит информацию: профили пользователей, списки товаров, записи в блоге. Думайте о ней как об умной таблице, которую приложение автоматически читает и заполняет.

## Базы данных в Bolt.new

Bolt.new включает встроенную поддержку баз данных. Не нужно настраивать серверы или что-то устанавливать. Просто скажите ИИ, какие данные нужно хранить.

## Пример: Приложение для контактов

Попробуйте такой промпт:

> «Создай приложение для управления контактами. Нужно хранить контакты с полями: имя, email, телефон и компания. Добавь форму для добавления контактов и таблицу для их просмотра. Поддержи редактирование и удаление.»

Bolt автоматически:
- Создаст базу данных для контактов
- Построит форму для добавления
- Построит таблицу для просмотра
- Добавит функции редактирования и удаления

## Как думать о данных

Когда просите Bolt хранить данные, подумайте:

- **Какую информацию** нужно сохранять? (поля)
- **Как вещи связаны?** (например, у пользователя много заказов)
- **Какие действия** нужны? (создать, прочитать, изменить, удалить)

## Типичные применения

| Применение | Что хранится |
|-----------|-------------|
| Блог | Посты, авторы, комментарии |
| Магазин | Товары, заказы, клиенты |
| Задачник | Задачи, категории, пользователи |
| Система записи | Встречи, клиенты, услуги |

## Советы

- **Начните просто** — пара полей, потом добавите ещё
- **Думайте, что меняется** — данные от пользователей нужна база, статический текст — нет

## Практическое задание

Попросите Bolt создать приложение-библиотеку, где можно добавлять книги (название, автор, год, рейтинг) и просматривать список.
