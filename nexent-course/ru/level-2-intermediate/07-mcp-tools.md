# Экосистема MCP-инструментов

## Что такое MCP?

MCP (Model Context Protocol) — это стандарт для создания инструментов, которые могут использовать AI-агенты. Думайте о нём как об универсальной системе розеток: любой инструмент, созданный по спецификации MCP, работает с любым агентом Nexent.

## Зачем нужны инструменты

Без инструментов агент может только генерировать текст из своих обучающих данных. С инструментами он может:

- Искать в интернете.
- Читать и обрабатывать файлы.
- Обращаться к базам данных.
- Отправлять письма и сообщения.
- Взаимодействовать с внешними сервисами.

## Встроенные инструменты

| Инструмент | Что делает |
|-----------|-----------|
| Веб-поиск | Ищет информацию через 5+ поисковых провайдеров |
| Поиск по базе знаний | Ищет ответы в загруженных документах |
| Чтение файлов | Читает загруженные файлы во время разговора |
| Генерация изображений | Создаёт картинки по текстовому описанию |

## Подключение внешних MCP-инструментов

1. Перейдите в **Настройки → Инструменты**.
2. Нажмите **Добавить удалённый MCP-инструмент**.
3. Введите URL сервера инструмента и конфигурацию.
4. Инструмент становится доступен для всех агентов.

## Использование инструментов в агентах

При создании агента вы выбираете, какие инструменты ему доступны. Агент сам решает, когда какой инструмент применить, исходя из контекста разговора.

## Сообщество MCP

Nexent интегрируется с широкой экосистемой MCP. Инструменты от сообщества можно найти на [странице MCP-экосистемы Nexent](https://modelengine-group.github.io/nexent/en/mcp-ecosystem/overview.html).

## Практическое задание

1. Проверьте, какие инструменты доступны в вашей установке.
2. Создайте агента с доступом к веб-поиску.
3. Задайте вопрос, требующий актуальной информации.
4. Посмотрите, как агент использует инструмент поиска.
