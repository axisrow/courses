# Основы базы знаний

## Что такое база знаний?

База знаний — это коллекция ваших документов, по которой Nexent может искать информацию при ответе на вопросы. Вместо того чтобы полагаться только на общие знания AI-модели, агент заглядывает в ваши файлы и даёт точные ответы.

## Поддерживаемые форматы

Nexent поддерживает более 20 форматов:

- PDF-документы
- Word-файлы (.docx)
- Таблицы Excel
- Презентации PowerPoint
- Текстовые файлы и Markdown
- Изображения (с OCR — распознаванием текста на картинках)

## Создание базы знаний

1. В интерфейсе Nexent перейдите в раздел **База знаний** (в боковой панели).
2. Нажмите **Создать новую базу знаний**.
3. Дайте ей название — например, «Политики компании».
4. Загрузите файлы, перетащив их в область загрузки.

## Как это работает

При загрузке файла:

1. Nexent **обрабатывает** документ — извлекает текст, таблицы и изображения.
2. Текст **разбивается на фрагменты** и сохраняется в поисковом индексе.
3. Генерируется **автоматическая сводка**, чтобы агент знал, что содержится в каждой базе.

При вопросе агент ищет по базе, находит нужные фрагменты и составляет ответ — с цитатами, указывающими на источник.

## Личные и общие базы

- **Личная** — видите только вы.
- **Общая** — доступна всем пользователям в рабочем пространстве.

## Лучшие практики

- **Разделяйте по темам.** Одна база на одну тему работает лучше одной гигантской коллекции.
- **Обновляйте регулярно.** Удаляйте устаревшие документы, добавляйте новые.
- **Используйте понятные имена файлов.** Это помогает и вам, и агенту.

## Попробуйте сейчас

1. Создайте базу знаний «Тест».
2. Загрузите короткий PDF или текстовый файл.
3. Задайте вопрос по содержимому документа в чате.
4. Проверьте, что ответ содержит цитату из файла.

Далее — встроенные агенты Nexent.
