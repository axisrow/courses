# 模型配置

## 为什么模型配置很重要

Nexent 支持多种 AI 模型。选择合适的模型会影响回答质量、速度和成本。

## 支持的模型提供商

Nexent 支持以下提供商：

- **OpenAI**（GPT-4、GPT-3.5 等）
- **SiliconFlow**
- **ModelEngine**
- **其他兼容 OpenAI 接口的服务**

## 添加模型提供商

1. 进入 **设置 → 模型提供商**。
2. 点击 **添加提供商**。
3. 选择提供商类型。
4. 输入 API 密钥和接口地址。
5. 测试连接。
6. 保存。

## 不同任务选择不同模型

| 任务 | 推荐模型 |
|------|---------|
| 聊天对话 | GPT-4 或同等模型（质量最好） |
| 文档摘要 | GPT-3.5 或同等模型（速度和质量平衡） |
| 嵌入向量 | 文本嵌入模型（用于知识库搜索） |
| 图片理解 | 支持视觉的模型（GPT-4V 或同等） |

## 模型健康监控

Nexent 内置模型健康检查功能，监控以下指标：

- 响应时间
- 错误率
- 可用性

如果某个模型不可用，Nexent 会提醒您切换到备用模型。

## 成本管理建议

- 简单任务使用小型、便宜的模型。
- 复杂推理任务使用强大的模型。
- 在提供商的控制台监控 API 用量。

## 练习

1. 检查您的 Nexent 中配置了哪些模型提供商。
2. 添加一个新的提供商或模型。
3. 为不同任务分配不同模型。
4. 对比不同模型的回答质量。
