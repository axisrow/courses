# 多智能体协作

## 什么是多智能体协作？

有时一个智能体不够处理复杂任务。Nexent 支持**多智能体协作**，由管理者智能体协调多个专业智能体共同完成工作。

## 工作原理

1. **管理者智能体**接收你的请求。
2. 将任务分解为子任务。
3. 每个子任务分配给具有相应技能的**受管智能体**。
4. 受管智能体完成各自的部分并汇报。
5. 管理者汇总最终回答。

## 示例场景

你问：*"分析我们第三季度的销售数据，与行业趋势对比，并写一份执行摘要。"*

管理者可能分配：

- **数据智能体** — 读取销售表格。
- **研究智能体** — 在网上搜索行业趋势。
- **写作智能体** — 撰写执行摘要。

## 设置多智能体工作流

1. 创建各专业智能体，描述要聚焦。
2. 创建管理者智能体，将其与专业智能体关联。
3. 在管理者配置中定义可调用的智能体。
4. 与管理者对话——它会自动协调其他智能体。

## 优势

- **更好的结果** — 每个智能体专注于自己最擅长的领域。
- **更快处理** — 子任务可以并行执行。
- **模块化设计** — 更新一个智能体不影响其他的。

## 监控协作

在聊天界面中，你可以看到哪些智能体参与了回答。每个智能体的贡献都会被追踪和显示。

## 练习

1. 创建两个专业智能体（如研究员和写手）。
2. 创建一个协调它们的管理者智能体。
3. 给管理者一个需要两种技能的复杂任务。
4. 观察工作是如何分配的。
