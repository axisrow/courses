# 通过提示词创建智能体

## 核心理念

Nexent 最强大的功能是**智能提示词生成**。你用自然语言描述需求，Nexent 自动：

- 生成智能体的系统提示词。
- 选择合适的工具。
- 规划最佳行动路径。

## 创建你的第一个自定义智能体

1. 在侧边栏进入**智能体**部分。
2. 点击**创建新智能体**。
3. 在描述字段中输入你的需求：

> "一个帮助 HR 团队筛选简历的助手。它应该阅读上传的 PDF，将候选人与职位要求进行比较，并对他们进行排名。"

4. 点击**生成**。Nexent 会创建完整的智能体配置。
5. 检查生成的提示词和工具，必要时调整。
6. 保存并开始与新智能体对话。

## 写好描述的技巧

- **说明角色。** "你是一个旅行规划师……"
- **定义受众。** "……为有小孩的家庭服务。"
- **列出关键任务。** "推荐目的地、估算预算、制定每日行程。"
- **提及约束条件。** "始终推荐 2000 美元以下的经济选项。"

## Nexent 如何生成提示词

1. AI 模型分析你的描述。
2. 从 MCP 生态中识别相关工具。
3. 生成结构化的系统提示词。
4. 智能体配置完毕，可以使用。

## 编辑智能体提示词

生成后你可以手动修改：

- **系统提示词** — 核心指令。
- **工具选择** — 添加或移除工具。
- **知识库连接** — 选择智能体可以访问的知识库。

## 智能体版本管理

Nexent 跟踪智能体版本。每次保存都会创建新版本，你可以：

- 查看版本历史。
- 回滚到之前的版本。
- 对比不同版本。

## 练习

创建一个这样的智能体：

> "一个烹饪助手，根据我家里现有的食材推荐菜谱。考虑饮食限制，并为缺少的食材推荐替代品。"

告诉智能体你冰箱里有什么，测试一下吧！
