# MCP 工具生态

## 什么是 MCP？

MCP（Model Context Protocol，模型上下文协议）是为 AI 智能体构建工具的标准。可以把它想象成通用插头系统：任何按照 MCP 规范构建的工具都能与 Nexent 智能体配合使用。

## 为什么工具很重要

没有工具，智能体只能根据训练数据生成文本。有了工具，它可以：

- 搜索互联网。
- 读取和处理文件。
- 查询数据库。
- 发送邮件或消息。
- 与外部服务交互。

## 内置工具

| 工具 | 功能 |
|------|------|
| 网络搜索 | 通过 5 个以上搜索引擎查找最新信息 |
| 知识库搜索 | 在上传的文档中查找信息 |
| 文件阅读器 | 在对话中读取上传的文件 |
| 图片生成 | 根据文字描述创建图片 |

## 添加远程 MCP 工具

你可以连接托管在远程服务器上的外部 MCP 工具：

1. 进入**设置 → 工具**。
2. 点击**添加远程 MCP 工具**。
3. 输入工具的服务器 URL 和配置。
4. 该工具将对所有智能体可用。

## 在智能体中使用工具

创建或编辑智能体时，你可以选择它能访问哪些工具。智能体会根据对话内容自动决定何时使用哪个工具。

## MCP 社区

Nexent 与更广泛的 MCP 生态集成。你可以在 [Nexent MCP 生态页面](https://modelengine-group.github.io/nexent/en/mcp-ecosystem/overview.html) 找到社区构建的工具。

## 练习

1. 查看你的 Nexent 安装中有哪些可用工具。
2. 创建一个有网络搜索权限的智能体。
3. 问一个需要当前信息的问题（如今天的新闻）。
4. 观察智能体如何使用网络搜索工具来回答。
