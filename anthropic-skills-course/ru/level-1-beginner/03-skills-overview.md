# Обзор навыков

## Введение

Репозиторий Anthropic содержит навыки трёх основных категорий: творческие, технические и корпоративные.

## Creative & Design (Творческие)

### algorithmic-art
Создание алгоритмического искусства с помощью p5.js. Использует сидированную случайность и интерактивные параметры.

### canvas-design
Создание визуального искусства в форматах .png и .pdf. Постеры, дизайн и другие статичные работы.

### frontend-design
Создание продакшен-качества фронтенд-интерфейсов с высоким уровнем дизайна.

### theme-factory
Набор из 10 предустановленных тем с цветами и шрифтами для стилизации артефактов.

### slack-gif-creator
Создание анимированных GIF, оптимизированных для Slack.

## Technical (Технические)

### webapp-testing
Тестирование веб-приложений с помощью Playwright. Скриншоты, логи, проверка UI.

### mcp-builder
Руководство по созданию MCP-серверов для интеграции LLM с внешними сервисами.

### web-artifacts-builder
Создание сложных HTML-артефактов с React, Tailwind CSS и shadcn/ui.

### skill-creator
Мета-навык: помогает создавать новые навыки.

## Enterprise & Communication (Корпоративные)

### brand-guidelines
Применение фирменных цветов и типографики Anthropic к артефактам.

### internal-comms
Шаблоны для внутренних коммуникаций: отчёты, рассылки, FAQ.

### doc-coauthoring
Совместное написание документации через структурированный процесс.

## Итоги

- Навыки делятся на три категории: творческие, технические, корпоративные
- Каждый навык решает конкретную задачу
- Навыки можно комбинировать и использовать как образцы для своих
