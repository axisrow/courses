# Что такое Skills

## Введение

Skills (навыки) — это папки с инструкциями, скриптами и ресурсами, которые Claude загружает динамически для улучшения работы над специализированными задачами. Навыки учат Claude выполнять конкретные задачи повторяемым способом.

## Зачем нужны навыки

Без навыков Claude работает на основе общих знаний. Навыки позволяют:

- **Стандартизировать** — задать единый процесс для повторяющихся задач
- **Специализировать** — научить Claude работать с конкретными инструментами и форматами
- **Масштабировать** — один навык может использовать вся команда

## Примеры использования

- Создание документов по брендбуку компании
- Анализ данных по специфическим процессам организации
- Автоматизация личных задач
- Генерация алгоритмического искусства
- Тестирование веб-приложений

## Как устроен навык

Каждый навык — это папка, содержащая минимум один файл `SKILL.md`. Этот файл включает:

1. **Метаданные** (frontmatter) — имя и описание
2. **Инструкции** — что Claude должен делать
3. **Примеры** — образцы использования
4. **Правила** — ограничения и рекомендации

```
my-skill/
├── SKILL.md          # Основной файл навыка
├── scripts/          # Вспомогательные скрипты (опционально)
├── examples/         # Примеры (опционально)
└── LICENSE.txt       # Лицензия (опционально)
```

## Итоги

- Skills — это система расширения возможностей Claude
- Навыки хранятся в папках с файлом `SKILL.md`
- Они позволяют стандартизировать и улучшать работу Claude
- Навыки можно создавать самостоятельно или использовать готовые
