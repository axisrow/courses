# Навыки для документов

## Введение

Anthropic создала навыки для работы с основными форматами документов. Эти навыки обеспечивают возможности Claude по созданию и редактированию файлов.

## DOCX — Word-документы

Навык `docx` позволяет:
- Создавать документы Word
- Редактировать существующие документы
- Добавлять комментарии и отслеживать изменения
- Принимать/отклонять правки (redlining)

Структура навыка:
```
docx/
├── SKILL.md
├── scripts/
│   ├── comment.py          # Работа с комментариями
│   ├── accept_changes.py   # Принятие изменений
│   └── office/
│       ├── validate.py     # Валидация документов
│       ├── pack.py         # Сборка .docx
│       ├── unpack.py       # Распаковка .docx
│       └── schemas/        # XML-схемы Office
└── LICENSE.txt
```

## PDF — PDF-документы

Навык `pdf` позволяет:
- Создавать PDF-документы
- Извлекать данные из PDF
- Работать с формами

## PPTX — Презентации

Навык `pptx` позволяет:
- Создавать презентации PowerPoint
- Добавлять слайды, текст, изображения
- Применять темы и стили

## XLSX — Таблицы

Навык `xlsx` позволяет:
- Создавать таблицы Excel
- Работать с формулами
- Строить графики

## Особенности документных навыков

Эти навыки отличаются от обычных:
1. **Содержат скрипты** для обработки бинарных форматов
2. **Включают XML-схемы** для валидации
3. **Source-available** — доступны для изучения, но не Open Source

## Пример использования

```
Пользователь: Создай Word-документ с отчётом о продажах за Q4

Claude (с навыком docx): Создаёт .docx файл с правильным 
форматированием, таблицами и стилями
```

## Итоги

- Anthropic предоставляет навыки для DOCX, PDF, PPTX, XLSX
- Документные навыки используют скрипты для обработки файлов
- Они обеспечивают функцию создания файлов в Claude
- Можно изучить их как образец сложных навыков
