# Публикация навыков

## Введение

После создания навыка его можно опубликовать для использования другими.

## Способы публикации

### 1. GitHub-репозиторий

Самый простой способ — создать публичный репозиторий:

```
my-skills-repo/
├── README.md
├── skill-one/
│   └── SKILL.md
├── skill-two/
│   ├── SKILL.md
│   └── scripts/
└── LICENSE
```

### 2. Claude Code Plugin Marketplace

Опубликуйте как плагин Claude Code:

1. Создайте репозиторий с навыками
2. Оформите как маркетплейс-пакет
3. Пользователи установят через `/plugin marketplace add`

### 3. Claude.ai

Загрузите навык в проект Claude.ai для командного использования.

## Подготовка к публикации

### Чек-лист

- [ ] `SKILL.md` содержит полные метаданные
- [ ] Описание на английском языке
- [ ] Добавлен `LICENSE.txt`
- [ ] Удалены чувствительные данные (API-ключи, пароли)
- [ ] Добавлен `README.md` с инструкциями
- [ ] Все скрипты протестированы
- [ ] Нет захардкоженных путей

### Лицензирование

Выберите лицензию:
- **Apache 2.0** — используется Anthropic для большинства навыков
- **MIT** — простая и популярная
- **Proprietary** — для внутренних навыков

## Партнёрские навыки

Anthropic публикует навыки от партнёров. Например:
- **Notion** — [Notion Skills for Claude](https://www.notion.so/notiondevs/Notion-Skills-for-Claude)

## Итоги

- Навыки публикуются через GitHub, Claude Code Marketplace или Claude.ai
- Подготовьте навык: лицензия, документация, тестирование
- Удалите чувствительные данные перед публикацией
- Партнёры могут публиковать навыки через Anthropic
