# Agent Skills — открытый стандарт

## Введение

Agent Skills — это открытый стандарт, определяющий, как AI-агенты могут использовать навыки. Спецификация доступна на [agentskills.io](https://agentskills.io).

## Зачем нужен стандарт

Без стандарта каждый провайдер AI создаёт свой формат навыков. Стандарт обеспечивает:

- **Переносимость** — навык работает с разными AI-системами
- **Совместимость** — навыки от разных авторов работают вместе
- **Экосистему** — общий формат стимулирует создание навыков

## Структура спецификации

Спецификация определяет:

1. **Формат файла** — SKILL.md с YAML frontmatter
2. **Метаданные** — обязательные и опциональные поля
3. **Структуру папки** — размещение скриптов, ассетов
4. **Обнаружение** — как агент находит и активирует навыки
5. **Выполнение** — как агент следует инструкциям

## Расположение спецификации

В репозитории Anthropic спецификация находится в `spec/agent-skills-spec.md`, но актуальная версия — на сайте [agentskills.io/specification](https://agentskills.io/specification).

## Связь со Skills в Claude

Skills в Claude — это реализация стандарта Agent Skills от Anthropic. Стандарт шире — он предназначен для любых AI-агентов, не только Claude.

## Итоги

- Agent Skills — открытый стандарт для AI-агентов
- Обеспечивает переносимость и совместимость навыков
- Актуальная спецификация на agentskills.io
- Skills в Claude — одна из реализаций стандарта
