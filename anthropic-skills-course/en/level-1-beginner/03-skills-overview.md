# Skills Overview

## Introduction

The Anthropic repository contains skills in three main categories: creative, technical, and enterprise.

## Creative & Design

- **algorithmic-art** — Generative art with p5.js
- **canvas-design** — Visual art in .png and .pdf
- **frontend-design** — Production-grade frontend interfaces
- **theme-factory** — 10 pre-set themes for styling artifacts
- **slack-gif-creator** — Animated GIFs optimized for Slack

## Technical

- **webapp-testing** — Web app testing with Playwright
- **mcp-builder** — Guide for creating MCP servers
- **web-artifacts-builder** — Complex HTML artifacts with React/Tailwind/shadcn
- **skill-creator** — Meta-skill for creating new skills

## Enterprise & Communication

- **brand-guidelines** — Apply brand colors and typography
- **internal-comms** — Templates for internal communications
- **doc-coauthoring** — Structured documentation co-authoring

## Summary

- Skills fall into three categories: creative, technical, enterprise
- Each skill solves a specific task
- Skills can be combined and used as templates for your own
