# Publishing Skills

## Introduction

After creating a skill, you can publish it for others to use.

## Publishing Methods

1. **GitHub repository** — create a public repo with your skills
2. **Claude Code Plugin Marketplace** — publish as a plugin package
3. **Claude.ai** — upload for team use in project settings

## Pre-publish Checklist

- [ ] Complete metadata in SKILL.md
- [ ] Description in English
- [ ] LICENSE.txt included
- [ ] No sensitive data (API keys, passwords)
- [ ] README.md with instructions
- [ ] All scripts tested
- [ ] No hardcoded paths

## Licensing

- **Apache 2.0** — used by Anthropic for most skills
- **MIT** — simple and popular
- **Proprietary** — for internal skills

## Summary

- Publish via GitHub, Claude Code Marketplace, or Claude.ai
- Prepare: license, documentation, testing
- Remove sensitive data before publishing
