# Complex Skills with Scripts

## Introduction

Some tasks require scripts, templates, and resources beyond just instructions.

## Complex Skill Structure

```
webapp-testing/
├── SKILL.md
├── scripts/
│   └── with_server.py
├── examples/
│   ├── static_html_automation.py
│   ├── element_discovery.py
│   └── console_logging.py
└── LICENSE.txt
```

## Scripts

Scripts extend skill capabilities for operations impossible in plain text (binary processing, validation, automation).

## Assets and Resources

- **Reference docs**: `mcp-builder/reference/` contains detailed documentation
- **Templates and archives**: `web-artifacts-builder/scripts/` includes component archives and shell scripts
- **XML schemas**: `docx/scripts/office/schemas/` for document validation

## Guidelines

1. **Scripts** — for binary data and complex logic
2. **Reference** — for documentation too large for SKILL.md
3. **Templates** — for standardized output
4. **Schemas** — for result validation

## Summary

- Complex skills include scripts, assets, and reference materials
- Scripts extend capabilities beyond text instructions
- Reference materials go in the `reference/` folder
