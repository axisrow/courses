# Contributing to Anthropic

## Introduction

The `anthropics/skills` repository is open for contributions.

## Process

### Step 1: Fork

```bash
git clone https://github.com/YOUR-USERNAME/skills.git
```

### Step 2: Create Your Skill

Follow the template format in `template/SKILL.md`. Add `LICENSE.txt` (Apache 2.0).

### Step 3: Pull Request

```bash
git checkout -b add-my-skill
git add . && git commit -m "Add my-new-skill"
git push origin add-my-skill
```

Include in your PR: what the skill does, usage examples, test results.

## Improving Existing Skills

You can also fix instructions, add examples, improve descriptions, or update scripts.

## Summary

- The Anthropic repo is open for contributions
- Follow standard Fork → Branch → PR workflow
- Use the template format
- You can also improve existing skills
