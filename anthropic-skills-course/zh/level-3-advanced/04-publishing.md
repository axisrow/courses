# 发布技能

## 简介

创建技能后，可以发布供他人使用。

## 发布方式

1. **GitHub仓库** — 创建公开仓库
2. **Claude Code插件市场** — 作为插件包发布
3. **Claude.ai** — 通过项目设置上传供团队使用

## 发布前检查清单

- [ ] SKILL.md中的完整元数据
- [ ] 英文描述
- [ ] 包含LICENSE.txt
- [ ] 无敏感数据（API密钥、密码）
- [ ] 包含README.md说明
- [ ] 所有脚本已测试
- [ ] 无硬编码路径

## 许可证

- **Apache 2.0** — Anthropic大多数技能使用
- **MIT** — 简单流行
- **专有** — 用于内部技能

## 总结

- 通过GitHub、Claude Code市场或Claude.ai发布
- 准备：许可证、文档、测试
- 发布前删除敏感数据
