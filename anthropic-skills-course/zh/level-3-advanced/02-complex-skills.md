# 包含脚本的复杂技能

## 简介

某些任务不仅需要指令，还需要脚本、模板和资源。

## 复杂技能结构

```
webapp-testing/
├── SKILL.md
├── scripts/
│   └── with_server.py
├── examples/
│   ├── static_html_automation.py
│   └── element_discovery.py
└── LICENSE.txt
```

## 脚本

脚本扩展技能能力，处理纯文本无法完成的操作（二进制处理、验证、自动化）。

## 资源类型

- **参考文档**：`mcp-builder/reference/`包含详细文档
- **模板和归档**：`web-artifacts-builder/scripts/`包含组件归档和shell脚本
- **XML模式**：`docx/scripts/office/schemas/`用于文档验证

## 总结

- 复杂技能包含脚本、资源和参考材料
- 脚本扩展文本指令之外的能力
- 参考材料放在`reference/`文件夹中
