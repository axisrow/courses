# 什么是Skills

## 简介

Skills（技能）是包含指令、脚本和资源的文件夹，Claude可以动态加载它们来提升专业任务的表现。技能教会Claude以可重复的方式完成特定任务。

## 为什么需要技能

没有技能时，Claude基于通用知识工作。技能允许你：

- **标准化** — 为重复任务定义统一流程
- **专业化** — 教Claude使用特定工具和格式
- **规模化** — 一个技能可供整个团队使用

## 使用场景

- 按照公司品牌指南创建文档
- 使用组织特定工作流分析数据
- 自动化个人任务
- 生成算法艺术
- 测试Web应用

## 技能结构

每个技能是一个至少包含`SKILL.md`文件的文件夹：

```
my-skill/
├── SKILL.md          # 主技能文件
├── scripts/          # 辅助脚本（可选）
├── examples/         # 示例（可选）
└── LICENSE.txt       # 许可证（可选）
```

## 总结

- Skills是扩展Claude能力的系统
- 技能存放在包含`SKILL.md`文件的文件夹中
- 可以创建自己的技能或使用现成的
