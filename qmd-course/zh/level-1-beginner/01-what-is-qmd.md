# 什么是 QMD？

## 问题

你有笔记、会议记录、文档和各种知识散落在许多文件夹中。找到你需要的东西很费劲。普通的文件搜索只能匹配精确的词语，无法理解你的*意图*。

## 解决方案

**QMD（Query Markup Documents）**是一个完全运行在你电脑上的个人搜索引擎。可以把它想象成你自己的私人 Google，但只搜索你的文件。

QMD 读取你的 markdown 文件，提供三种搜索方式：

1. **关键词搜索** — 查找精确的词语（如传统搜索）
2. **语义搜索** — 理解含义（即使用词不同也能找到结果）
3. **混合搜索** — 结合两种方法，获得最佳结果

## 核心特点

- **本地运行** — 数据不会离开你的电脑
- **支持 markdown 文件** — 笔记、文档、会议记录
- **理解含义** — 不仅仅是关键词匹配
- **支持 AI 代理** — 专为 Claude 等工具设计

## 适合谁？

QMD 非常适合：

- 用 markdown 写大量笔记的人
- 需要搜索会议记录的人
- 维护项目文档的人
- 希望 AI 助手能访问知识库的人
