# Обслуживание индекса

## Обновление индекса

```sh
# Переиндексация всех коллекций
qmd update

# С предварительным git pull
qmd update --pull
```

## Проверка статуса

```sh
qmd status
```

## Очистка

```sh
qmd cleanup
```

Удаляет осиротевшие данные. Безопасно запускать в любое время.

## Именованные индексы

```sh
qmd --index work collection add ~/work/docs --name work-docs
qmd --index work embed
qmd --index work search "отчёт за квартал"
```

## Хранение данных

- **Индекс**: `~/.cache/qmd/index.sqlite`
- **ИИ-модели**: `~/.cache/qmd/models/`

Для полного сброса удалите `~/.cache/qmd/`.

## Советы

1. **Запускайте `qmd update` ежедневно** — в начале рабочего дня
2. **Используйте `--pull`** — если документы в git-репозиториях
3. **Запускайте `qmd embed` после крупных изменений**
4. **Проверяйте `qmd status` периодически**
