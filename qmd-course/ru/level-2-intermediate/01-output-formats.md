# Форматы вывода

## Зачем разные форматы?

По умолчанию QMD показывает цветной вывод в терминале. Но для скриптов и ИИ-агентов нужен машиночитаемый формат.

## Доступные форматы

| Флаг | Формат | Для чего |
|------|--------|----------|
| *(по умолчанию)* | Цветной CLI | Чтение в терминале |
| `--json` | JSON | Скрипты и ИИ-агенты |
| `--files` | Список файлов | Пайплайны и интеграции |
| `--csv` | CSV | Электронные таблицы |
| `--md` | Markdown | Документация и отчёты |
| `--xml` | XML | Структурированный обмен данными |

## Примеры

```sh
# JSON для ИИ
qmd search "аутентификация" --json -n 10

# Список файлов
qmd query "обработка ошибок" --all --files --min-score 0.4

# Markdown с полным текстом
qmd search "дизайн API" --md --full

# CSV для таблиц
qmd search "план проекта" --csv
```

## Управление количеством результатов

```sh
# Показать 10 результатов
qmd search "запрос" -n 10

# Все результаты выше порога
qmd search "запрос" --all --min-score 0.3
```

## Отключение цветов

```sh
NO_COLOR=1 qmd search "запрос"
```
