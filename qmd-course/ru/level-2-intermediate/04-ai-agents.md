# Использование QMD с ИИ-агентами

## Зачем подключать QMD к ИИ?

ИИ-ассистенты вроде Claude очень мощные, но не знают о *ваших* файлах. QMD соединяет их — позволяет ИИ искать в вашей базе знаний.

## Интеграция через командную строку

Просто скажите ИИ-агенту использовать команды QMD:

```sh
qmd search "аутентификация" --json -n 10
qmd query "обработка ошибок" --all --files --min-score 0.4
qmd get "docs/api-reference.md" --full
```

## MCP-сервер

MCP (Model Context Protocol) — стандарт подключения ИИ к источникам данных.

### Доступные инструменты MCP

| Инструмент | Описание |
|-----------|----------|
| `qmd_search` | Быстрый поиск по ключевым словам |
| `qmd_vector_search` | Семантический поиск |
| `qmd_deep_search` | Глубокий поиск с расширением и ранжированием |
| `qmd_get` | Получение документа по пути или ID |
| `qmd_multi_get` | Получение нескольких документов |
| `qmd_status` | Статус индекса |

### Настройка для Claude Desktop

Добавьте в конфигурацию:

```json
{
  "mcpServers": {
    "qmd": {
      "command": "qmd",
      "args": ["mcp"]
    }
  }
}
```

### Настройка для Claude Code

```bash
claude marketplace add tobi/qmd
claude plugin add qmd@qmd
```

## Как это работает

После подключения можно спрашивать Claude:

- «Найди в моих заметках всё о бюджете Q4»
- «Найди все протоколы встреч о запуске продукта»
- «Что я писал о миграциях баз данных?»
