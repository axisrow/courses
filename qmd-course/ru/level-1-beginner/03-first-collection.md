# Ваша первая коллекция

## Что такое коллекция?

**Коллекция** — это просто папка с markdown-файлами, в которой QMD будет искать. Можно создать отдельные коллекции для:

- Личных заметок
- Рабочих документов
- Протоколов встреч
- Проектной документации

## Добавление коллекции

Допустим, ваши заметки лежат в `~/notes`:

```sh
qmd collection add ~/notes --name notes
```

Готово! QMD просканирует папку и проиндексирует все markdown-файлы.

## Добавление нескольких коллекций

```sh
qmd collection add ~/Documents/meetings --name meetings
qmd collection add ~/work/docs --name docs
```

## Пользовательские шаблоны файлов

По умолчанию QMD ищет `**/*.md`. Можно задать свой шаблон:

```sh
qmd collection add ~/docs --name docs --mask "**/*.md"
```

## Список коллекций

```sh
qmd collection list
```

## Удаление коллекции

```sh
qmd collection remove notes
```

Удаляется только запись из индекса QMD — ваши файлы не затрагиваются.

## Переименование коллекции

```sh
qmd collection rename notes my-notes
```

## Просмотр файлов коллекции

```sh
qmd ls notes
qmd ls notes/subfolder
```

## Обновление индекса

Когда вы добавили или изменили файлы:

```sh
qmd update
```

## Практика

1. Создайте папку с несколькими markdown-файлами
2. Добавьте её как коллекцию
3. Выполните `qmd collection list` для проверки
4. Выполните `qmd ls <имя-коллекции>` для просмотра файлов
