# Что такое QMD?

## Проблема

У вас есть заметки, протоколы встреч, документация и знания, разбросанные по множеству папок. Найти нужное — целое дело. Обычный поиск по файлам находит только точные слова и не понимает *смысл* запроса.

## Решение

**QMD (Query Markup Documents)** — это персональная поисковая система, которая работает полностью на вашем компьютере. Представьте себе личный Google, но только для ваших файлов.

QMD читает ваши markdown-файлы и позволяет искать тремя способами:

1. **Поиск по ключевым словам** — находит точные слова (как обычный поиск)
2. **Семантический поиск** — понимает смысл (находит результаты, даже если вы используете другие слова)
3. **Гибридный поиск** — сочетает оба метода для лучших результатов

## Ключевые особенности

- **Работает локально** — ваши данные не покидают компьютер
- **Работает с markdown-файлами** — заметки, документы, протоколы встреч
- **Понимает смысл** — не только совпадение слов
- **Работает с ИИ-агентами** — спроектирован для интеграции с Claude и другими инструментами

## Для кого QMD?

QMD идеально подходит тем, кто:

- Ведёт много заметок в markdown
- Хранит протоколы встреч, которые нужно искать
- Поддерживает документацию для проектов
- Хочет, чтобы ИИ-ассистент имел доступ к базе знаний
