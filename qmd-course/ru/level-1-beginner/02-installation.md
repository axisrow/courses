# Установка QMD

## Что вам понадобится

Перед установкой убедитесь, что у вас есть:

- **Node.js** версии 22 или новее
- **Для macOS**: SQLite через Homebrew (`brew install sqlite`)

## Проверьте версию Node.js

Откройте терминал и введите:

```sh
node --version
```

Вы должны увидеть `v22.0.0` или выше. Если нет, скачайте Node.js с [nodejs.org](https://nodejs.org).

## Установите QMD

Выполните команду:

```sh
npm install -g @tobilu/qmd
```

Или через Bun:

```sh
bun install -g @tobilu/qmd
```

## Проверьте установку

```sh
qmd status
```

Вы увидите информацию об индексе QMD.

## Попробовать без установки

```sh
npx @tobilu/qmd status
```

## ИИ-модели

QMD использует три локальные ИИ-модели, которые скачиваются автоматически при первом запуске:

| Модель | Назначение | Размер |
|--------|-----------|--------|
| EmbeddingGemma | Понимание смысла документов | ~300 МБ |
| Qwen3 Reranker | Сортировка по релевантности | ~640 МБ |
| QMD Query Expansion | Улучшение поисковых запросов | ~1.1 ГБ |

Модели хранятся в `~/.cache/qmd/models/`.
