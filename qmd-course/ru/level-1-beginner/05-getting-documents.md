# Получение документов

## Получение одного документа

Нашли что-то интересное в результатах поиска? Прочитайте полный документ:

```sh
qmd get "notes/meeting-2024-01-15.md"
```

## Использование ID документа

Каждый результат поиска содержит короткий ID, например `#a1b2c3`:

```sh
qmd get "#a1b2c3"
```

## Чтение с определённой строки

```sh
qmd get "notes/meeting.md:50"
```

## Ограничение вывода

```sh
qmd get notes/meeting.md -l 100
qmd get notes/meeting.md:50 -l 100
```

## Получение нескольких документов

```sh
# По шаблону
qmd multi-get "journals/2025-05*.md"

# Списком
qmd multi-get "doc1.md, doc2.md, #abc123"
```

## Полный контент в результатах поиска

```sh
qmd search "API" --full
```

## Нумерация строк

```sh
qmd get notes/meeting.md --line-numbers
```

## Экспорт всех совпадений

```sh
qmd search "API" --all --files --min-score 0.3
```

## Практика

1. Найдите ID документа в результатах поиска
2. Используйте `qmd get "#<id>"` для получения документа
3. Попробуйте `qmd multi-get` с шаблоном
