# 运行模式

## 三种运行方式

### 1. 交互模式（你来控制）
代理在每个步骤后暂停，等待你的输入。

**适合：** 学习、敏感任务、新工作流。

### 2. 自主模式（全自动）
代理自动执行所有步骤，无需停顿。

**适合：** 已验证的工作流、批量处理。

### 3. 持续模式（自动推进）
代理自动进入下一步，但不发送额外提示词。

**适合：** 简单的顺序工作流。

## 运行时切换

- **Shift+Tab** — 切换自主模式
- **P** — 暂停
- **Ctrl+S** — 跳过步骤

## 在工作流中配置

```javascript
export default {
  name: 'My Workflow',
  autonomousMode: 'always',
  steps: [
    resolveStep('agent-1'),
    resolveStep('agent-2', { interactive: true }), // 此步骤暂停
  ],
};
```

## 总结

- 交互 = 你控制每一步
- 自主 = 全自动
- 持续 = 自动推进
- 用快捷键随时切换
- 可以为每个步骤单独配置
