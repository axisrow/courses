# Паттерны оркестрации

## Спектр автоматизации

```
Полный контроль ←──────────────────→ Полная автоматизация
  (вы ведёте)        (микс)              (AI ведёт)
```

## Паттерн 1: Человек в цикле

Каждый шаг требует одобрения. Для критических задач.

```javascript
export default {
  autonomousMode: 'never',
  steps: [
    resolveStep('planner'),
    resolveStep('implementer'),
    resolveStep('reviewer'),
  ],
};
```

## Паттерн 2: Контролируемая автоматизация

Автоматически, но с паузами на ключевых шагах.

```javascript
export default {
  autonomousMode: 'always',
  steps: [
    resolveStep('researcher'),
    resolveStep('implementer'),
    resolveStep('reviewer', { interactive: true }),  // ПАУЗА
    resolveStep('deployer'),
  ],
};
```

## Паттерн 3: Полная автоматизация

Без вмешательства. Для проверенных workflow.

## Паттерн 4: Итеративная доработка

С циклами до достижения качества.

```javascript
steps: [
  resolveStep('implementer'),
  resolveModule('quality-gate', {
    loopSteps: 1,
    loopMaxIterations: 5,
  }),
],
```

## Паттерн 5: Параллельная команда

```javascript
subAgentIds: ['frontend-dev', 'backend-dev', 'qa-engineer'],
```

## Выбор паттерна

| Паттерн | Доверие | Скорость | Контроль |
|---------|---------|----------|----------|
| Человек в цикле | Низкое | Медленно | Максимум |
| Контролируемый | Среднее | Средне | Баланс |
| Полная автоматизация | Высокое | Быстро | Минимум |
| Итеративный | Среднее | Переменная | Качество |
| Параллельная команда | Высокое | Максимум | Распределён |

## Итог

- Выбирайте паттерн по уровню доверия и задаче
- Начинайте с интерактивного, переходите к автономному
- Комбинируйте паттерны внутри workflow
- Циклы — для контроля качества
