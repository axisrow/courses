# Режимы запуска

## Три способа запуска workflow

### 1. Интерактивный режим (вы управляете)
Агент останавливается после каждого шага и ждёт вашего ввода.

**Подходит для:** обучения, важных задач, новых workflow.

### 2. Автономный режим (полностью автоматический)
Агент выполняет все шаги без остановок.

**Подходит для:** проверенных workflow, пакетной обработки.

### 3. Непрерывный режим (автопереход)
Агент переходит к следующему шагу автоматически, но не отправляет дополнительных промптов.

**Подходит для:** простых последовательных workflow.

## Переключение режимов

Вы можете переключаться прямо во время работы:

- **Shift+Tab** — переключить автономный режим
- **P** — пауза
- **Ctrl+S** — пропустить шаг

## Настройка в workflow

```javascript
export default {
  name: 'My Workflow',
  autonomousMode: 'always',
  steps: [
    resolveStep('agent-1'),
    resolveStep('agent-2', { interactive: true }), // Пауза на этом шаге
  ],
};
```

## Итог

- Интерактивный = вы контролируете каждый шаг
- Автономный = полностью автоматический
- Непрерывный = автопереход без промптов
- Переключайтесь горячими клавишами
- Настраивайте режим для каждого шага
