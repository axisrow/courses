# Установка CodeMachine

## Что вам понадобится

Перед установкой убедитесь, что у вас есть:

1. **Node.js** (версия 18 или выше) — программа для запуска JavaScript на компьютере
2. **Терминал** — текстовый интерфейс (Terminal на Mac, PowerShell на Windows)
3. **AI-движок** — хотя бы один из: Claude Code, Codex CLI или Cursor

> **Нет Node.js?** Зайдите на [nodejs.org](https://nodejs.org) и скачайте последнюю версию.

## Пошаговая установка

Откройте терминал и введите:

```bash
npm i -g codemachine
```

Эта команда устанавливает CodeMachine глобально. Флаг `-g` означает «глобально» — вы сможете использовать его из любой папки.

## Проверка установки

```bash
codemachine --version
```

Вы должны увидеть номер версии (например, `0.8.0`).

## Короткая команда

Вместо `codemachine` можно набирать:

```bash
cm
```

Обе команды делают одно и то же.

## Важное правило

CodeMachine нужно запускать внутри **папки проекта**, а не в домашней директории:

```bash
cd ~/my-project
codemachine
```

Или указать директорию:

```bash
codemachine --dir ~/my-project
```

## Итог

- Установка: `npm i -g codemachine`
- Проверка: `codemachine --version`
- Короткая команда: `cm`
- Запускайте только в папке проекта

> **Следующий урок:** Создадим первый проект и познакомимся с интерфейсом.
