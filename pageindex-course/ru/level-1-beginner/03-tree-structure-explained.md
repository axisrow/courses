# Древовидная структура

## Что такое древовидная структура?

Древовидная структура — это способ организации информации слоями, от общего к частному. Вы уже знакомы с такими структурами:

- **Организационная структура компании**: Директор → Отделы → Команды → Сотрудники
- **Файловая система на компьютере**: Папки → Подпапки → Файлы
- **Оглавление книги**: Части → Главы → Разделы → Подразделы

PageIndex создаёт именно такую структуру для любого документа.

## Как выглядит дерево PageIndex

Упрощённый пример дерева для годового отчёта:

```
Годовой отчёт 2023
├── Письмо акционерам (стр. 1-3)
├── Обзор компании (стр. 4-10)
│   ├── Наша миссия (стр. 4-5)
│   ├── Ключевые продукты (стр. 6-8)
│   └── Глобальное присутствие (стр. 9-10)
├── Финансовая отчётность (стр. 11-50)
│   ├── Отчёт о прибылях и убытках (стр. 11-20)
│   ├── Балансовый отчёт (стр. 21-35)
│   └── Отчёт о движении денежных средств (стр. 36-50)
└── Факторы риска (стр. 51-60)
```

## Формат JSON

Внутри PageIndex хранит дерево в формате JSON. Каждый раздел — это **узел** (node), который содержит:

- **title**: Название раздела
- **node_id**: Уникальный идентификатор
- **start_index**: Первая страница раздела
- **end_index**: Последняя страница раздела
- **summary**: Краткое описание содержимого
- **nodes**: Подразделы (дочерние узлы)

```json
{
  "title": "Финансовая отчётность",
  "node_id": "0003",
  "start_index": 11,
  "end_index": 50,
  "summary": "Полная финансовая отчётность за 2023 финансовый год",
  "nodes": [
    {
      "title": "Отчёт о прибылях и убытках",
      "node_id": "0004",
      "start_index": 11,
      "end_index": 20
    }
  ]
}
```

## Почему деревья так эффективны

Деревья позволяют PageIndex:

1. **Начать широко, затем сужать** — как карта: страна → город → улица
2. **Пропускать нерелевантные разделы** — не нужно просматривать все 300 страниц
3. **Сохранять контекст** — каждый узел знает своё место в общей картине

## Итоги

Древовидная структура — основа PageIndex. Она организует документ в иерархию разделов с номерами страниц, позволяя ИИ эффективно навигировать, как это делал бы эксперт с подробным оглавлением.
