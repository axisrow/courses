# Сценарии использования

## Где PageIndex работает лучше всего

PageIndex создан для длинных, структурированных профессиональных документов.

## Финансы и бухгалтерия

**Годовые отчёты, отчётность SEC, раскрытие информации о доходах**

Финансовые аналитики регулярно работают с документами на сотни страниц. PageIndex может:
- Быстро находить конкретные финансовые формы
- Находить примечания и раскрытия по конкретным темам
- Сопоставлять информацию из разных разделов

Система Mafin 2.5 на базе PageIndex достигла **точности 98,7%** на бенчмарке FinanceBench.

## Юриспруденция

**Контракты, нормативные акты, юридические заключения**

PageIndex помогает:
- Понимать иерархию пунктов и подпунктов
- Находить конкретные положения и их исключения
- Навигировать по документам со сложной структурой

## Наука и образование

**Учебники, научные статьи, технические руководства**

- Навигация по учебникам по теме, а не по ключевым словам
- Поиск нужных разделов в длинных отчётах
- Понимание связей между главами

## Здравоохранение

**Клинические рекомендации, нормативные документы**

- Поиск конкретных рекомендаций по лечению
- Навигация по требованиям соответствия
- Поиск разделов в документах на утверждение препаратов

## Как получить доступ к PageIndex

1. **Чат-платформа** — [chat.pageindex.ai](https://chat.pageindex.ai)
2. **API** — для разработчиков
3. **MCP** — для использования с Claude, Cursor и другими ИИ-инструментами
4. **Собственный сервер** — открытый исходный код

## Какие документы подходят лучше всего?

| Тип документа | Пригодность | Почему |
|--------------|-------------|--------|
| Годовые отчёты | ⭐⭐⭐⭐⭐ | Длинные, структурированные, с оглавлением |
| Юридические договоры | ⭐⭐⭐⭐⭐ | Иерархическая структура пунктов |
| Учебники | ⭐⭐⭐⭐⭐ | Чёткая структура глав и разделов |
| Короткие письма | ⭐ | Слишком короткие, структура не нужна |
| Чат-переписки | ⭐ | Нет иерархической структуры |

## Итоги

PageIndex идеален для длинных, структурированных документов в финансах, юриспруденции, науке и здравоохранении. Доступен как чат-платформа, API, MCP-интеграция или решение для самостоятельного развёртывания.
