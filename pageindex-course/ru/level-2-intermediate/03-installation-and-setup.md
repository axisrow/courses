# Установка и настройка

## Что вам понадобится

- **Python 3.8 или новее**
- **Ключ API OpenAI** (PageIndex использует GPT-4o по умолчанию)
- **PDF-документ** для тестирования

## Шаг 1: Скачайте код

```bash
git clone https://github.com/VectifyAI/PageIndex.git
cd PageIndex
```

## Шаг 2: Установите зависимости

```bash
pip3 install --upgrade -r requirements.txt
```

## Шаг 3: Настройте API-ключ

Создайте файл `.env` в папке PageIndex:

```bash
CHATGPT_API_KEY=ваш_ключ_openai
```

Замените `ваш_ключ_openai` на реальный ключ с [platform.openai.com](https://platform.openai.com).

## Шаг 4: Обработайте первый документ

```bash
python3 run_pageindex.py --pdf_path /путь/к/документу.pdf
```

Система:
1. Прочитает PDF
2. Проверит наличие оглавления
3. Построит древовидную структуру
4. Сохранит результат как JSON-файл в папке `./results/`

## Основные параметры настройки

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `--model` | gpt-4o-2024-11-20 | Модель ИИ |
| `--toc-check-pages` | 20 | Сколько страниц проверять для TOC |
| `--max-pages-per-node` | 10 | Макс. страниц до разделения узла |
| `--max-tokens-per-node` | 20000 | Макс. токенов до разделения узла |
| `--if-add-node-summary` | yes | Генерировать резюме разделов |
| `--if-add-node-id` | yes | Добавлять уникальные ID узлам |

## Поддержка Markdown

PageIndex может обрабатывать и Markdown-файлы:

```bash
python3 run_pageindex.py --md_path /путь/к/документу.md
```

Файл должен использовать правильную иерархию заголовков (`#`, `##`, `###`).

## Облачный сервис

Если не хотите ничего устанавливать:

- **Чат-платформа**: [chat.pageindex.ai](https://chat.pageindex.ai)
- **API**: для программной интеграции
- **MCP**: для использования с Claude, Cursor и другими инструментами

## Итоги

Для настройки PageIndex нужен Python, ключ API OpenAI и несколько команд. Можно обрабатывать PDF или Markdown локально, или использовать облачный сервис.
