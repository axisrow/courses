# 贡献与路线图

## Sentient是开源的

Sentient是一个开源项目，这意味着任何人都可以查看它的代码、提出改进建议，甚至直接参与开发。

## 如何贡献

### 不需要编程也能贡献

- **报告问题**：使用中遇到bug？在GitHub上提交Issue
- **改进文档**：发现文档不清楚？提交修改建议
- **分享经验**：写一篇你如何使用Sentient的文章
- **翻译**：帮助把文档翻译成其他语言
- **回答问题**：在社区中帮助其他用户

### 代码贡献

如果你会编程：
1. Fork项目仓库
2. 创建一个新分支
3. 做出你的修改
4. 提交Pull Request
5. 等待代码审核

### 贡献指南

- 遵循项目的代码风格
- 为新功能编写测试
- 更新相关文档
- 保持提交信息清晰

## 项目路线图

Sentient团队的未来计划包括：

### 近期目标
- 更多MCP集成
- 改进语音交互
- 移动端应用优化

### 中期目标
- 多模态支持（图片、视频理解）
- 团队协作功能
- 插件市场

### 长期愿景
- 成为最好的开源个人AI助手
- 支持完全离线运行
- 让每个人都能拥有真正了解自己的AI伙伴

## 社区

- **GitHub**：代码和问题追踪
- **Discord**：实时交流和讨论
- **论坛**：深度讨论和教程分享

## 小结

开源项目的力量来自社区。无论你是用户还是开发者，你的参与都能让Sentient变得更好。从最小的贡献开始——哪怕只是报告一个bug或分享一个使用技巧。
