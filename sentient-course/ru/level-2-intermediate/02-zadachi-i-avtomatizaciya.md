# Задачи и автоматизация

## Больше, чем чат

Sentient умеет выполнять **фоновые задачи** — действия по расписанию, по триггеру или в виде длительных процессов.

## Типы задач

| Тип | Описание | Пример |
|---|---|---|
| **Запланированная** | Выполняется однократно в заданное время. | «Отправь мне сводку писем в 18:00.» |
| **Периодическая** | Выполняется регулярно. | «Каждый понедельник создавай план недели из календаря.» |
| **Триггерная** | Срабатывает при определённом событии. | «Когда придёт письмо от начальника — уведоми в Slack.» |
| **Рой (Swarm)** | Сложная задача, разбитая на подзадачи для нескольких AI-агентов. | «Исследуй конкурентов и создай отчёт.» |

## Создание задачи

1. Откройте страницу **Tasks**.
2. Нажмите **New Task**.
3. Опишите задачу в **Task Composer**.
4. Выберите тип и настройте расписание или триггер.
5. Сохраните. **Celery-воркеры** выполнят задачу.

## История задач

На странице Tasks видны:
- Активные задачи и время следующего запуска.
- Завершённые задачи с результатами.
- Ошибки с деталями.

## Как это работает внутри

1. Задача сохраняется в базе данных.
2. **Celery Beat** (планировщик) отслеживает расписание.
3. **Celery Worker** забирает задачу из очереди Redis.
4. Воркер использует тот же AI + MCP конвейер, что и чат.

## Итог

Задачи превращают Sentient из реактивного чат-бота в проактивный движок автоматизации, работающий круглосуточно.
