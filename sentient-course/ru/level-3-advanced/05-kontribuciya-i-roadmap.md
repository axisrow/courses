# Вклад в проект и дорожная карта

## Зачем участвовать?

Sentient — проект сообщества под лицензией **GNU AGPL**. Вклад помогает сделать персональный AI доступным для всех.

## Как внести вклад

1. Сделайте **Fork** репозитория на GitHub.
2. **Клонируйте** форк.
3. Создайте **ветку** для изменений.
4. Внесите изменения и напишите тесты.
5. Отправьте **Pull Request**.
6. Подпишите **CLA** (см. `CLA.md`).

### Настройка разработки

```bash
# Сервер
cd src/server
pip install -r requirements-dev.txt
pytest

# Клиент
cd src/client
npm install
npm test
```

## Где нужна помощь

- **Новые MCP-интеграции** — Jira, Linear, Spotify и др.
- **Улучшение памяти** — дедупликация, суммаризация, ранжирование.
- **Мобильное приложение** — React Native или Flutter.
- **Многоязычный интерфейс** — пока только английский.
- **Производительность** — оптимизация WebSocket и задач.
- **Документация** — руководства и API-документы.

## Дорожная карта

1. **Сейчас** — Проактивный ассистент с чатом, задачами и интеграциями.
2. **Ближайшее будущее** — Полностью автономные агенты для сложных многодневных процессов.
3. **Долгосрочно** — Персональный сверхинтеллект — AI, глубоко знающий вас и действующий как настоящий партнёр.

## Что вы изучили

За 15 уроков на 3 уровнях вы прошли путь от «Что такое Sentient?» до понимания:
- Архитектуры и установки.
- Чата, голоса и памяти.
- Интеграций, задач и AI-моделей.
- Собственных MCP-серверов, Celery, векторных баз.
- Безопасности, self-hosting и участия в проекте.

**Теперь вы готовы эффективно использовать Sentient, настроить его под свои нужды и вносить вклад в проект. Добро пожаловать в сообщество!**

## Итог

Sentient — больше чем программа. Это движение к персональному AI для каждого. Ваш вклад, большой или маленький, помогает формировать это будущее.
