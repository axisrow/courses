# Анимация с interpolate()

Анимации оживляют ваши видео. Функция `interpolate()` в Remotion — главный инструмент для создания плавного движения.

## Что такое интерполяция?

Интерполяция — это вычисление значений между двумя точками. Например, перемещение объекта с позиции 0 в позицию 100 за 30 кадров.

## Функция interpolate

```tsx
import { interpolate, useCurrentFrame } from "remotion";

export const FadeIn = () => {
  const frame = useCurrentFrame();
  const opacity = interpolate(frame, [0, 30], [0, 1]);

  return (
    <div style={{ opacity, fontSize: 80 }}>
      Привет, мир
    </div>
  );
};
```

Текст плавно появляется за 30 кадров (1 секунда при 30 fps).

## Как это работает

`interpolate(значение, входнойДиапазон, выходнойДиапазон)`

- **значение**: текущий кадр
- **входнойДиапазон**: когда анимация начинается и заканчивается `[0, 30]`
- **выходнойДиапазон**: в какие значения преобразовать `[0, 1]`

## Ограничение значений (Clamping)

По умолчанию значения за пределами диапазона продолжают расти. Чтобы это остановить:

```tsx
const opacity = interpolate(frame, [0, 30], [0, 1], {
  extrapolateRight: "clamp",
});
```

## Популярные анимации

- **Появление**: opacity от 0 до 1
- **Выезд**: translateX от -100 до 0
- **Увеличение**: scale от 0 до 1
- **Вращение**: rotate от 0 до 360

## Итог

`interpolate()` преобразует номера кадров в значения анимации. Это основа всех анимаций в Remotion.
