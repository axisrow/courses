# Тестирование и отладка

Профессиональные видеопроекты требуют стратегий тестирования и отладки.

## Визуальное тестирование в Студии

Remotion Studio — ваш основной инструмент отладки:
- Прокручивайте кадры для проверки анимаций
- Используйте таймлайн для проверки тайминга
- Проверяйте разные разрешения

## Логирование

Добавьте `console.log()` в компоненты — вывод появится в консоли браузера:

```tsx
const frame = useCurrentFrame();
console.log("Текущий кадр:", frame);
```

## Частые проблемы и решения

### Мерцание

**Проблема**: изображения мигают при рендеринге.
**Решение**: используйте `<Img>` вместо `<img>` и `delayRender()` для асинхронных данных.

### Рассинхрон аудио

**Проблема**: звук не совпадает с видео.
**Решение**: проверьте, что аудио начинается с правильного кадра.

### Медленный рендеринг

**Проблема**: рендеринг слишком долгий.
**Решение**:
- Упростите сложный CSS
- Сократите API-вызовы
- Используйте `getStaticFiles()` вместо загрузки

## Снэпшот-тестирование

```tsx
import { renderFrames } from "@remotion/renderer";

await renderFrames({
  composition,
  onFrameUpdate: (frame) => console.log(`Кадр ${frame}`),
  outputDir: "./test-frames",
  imageFormat: "png",
});
```

## Инспектор Студии

Нажмите на любой элемент в Студии, чтобы увидеть его свойства и позицию.

## Советы

- Тестируйте начало, середину и конец видео
- Проверяйте крайние случаи (кадр 0, последний кадр)
- Тестируйте с разными значениями параметров
- Рендерите в низком разрешении перед финальным рендером

## Итог

Используйте Студию для визуальной отладки, console.log для данных и проверяйте крайние случаи.
