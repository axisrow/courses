# Ваша первая композиция

Remotion установлен — давайте разберёмся, как устроено видео.

## Что такое композиция?

Композиция — это чертёж вашего видео. Она задаёт:
- **Ширину** и **Высоту** (в пикселях) — разрешение
- **FPS** (кадры в секунду) — плавность
- **Длительность** (в кадрах) — продолжительность

Например: 1920×1080, 30 fps, 150 кадров = 5-секундное Full HD видео.

## Смотрим код

Откройте `src/Root.tsx` в вашем проекте:

```tsx
import { Composition } from "remotion";
import { MyComposition } from "./MyComposition";

export const RemotionRoot = () => {
  return (
    <Composition
      id="MyVideo"
      component={MyComposition}
      durationInFrames={150}
      fps={30}
      width={1920}
      height={1080}
    />
  );
};
```

## Разбираем каждую часть

| Свойство | Назначение |
|---|---|
| `id` | Название видео |
| `component` | React-компонент, рисующий видео |
| `durationInFrames` | Общее число кадров |
| `fps` | Кадры в секунду |
| `width` / `height` | Разрешение видео |

## Попробуйте сами

1. Измените `durationInFrames` на `300` (теперь 10 секунд)
2. Сохраните файл
3. Предпросмотр обновится автоматически

## Итог

Композиция определяет размер, скорость и длину видео и связана с компонентом, который рисует каждый кадр.
