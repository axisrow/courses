# Рендеринг видео

Вы просматривали видео в браузере. Теперь превратим его в настоящий видеофайл.

## Что такое рендеринг?

Рендеринг — это процесс превращения кода в видеофайл (например, MP4). Remotion делает скриншот каждого кадра и склеивает их вместе.

## Рендеринг из командной строки

В папке проекта выполните:

```bash
npx remotion render MyVideo out/video.mp4
```

- `MyVideo` — ID композиции из `Root.tsx`
- `out/video.mp4` — куда сохранить файл

Это может занять несколько минут.

## Рендеринг из Студии

Можно также нажать кнопку **Render** в Remotion Studio (предпросмотр в браузере). Там можно выбрать формат, качество и другие параметры.

## Форматы

| Формат | Применение |
|---|---|
| MP4 | Самый популярный, работает везде |
| WebM | Меньший размер, хорош для веба |
| GIF | Короткие зацикленные ролики |
| PNG-последовательность | Отдельные кадры для постобработки |

## Советы для ускорения

- Закройте другие программы для освобождения процессора
- Для тестов используйте меньшее разрешение
- Remotion автоматически использует все ядра процессора

## Итог

Рендеринг превращает код в видеофайл. Используйте командную строку или интерфейс Студии.
