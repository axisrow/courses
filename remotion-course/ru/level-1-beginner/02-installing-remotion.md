# Установка Remotion

Прежде чем создавать видео с Remotion, нужно подготовить компьютер. Не переживайте — идём шаг за шагом.

## Шаг 1: Установите Node.js

Node.js — это программа, которая позволяет компьютеру запускать JavaScript-код. Remotion без неё не работает.

1. Перейдите на [nodejs.org](https://nodejs.org)
2. Скачайте версию **LTS** (с долгосрочной поддержкой)
3. Запустите установщик и следуйте подсказкам

Проверьте установку, набрав в терминале:

```bash
node --version
```

Вы увидите что-то вроде `v20.11.0`.

## Шаг 2: Создайте первый проект

```bash
npx create-video@latest my-first-video
```

Эта команда скачает шаблон проекта. На вопросы просто нажимайте Enter.

## Шаг 3: Запустите предпросмотр

```bash
cd my-first-video
npm start
```

Откроется браузер с предпросмотром вашего видео.

## Что произошло?

Вы создали проект с:
- Файлами исходного кода
- Локальным сервером для предпросмотра
- Файлами конфигурации

## Решение проблем

- **«command not found»**: убедитесь, что Node.js установлен
- **Медленная загрузка**: это нормально в первый раз

## Итог

Установка Remotion — три шага: Node.js, создание проекта, запуск предпросмотра.
